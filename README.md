# 台灣公司法知識庫

> 完整涵蓋台灣《公司法》全部 449 條條文，提供條文原文、白話解說、實務案例與學習資源。
>
> 法源依據：[全國法規資料庫 - 公司法](https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=J0080001)

## 使用指南

本知識庫適合以下使用者：

- **企業經營者** — 快速查閱公司營運相關法規與實務案例
- **法律從業人員** — 條文交叉引用、立法沿革查詢
- **一般學習者** — 白話解說搭配案例，輕鬆理解公司法

**建議使用方式：**
- 以 [Obsidian](https://obsidian.md/) 開啟本資料夾作為 Vault，可使用圖譜檢視與反向連結
- 直接在 GitHub 上瀏覽，所有連結均使用相對路徑
- 任何支援 Markdown 的編輯器或閱讀器皆可使用
