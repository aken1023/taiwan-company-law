# 第二章 無限公司

> 規範無限公司之設立、內部關係、外部關係、退股、解散及清算等事項。

## 概述

公司法第二章無限公司（第40條至第97條）規範無限公司之各項法律關係。無限公司係指二人以上股東所組織，對公司債務負連帶無限清償責任之公司。

本章分為六節：設立、公司之內部關係、公司之外部關係、退股、解散及清算。設立一節規範無限公司之設立要件與程序；內部關係規範股東間之權利義務；外部關係規範公司與第三人間之法律關係。

退股一節規範股東退出公司之事由與程序；解散一節規範公司解散之原因；清算一節則規範公司解散後之清算程序，包括清算人之選任、職務、財產分配等事項。
