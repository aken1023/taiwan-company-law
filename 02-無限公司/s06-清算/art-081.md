# 第 81 條

> [!info] 章節位置
> 第二章 無限公司 > 第六節 清算

## 條文原文

> 法院因利害關係人之聲請，認為必要時，得將清算人解任。

## 白話解說

本條賦予法院在必要時解任清算人的權力。當利害關係人（如股東、債權人等）認為清算人有不適任的情形，可以向法院聲請將清算人解任。法院審酌後，如果認為確有必要，就可以裁定將清算人解任。

所謂「必要」的情形，包括清算人不忠實執行職務、挪用公司財產、怠忽清算事務、或有其他不適任的情事等。法院的介入是一種外部監督機制，確保清算程序能公正、妥善地進行。

本條與第80條的股東解任不同。第80條是股東自治的內部機制，本條則是法院介入的外部機制，適用於股東自治機制無法有效運作的情形。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算人及利害關係人 |
| 核心規定 | 法院得因利害關係人聲請，於必要時解任清算人 |
| 違反效果 | 清算人被法院解任後不得繼續執行清算職務 |
| 注意事項 | 須利害關係人聲請且法院認為必要 |

## 實務案例

**情境：** 無限公司清算人甲在清算過程中，將公司資產低價出售給自己的親友，債權人乙發現後認為清算人不適任，向法院聲請解任。

**分析：** 依本條規定，乙為利害關係人（債權人），得向法院聲請解任清算人甲。甲將公司資產低價出售予親友，涉嫌不忠實執行清算職務，法院應認為有解任之必要。法院裁定解任後，應另行選派清算人。

**結論：** 法院應准許乙之聲請，解任清算人甲，並另行選派適當之清算人。

## 立法沿革

- 民國18年公司法制定時即有本條規定。
- 本條法院解任清算人之規定，歷次修正均維持不變。
- 民國90年配合公司法全面修正，調整條次。
