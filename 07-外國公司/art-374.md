# 第 374 條

> [!info] 章節位置
> 第七章 外國公司

## 條文原文

> 外國公司應將章程備置於中華民國境內指定之訴訟及非訟代理人處所或其分公司。
>
> 外國公司違反前項規定者，處代表人新臺幣一萬元以上五萬元以下罰鍰。

## 白話解說

外國公司必須將其公司章程放置在台灣境內可供查閱的地方，具體而言是在其指定的訴訟及非訟代理人的處所，或在其分公司所在地。如果外國公司未依規定備置章程，其在台代表人將被處以新台幣 1 萬元以上 5 萬元以下的罰鍰。此規定目的在於確保利害關係人能在台灣境內查閱外國公司的基本組織文件。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 在中華民國境內營業之外國公司 |
| 核心規定 | 章程應備置於台灣境內代理人處所或分公司 |
| 違反效果 | 代表人處 NT$10,000-50,000 罰鍰 |
| 注意事項 | 處罰對象為代表人 |

## 實務案例

**情境：** 德商 C 公司在台設有分公司，但其章程僅存放於德國總公司，未在台灣分公司備置。
**分析：** 違反本條第一項規定，未將章程備置於分公司或代理人處所。
**結論：** C 公司在台代表人應被處新台幣 1 萬元以上 5 萬元以下罰鍰。
