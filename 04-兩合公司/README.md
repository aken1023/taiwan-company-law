# 第四章 兩合公司

> 規範兩合公司之設立、組織及準用無限公司規定等事項。

## 概述

公司法第四章兩合公司（第114條至第127條）規範兩合公司之各項法律關係。兩合公司係指一人以上無限責任股東與一人以上有限責任股東所組織，其無限責任股東對公司債務負連帶無限清償責任，有限責任股東就其出資額為限，對公司負其責任之公司。

本章規範兩合公司之設立要件、章程應載事項、有限責任股東與無限責任股東之權利義務差異、有限責任股東之出資種類限制、業務執行權限、競業禁止等事項。

兩合公司兼具人合與資合之性質，其規範多準用無限公司之規定，但對有限責任股東設有特別保護與限制規定。
