# 第三章 有限公司（§98-§113）

## 章節概述

有限公司是台灣中小企業最常採用之公司組織型態，兼具資合與人合之特徵。股東以其出資額為限對公司負責任，組織架構簡便，一人即可設立。本章僅十六條（§98-§113），為公司法中條文最精簡的公司類型專章。

## 核心概念

### 設立與組織
有限公司由一人以上股東組織之（§98 I）。應訂立章程載明公司名稱、所營事業、股東姓名住所、資本總額及各股東出資額、董事姓名住所等（§101）。各股東每人有一表決權，但得以章程另定按出資多寡比例分配（§102）。

### 出資
股東之出資以現金為主，得以現金以外之財產為出資，但不得以勞務或信用為出資（§99 II）。股東出資之轉讓非經全體股東過半數同意不得為之（§111 I），但二親等以內親屬間轉讓不受此限（§111 III）。

### 治理架構
有限公司應至少設置董事一人，最多三人，執行業務並代表公司（§108 I）。董事由股東選任之，須有三分之二以上股東之同意。不執行業務之股東均得行使監察權，得隨時向董事質詢公司營業情形並查閱帳冊文件（§109）。有限公司不設監察人及董事會，治理架構較股份有限公司簡化。

### 盈餘分配
每屆會計年度終了，董事應編造各項表冊分送各股東請求承認（§110 I）。公司有盈餘時，應先完納稅捐、彌補虧損、提列法定盈餘公積後，始得分配盈餘。

### 變更組織
有限公司得經全體股東同意變更組織為股份有限公司（§106）。此為中小企業成長後轉型之常見路徑。

## 重要條文

| 條號 | 主題 |
|------|------|
| §98 | 有限公司之定義（一人以上股東） |
| §101 | 章程應載事項 |
| §102 | 表決權之分配 |
| §108 | 董事之設置 |
| §109 | 不執行業務股東之監察權 |
| §111 | 出資轉讓限制 |

## 實務意義

有限公司因設立簡便、治理成本低、一人即可設立，為台灣中小企業、新創事業及家族企業之首選。2018年修法取消最低資本額後，設立門檻進一步降低。惟有限公司不得公開發行，無法上市櫃，若企業成長至需要外部融資時，通常須變更組織為股份有限公司。
