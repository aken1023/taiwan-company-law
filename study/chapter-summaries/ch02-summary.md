# 第二章 無限公司（§40-§97）

## 章節概述

無限公司為最典型的人合公司，全體股東對公司債務負連帶無限清償責任。本章分為五節：設立（§40-§44）、公司之內部關係（§45-§58）、公司之對外關係（§59-§68）、退股（§65-§70）、解散、合併及變更組織（§71-§76）及清算（§79-§97）。

## 核心概念

### 設立
無限公司須二人以上股東組織之。設立時應訂立章程，載明公司名稱、所營事業、股東姓名住所、資本總額及各股東出資額、盈餘及虧損分派比例等（§41）。股東得以現金、現金以外之財產、勞務或信用為出資（§43）。

### 內部關係
各股東均有執行業務之權利與義務，但得以章程互約推定執行業務之股東（§45）。業務執行採多數決，但章程另有規定從其規定。不執行業務之股東有監察權（§48）。股東非經其他股東全體同意不得轉讓出資（§55）。

### 對外關係
公司之代表原則上每位執行業務股東均得代表公司，但章程得另定由特定股東代表（§59）。公司股東對公司債務負連帶無限清償之責（§60），但新加入股東對加入前公司之債務亦負連帶責任（§62），退股股東對退股前之債務於退股登記後二年內仍負責任（§69）。

### 退股與入股
股東非經其他股東全體同意不得退股，但章程有訂定得聲明退股之事由者從其規定（§65）。死亡、受禁治產宣告、受破產宣告等亦為法定退股事由（§66）。退股時依退股時公司資產狀況結算退還（§70）。

### 清算
公司解散後應行清算。清算人原則上以全體股東為之，但得另選任（§79）。清算程序包括了結現務、收取債權、清償債務及分配財產等（§84）。

## 重要條文

| 條號 | 主題 |
|------|------|
| §41 | 章程應載事項 |
| §43 | 出資種類（含勞務、信用） |
| §45 | 業務執行 |
| §55 | 出資轉讓限制 |
| §60 | 連帶無限責任 |
| §79 | 清算人之選任 |

## 實務意義

無限公司因股東須負無限責任，實務中設立數量極少。其規範之重要性在於：(1)部分規定為兩合公司所準用；(2)清算規定為公司法清算制度之基礎；(3)退股及責任規定涉及股東個人財產安全。
