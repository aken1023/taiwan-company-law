# 第四章 兩合公司（§114-§127）

## 章節概述

兩合公司是由一人以上無限責任股東與一人以上有限責任股東所組成之公司。結合了無限公司（經營者承擔無限責任）與有限公司（投資者承擔有限責任）之特點。本章僅十四條（§114-§127），多數規定準用無限公司之規定。

## 核心概念

### 設立與組成
兩合公司至少須有一名無限責任股東及一名有限責任股東（§114）。章程應載明各股東為無限或有限責任股東（§115）。有限責任股東不得以勞務或信用為出資（§117）。

### 業務執行
公司業務由無限責任股東執行，有限責任股東不得執行業務或對外代表公司（§122）。若有限責任股東有參與執行業務之行為，對第三人即負與無限責任股東同一之責任（§125）。有限責任股東得行使監察權，查閱公司帳冊及財產文件（§118）。

### 股東責任
無限責任股東對公司債務負連帶無限清償責任，有限責任股東以出資額為限負其責任（§2 I③、§114）。

### 出資轉讓
無限責任股東之出資轉讓準用無限公司規定，須全體股東同意。有限責任股東之出資轉讓須得全體無限責任股東之過半數同意（§119、§120）。

### 準用規定
本章大量準用無限公司之規定（§115），有限責任股東之權利義務則參考有限公司之相關規範。

## 重要條文

| 條號 | 主題 |
|------|------|
| §114 | 兩合公司之組成 |
| §115 | 章程應載事項及準用規定 |
| §117 | 有限責任股東之出資限制 |
| §118 | 有限責任股東之監察權 |
| §122 | 業務執行由無限責任股東負責 |
| §125 | 有限責任股東參與執行業務之責任 |

## 實務意義

兩合公司在台灣實務中極為罕見，幾乎無人採用。主要原因包括：(1)有限公司及股份有限公司已能滿足多數需求；(2)無限責任之風險使經營者卻步；(3)兩類股東之權利義務不對等，容易產生紛爭。本章之學習意義主要在於理解公司法體系之完整性，以及人合與資合公司之交叉型態。
