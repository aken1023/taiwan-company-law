# 第八章 登記（§387-§399）

## 章節概述

本章規範公司登記之程序、效力及主管機關之監督權限。公司登記制度為公司法之重要機制，確保公司資訊之公示與透明，保護交易安全。

## 核心概念

### 登記之種類與程序
公司登記事項包括設立登記、變更登記、解散登記及其他法定應登記事項。公司及外國公司登記之申請，由代表公司之負責人或外國公司之代理人為之（§387 I）。登記事項發生變更後十五日內應申請變更登記。

### 登記之效力
公司設立登記後始取得法人格。已登記事項有變更而未登記者，不得以其變更對抗第三人（§12）。此為登記之消極公示效力，保護善意第三人之交易安全。

### 主管機關之監督
主管機關得依職權或依利害關係人之申請，命令公司提出證明文件、帳冊或派員檢查（§21）。公司應將登記表件及公司章程備置於本公司。

### 登記之撤銷與廢止
公司登記有虛偽不實之情事者，主管機關得依職權或利害關係人之聲請撤銷之（§9、§397）。公司設立登記後六個月尚未開始營業或自行停業六個月以上者，主管機關得廢止其登記（§397）。

### 規費
公司登記應繳納登記規費，由中央主管機關定之（§392 II）。

## 重要條文

| 條號 | 主題 |
|------|------|
| §12 | 登記之對抗效力 |
| §387 | 登記之申請人 |
| §388 | 主管機關之審查 |
| §391 | 登記文件之保存 |
| §397 | 撤銷或廢止登記 |

## 實務意義

登記制度在實務上極為重要。公司設立、增減資、董監事變更、地址遷移等均須辦理登記。未依限辦理者將受罰鍰處分。§12之對抗效力規定常見於訴訟中，如公司代表人已變更但未辦理登記時，舊代表人所為之行為對善意第三人仍屬有效。目前公司登記已全面線上化，可透過經濟部商業司之線上系統辦理。
