# 第六章之一 關係企業（§369-1-§369-12）

## 章節概述

關係企業專章於1997年增訂，規範具有控制從屬關係或相互投資之公司群。本章旨在保護從屬公司少數股東及債權人之權益，防止控制公司濫用控制力損害從屬公司利益。

## 核心概念

### 關係企業之定義
關係企業指獨立存在而相互間具有控制與從屬關係之公司，或相互投資之公司（§369-1）。

### 控制與從屬關係
有下列情形之一者推定為有控制從屬關係（§369-2、§369-3）：(1)公司持有他公司有表決權之股份或出資額超過半數者；(2)公司直接或間接控制他公司之人事、財務或業務經營者；(3)公司與他公司之已發行有表決權之股份總數或資本總額有半數以上為相同之股東持有或出資者。

### 相互投資
公司與他公司相互投資各達對方有表決權之股份總數或資本總額三分之一以上者為相互投資公司（§369-9）。相互投資公司知有相互投資之事實者，其得行使之表決權不得超過被投資公司已發行有表決權股份總數或資本總額之三分之一（§369-10）。

### 控制公司之責任
控制公司直接或間接使從屬公司為不合營業常規或其他不利益之經營者，應對從屬公司因此所受之損害負賠償責任（§369-4 I）。控制公司負責人使從屬公司為不利益經營者，應與控制公司連帶賠償（§369-4 III）。

### 資訊揭露
從屬公司應於每會計年度終了時編製關係報告書，載明與控制公司間之法律行為、資金往來及損益情形（§369-12）。控制公司應編製關係企業合併營業報告書及合併財務報表。

## 重要條文

| 條號 | 主題 |
|------|------|
| §369-1 | 關係企業之定義 |
| §369-2 | 控制公司之定義 |
| §369-3 | 控制從屬關係之推定 |
| §369-4 | 控制公司之賠償責任 |
| §369-9 | 相互投資之定義 |
| §369-10 | 相互投資表決權之限制 |

## 實務意義

台灣企業集團普遍存在控制從屬關係，關係企業專章提供了規範框架。實務上§369-4的控制公司責任是保護從屬公司少數股東及債權人的重要武器。相互投資之表決權限制則避免公司間交叉持股造成虛增資本及經營權鞏固之弊端。
