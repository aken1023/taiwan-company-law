# 第三章 有限公司

> 規範有限公司之設立、組織、股東權利義務及公司運作等事項。

## 概述

公司法第三章有限公司（第98條至第113條）規範有限公司之各項法律關係。有限公司係由一人以上股東所組織，就其出資額為限，對公司負其責任之公司。

本章規範有限公司之設立要件、章程應載事項、股東出資、股東表決權、盈餘分配、董事之選任與職權、公司業務執行、股東之退股與轉讓出資等事項。

有限公司為我國中小企業常見之組織型態，其規範兼顧人合與資合之特性，在股東人數、出資轉讓、組織運作等方面均有別於股份有限公司。
