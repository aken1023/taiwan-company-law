# 第 110 條

> [!info] 章節位置
> 第三章 有限公司

## 條文原文

> 每屆會計年度終了，董事應依第二百二十八條之規定，造具各項表冊，分送各股東，請其承認。
>
> 前項表冊，應經股東表決權過半數之同意。
>
> 董事違反前二項規定時，各科新臺幣一萬元以上五萬元以下罰鍰。

## 白話解說

本條規定有限公司董事在每個會計年度結束後的財務報告義務。董事必須依照第228條的規定，編製營業報告書、財務報表及盈餘分派或虧損撥補之議案等各項表冊，並分送給每位股東，請求股東承認。

這些表冊須經股東表決權過半數同意才算通過。表決權的計算方式依第102條規定，原則上一人一票，但章程可約定按出資比例分配。股東承認表冊的程序，等同於股份有限公司股東會承認財務報表的功能，是對董事經營成果的審查與確認。

如果董事違反本條規定，未依法編製表冊或未分送股東請求承認，將被處以新臺幣一萬元以上五萬元以下的行政罰鍰。這項罰則督促董事確實履行財務報告義務，保障股東的知情權。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 有限公司董事及全體股東 |
| 核心規定 | 董事應於年度終了造具表冊分送股東，經表決權過半數同意承認 |
| 違反效果 | 董事違反者，科新臺幣一萬元以上五萬元以下罰鍰 |
| 注意事項 | 表冊內容準用第228條規定，須含營業報告書及財務報表等 |

## 實務案例

辛有限公司的會計年度於每年12月31日終了。董事A在次年3月前依第228條規定編製了營業報告書、資產負債表、損益表及盈餘分派議案，將這些表冊分送給三位股東。股東審閱後，以表決權過半數同意承認該年度財務報表，並同意將盈餘之一部分依章程比例分派予各股東。若董事A怠於編製表冊或未分送股東，主管機關查知後可對A處以罰鍰。

## 立法沿革

本條於民國107年修正時有重要調整。修正前規定表冊須經「股東過半數之同意」，修正後改為「股東表決權過半數之同意」，以配合第102條表決權制度之修正。同時新增第三項關於董事違反義務之罰鍰規定，強化對董事之課責機制。
