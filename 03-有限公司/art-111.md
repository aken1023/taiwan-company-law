# 第 111 條

> [!info] 章節位置
> 第三章 有限公司

## 條文原文

> 股東非得其他全體股東過半數之同意，不得以其出資之全部或一部，轉讓於他人。
>
> 前項轉讓，不同意之股東有優先受讓權；如不承受，視為同意轉讓。
>
> 法院依強制執行程序，將股東之出資轉讓於他人時，應通知公司及其他全體股東，於二十日內，依第一項及第二項之規定行使權利。逾期未行使者，視為同意轉讓。

## 白話解說

本條規範有限公司股東出資轉讓的程序與限制。由於有限公司具有「人合」性質，股東之間的信賴關係很重要，因此股東要將出資轉讓給外部第三人時，必須取得其他全體股東過半數的同意。無論是轉讓全部或一部分的出資，都適用這個規定。

第二項設有「優先受讓權」機制：不同意轉讓的股東享有優先承購的權利。如果不同意的股東自己也不願意承購，就視為同意該筆轉讓。這項規定兼顧了維護有限公司人合性與保障股東退出權的平衡。

第三項則處理法院強制執行的情況。當股東的出資因債務問題被法院查封並強制轉讓時，法院應通知公司及其他全體股東，給予二十日的期間讓其他股東決定是否同意或行使優先受讓權。逾期未表示意見者，視為同意轉讓。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 有限公司股東及擬受讓人 |
| 核心規定 | 出資轉讓須其他全體股東過半數同意，不同意者有優先受讓權 |
| 違反效果 | 未經合法程序之轉讓對公司不生效力 |
| 注意事項 | 不同意之股東若不承受，視為同意；強制執行時有20日行使期間 |

## 實務案例

壬有限公司有A、B、C、D四位股東。A因個人財務需求，欲將其全部出資轉讓給公司外部人士E。A須取得B、C、D中過半數（即至少二人）的同意。B和C同意，但D反對。因已取得過半數同意，A原則上可以進行轉讓。但D因反對而享有優先受讓權，可以優先用相同條件承購A的出資。如果D在合理期間內表示不願承購，則視為D也同意轉讓，A得將出資轉讓予E。轉讓完成後，公司應更新股東名簿，將E登記為新股東。

## 立法沿革

本條歷經多次修正。民國107年修正時，將「其他股東過半數之同意」修正為「其他全體股東過半數之同意」，更加明確計算基準。同時完善優先受讓權之規定，並新增第三項關於強制執行程序中出資轉讓之特別規定，使法律關係更加完備。此修正在維護公司人合性的同時，也兼顧了股東出資流通性及債權人之利益。
