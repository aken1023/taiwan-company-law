# 第 18 條

> [!info] 章節位置
> 第一章 總則

## 條文原文

> 公司名稱，應使用我國文字，且不得與他公司或有限合夥名稱相同。
>
> 公司名稱不得使人誤認為政府機關或公益團體者。
>
> 公司所營事業除許可業務應載明於章程外，其餘不受限制。
>
> 公司所營事業應依中央主管機關所定營業項目代碼表登記。

## 白話解說

這條規範了公司名稱和所營事業兩大重點。

關於公司名稱：第一，必須使用中文（我國文字），不能只用英文或其他外國文字來登記公司名稱。第二，公司名稱不能跟其他已登記的公司或有限合夥「完全相同」，但如果只是「類似」而非完全一樣，則不在此限。第三，公司名稱不能讓人誤以為是政府機關或公益團體，例如不能取名叫「台灣經濟發展署有限公司」之類容易混淆的名稱。

關於所營事業：2018 年修法後大幅放寬，除了需要政府許可的業務（如銀行、保險等）必須在章程中載明外，其他所營事業不再受限制。也就是說，公司章程中不必逐一列出所有經營項目，大大增加了公司經營的彈性。但在登記時，仍需依經濟部公布的營業項目代碼表來申報。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有公司 |
| 核心規定 | 公司名稱須用中文、不得與他公司相同、不得誤認為政府機關；所營事業除許可業務外不受限制 |
| 違反效果 | 名稱與他公司相同者，主管機關不予登記 |
| 注意事項 | 2018 年修法放寬所營事業限制，許可業務仍須載明於章程 |

## 實務案例

**情境：** 李小姐想設立一家科技公司，取名為「Taiwan Innovation Co., Ltd.」，並計畫只用英文名稱辦理公司登記。此外，她也想登記「ZZ99 批發業」等多項營業項目。

**分析：** 依本條第一項，公司名稱必須使用我國文字（中文），因此李小姐不能僅以英文名稱登記。她可以取一個中文公司名稱如「台灣創新科技股份有限公司」，英文名稱則可另外申請登記。至於營業項目，2018 年修法後，除許可業務外已不受限制，但登記時仍須按經濟部營業項目代碼表選填代碼。

**結論：** 公司名稱一定要有中文名稱。營業項目方面，現行法已大幅鬆綁，建議可登記「ZZ99999 除許可業務外，得經營法令非禁止或限制之業務」作為概括性登記。
