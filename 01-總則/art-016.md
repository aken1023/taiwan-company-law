# 第 16 條

> [!info] 章節位置
> 第一章 總則

## 條文原文

> 公司除依其他法律或公司章程規定得為保證者外，不得為任何保證人。
>
> 公司負責人違反前項規定時，應自負保證責任，並各科新臺幣十五萬元以下罰鍰，如公司受有損害時，亦應負賠償責任。

## 白話解說

公司原則上**不能替別人做保證人**（例如替他人向銀行借款作保）。只有在兩種例外情況下可以：
1. **其他法律允許**（如銀行法允許銀行提供保證）
2. **公司章程有規定**

如果公司負責人違規替公司作保，後果是：
- 負責人自己要承擔保證責任（公司不用負責）
- 被處 15 萬元以下罰鍰
- 如果公司因此受損害，還要賠償公司

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有公司 |
| 核心規定 | 公司原則上不得為保證人 |
| 違反效果 | 負責人自負保證責任、15 萬元以下罰鍰、損害賠償 |
| 注意事項 | 章程有規定或其他法律允許者除外 |

## 實務案例

**情境：** B 公司的董事長未經章程授權，擅自以 B 公司名義為關係企業 C 公司向銀行貸款 5,000 萬元作保。

**分析：** 依第 16 條，B 公司章程若未授權可作保，則該保證行為違法。此時 B 公司不負保證責任，改由董事長個人承擔保證責任。銀行可以向董事長個人追償，但不能向 B 公司追償。

**結論：** 公司為他人作保前，務必確認章程中有相關授權條款。
