# 第一章 總則

> 公司法總則規定公司之基本定義、種類、能力、登記及其他通則性事項。

## 概述

公司法第一章總則（第1條至第39條）為公司法之基礎規範，涵蓋公司之定義、種類（無限公司、有限公司、兩合公司、股份有限公司）、公司名稱、住所、能力與權利義務等基本事項。

本章亦規範公司之設立登記、變更登記、負責人之定義與責任、公司之轉投資限制、資金貸與限制、公司負責人忠實義務及注意義務等重要原則。

此外，總則中對於公司之合併、解散、清算等程序亦設有通則性規定，為後續各章之個別公司型態規範奠定基礎。
