# 第五章 股份有限公司

> 公司法中最重要且篇幅最大的章節，共分 13 節，完整規範股份有限公司從設立、營運到解散清算的全生命週期。

## 章節概覽

股份有限公司是台灣最常見的公司型態，本章涵蓋條文約 240 條（§128 - §356-14），佔公司法全部條文的一半以上。內容從公司設立、股份發行、股東會運作、董監事制度、財務會計、公司債，到發行新股、變更章程、公司重整、解散合併分割、清算程序，以及特殊的閉鎖性股份有限公司，均有詳細規範。

## 學習建議

1. **初學者**：建議先閱讀第一節（設立）與第三節（股東會），了解公司基本運作
2. **實務工作者**：重點關注第四節（董事及董事會）與第六節（會計）
3. **進階研究**：第十節（公司重整）與第十三節（閉鎖性公司）為近年修法重點
