# 第 222 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人不得兼任公司董事、經理人或其他職員。

## 白話解說

本條是監察人「兼任禁止」的規定，目的在確保監察人的獨立性。監察人的核心職責是監督公司業務的執行，如果監察人同時兼任被監督的對象（董事、經理人或其他職員），將形成「自己監督自己」的利益衝突，使監督機制形同虛設。

所謂「公司董事」，包括一般董事及董事長。所謂「經理人」，指依第29條經董事會決議委任之經理人。所謂「其他職員」，泛指公司內任何受僱從事業務之人員，包括一般員工。

此禁止規定是絕對性的，沒有例外。違反本條規定而兼任者，其兼任之行為無效。若監察人當選後被任命為公司經理人，則應認其監察人資格不受影響，但其經理人之任命因違反本條而無效（或反之，視先後順序而定）。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人 |
| 核心規定 | 監察人不得兼任公司董事、經理人或其他職員 |
| 違反效果 | 兼任之行為無效 |
| 注意事項 | 此為絕對禁止規定，無例外 |

## 實務案例

**情境：** 丑公司監察人A因公司人手不足，被董事會決議聘任為公司財務經理。A接受任命後同時擔任監察人與財務經理。

**分析：** 依本條規定，監察人不得兼任公司經理人或其他職員。A同時擔任監察人與財務經理，已違反本條之兼任禁止規定。A作為財務經理所處理的財務事項，本應受監察人之監督，若A自己擔任兩個角色，形成自己監督自己的情形，失去監督意義。A之財務經理任命應認定為無效。

**結論：** 公司不應任命監察人擔任任何公司職務。若有此需要，應先由股東會解任其監察人身分，再另行擔任公司職務。
