# 第 218-1 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 董事發現公司有受重大損害之虞時，應立即向監察人報告。

## 白話解說

本條規定董事對監察人的主動報告義務。當董事在執行職務過程中，發現公司有可能受到重大損害時（例如重大財務危機、重大訴訟敗訴風險、重大合約違約等），董事有義務「立即」向監察人報告，不得遲延。

此規定的目的在於強化公司內部的預警機制。監察人雖然有權隨時調查公司業務，但畢竟不可能時時刻刻掌握公司所有狀況。董事身處經營第一線，最能即時發現公司面臨的風險。透過董事的即時報告，監察人才能及時採取必要的監督行動，例如召集董事會、通知股東會，或者採取法律行動以保護公司利益。

所謂「重大損害之虞」，不限於已經發生的損害，而是指可以合理預見將會發生重大損害的情形，亦即具有損害發生之蓋然性即足。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之全體董事 |
| 核心規定 | 董事發現公司有受重大損害之虞時，應立即向監察人報告 |
| 違反效果 | 違反者可能構成董事怠忽職務之損害賠償責任 |
| 注意事項 | 「重大損害之虞」包括尚未發生但可合理預見之損害 |

## 實務案例

**情境：** 己公司董事長在與某大客戶洽談時，得知該客戶即將對公司提起鉅額訴訟（金額高達公司資本額一半以上），但董事長認為公司有勝訴把握，故未向監察人報告。數月後公司敗訴，遭受重大損失。

**分析：** 依本條規定，董事在發現公司有受重大損害之虞時，應「立即」向監察人報告。面對鉅額訴訟風險，即使董事長主觀認為勝訴機會大，此仍屬「有受重大損害之虞」之客觀情形，董事長有報告義務。未報告導致監察人無法及時介入（例如要求董事會評估和解方案、提列訴訟準備金等），可能使公司損害擴大。

**結論：** 董事發現可能的重大損害風險時，不應以個人判斷取代報告義務。即時通報監察人是法定義務，怠於報告者應對因此擴大之損害負賠償責任。
