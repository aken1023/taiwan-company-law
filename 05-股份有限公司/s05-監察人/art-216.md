# 第 216 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 公司監察人，由股東會選任之，監察人中至少須有一人在國內有住所。
>
> 公開發行股票之公司依前項選任之監察人須有二人以上，其全體監察人合計持股比例，證券管理機關另有規定者，從其規定。
>
> 第三十條之規定，於監察人準用之。
>
> 公司與監察人間之關係，從民法關於委任之規定。
>
> 第一百九十二條第一項但書及第四項之規定，對監察人準用之。

## 白話解說

本條規定了監察人的選任方式與基本資格要求。監察人是由股東會選舉產生的，其主要職責在於監督公司業務的執行，與董事會形成制衡關係。為確保監察人能夠有效履行監督職責，法律要求監察人中至少須有一人在國內有住所，以利其隨時對公司進行監督。

對於公開發行股票的公司，因其涉及廣大投資人權益，法律特別加重要求，監察人至少須二人以上，以發揮相互配合之監督效果。至於監察人的持股比例，則授權證券管理機關另行規定。

本條並準用第30條消極資格之規定，亦即有前科紀錄、受破產宣告等情形者，不得擔任監察人。公司與監察人間的法律關係係屬「委任」關係，適用民法委任之規定。此外，監察人得由政府或法人股東當選，並準用第192條有關當選後指定代表行使職務之規定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司 |
| 核心規定 | 監察人由股東會選任，至少一人須在國內有住所 |
| 違反效果 | 不具資格者之選任無效 |
| 注意事項 | 公開發行公司監察人至少二人；準用第30條消極資格規定 |

## 實務案例

**情境：** 甲公開發行公司僅選任一名監察人A，且A長年居住於國外，未在國內設有住所。

**分析：** 依本條規定，公開發行公司監察人至少須二人以上，且至少須有一人在國內有住所。甲公司僅選任一人已違反法定人數要求。此外，唯一的監察人若無國內住所，亦違反「至少一人在國內有住所」之要求。主管機關得命令公司限期補選，逾期不補選者將受罰。

**結論：** 公開發行公司應確保監察人人數及住所要件均符合法定要求，否則選任程序將有瑕疵，公司並可能面臨主管機關之處分。
