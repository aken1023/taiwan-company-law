# 第 217-1 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人全體均解任時，董事會應於三十日內召集股東臨時會選任之。但公開發行股票之公司，董事會應於六十日內召集股東臨時會選任之。

## 白話解說

本條針對公司監察人全體均解任的緊急情況，設有強制補選機制。由於監察人是公司治理中不可或缺的監督機關，若監察人全部缺位，公司將失去內部監督機制，對股東權益保護極為不利。

因此，當監察人全體均解任時（無論是因任期屆滿當然解任、被股東會決議解任，或是辭職等原因），董事會有義務在法定期限內召集股東臨時會來選任新的監察人。一般公司的期限為三十日，而公開發行公司因召集程序較為繁複（須依證券交易法規定辦理公告等），期限放寬為六十日。

這項規定確保公司不會長期處於無監察人的狀態，維護公司治理的完整性。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 監察人全體均解任之股份有限公司 |
| 核心規定 | 董事會應於30日（公開發行公司60日）內召集臨時股東會選任 |
| 違反效果 | 董事會怠於召集者，可能構成董事之損害賠償責任 |
| 注意事項 | 公開發行公司與非公開發行公司期限不同 |

## 實務案例

**情境：** 丁公開發行公司之三名監察人因公司經營理念分歧，同時向公司提出辭呈。董事會收到辭呈後，因正值公司合併案繁忙期間，遲遲未召集股東臨時會。

**分析：** 三名監察人同時辭職，已構成「監察人全體均解任」之情形。依本條規定，丁公司董事會應於六十日內召集股東臨時會選任新監察人。董事會不得以業務繁忙為由拖延，否則將違反法定義務。在新監察人選出前，公司之財務報表審查、董事會列席監督等功能將出現空窗。

**結論：** 公司董事會在監察人全體解任後，應立即啟動臨時股東會之召集程序，在法定期限內完成新任監察人之選任，以維護公司治理之健全。
