# 第 226 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人對於董事會編造提出股東會之各種表冊，認為有不當時，得提出意見於股東會；第二百十四條之規定，對監察人準用之。

## 白話解說

本條包含兩部分規定。第一部分賦予監察人在查核董事會編造之表冊後，若認為有不當之處，有權在股東會上提出其異議意見，讓股東在承認表冊時有充分的參考資訊。這與第219條的查核報告義務相互配合：第219條要求監察人查核並報告意見，本條則進一步強調監察人對於不當之處的特別報告權。

第二部分則規定第214條（股東代位訴訟）之規定準用於監察人。亦即，繼續一年以上持有已發行股份總數百分之一以上之股東，得以書面請求監察人為公司對董事提起訴訟。若監察人自收到請求之日起三十日內不提起訴訟，該股東得為公司提起訴訟。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人 |
| 核心規定 | 監察人認為表冊不當得向股東會提出意見；準用股東代位訴訟規定 |
| 違反效果 | 監察人怠於提訴時，股東得代位提起 |
| 注意事項 | 第214條股東代位訴訟之規定準用於監察人 |

## 實務案例

**情境：** 巳公司監察人查核年度財務報表後，發現董事會將部分研發費用不當資本化，導致利潤虛增。監察人於股東會中報告「查核尚無重大異常」，未提出異議。少數股東事後發現此問題。

**分析：** 依本條規定，監察人認為表冊有不當時，應提出意見於股東會。本案監察人未對財務報表中不當資本化之問題提出意見，已有違其職責。此外，依本條準用第214條之規定，繼續一年以上持有已發行股份1%以上之股東，得以書面請求監察人為公司對編造不實表冊之董事提起訴訟。

**結論：** 監察人發現表冊不當時有義務向股東會報告，怠於報告者須依第224條負賠償責任。股東亦可透過準用第214條之代位訴訟機制追究董事責任。
