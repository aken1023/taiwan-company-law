# 第 221 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人各得單獨行使監察權。

## 白話解說

本條規定監察人得各自「單獨」行使監察權，不需要全體監察人共同行動或經過多數決。這是監察人制度與董事會制度最大的不同之處。董事會採合議制，決策須經董事會會議決議；但監察人則採「獨任制」，每位監察人均有獨立完整的監察權限。

此設計的理由在於：監察人的主要職責是監督，如果要求監察人像董事會一樣採合議制，則可能出現多數監察人與經營階層串通而壓制少數監察人的情形，反而削弱監督功能。採獨任制可以確保即使只有一位監察人發現問題，也能單獨採取行動進行調查、查核帳冊、通知停止違法行為等。

因此，任何一位監察人都可以獨自行使第218條的調查權、第219條的財務報表查核權、第218-2條的列席董事會權、第220條的召集股東會權等全部監察權限，不須其他監察人之同意。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之各個監察人 |
| 核心規定 | 監察人各得單獨行使監察權，不須合議 |
| 違反效果 | 不適用，本條為授權規定 |
| 注意事項 | 與董事會之合議制不同，監察人採獨任制 |

## 實務案例

**情境：** 子公司設有三名監察人。監察人A發現公司可能有關係人交易問題，擬調閱公司帳冊。監察人B與C認為沒有必要，反對A之調查行為。

**分析：** 依本條規定，監察人各得「單獨」行使監察權。監察人A有權獨自行使第218條之調查權，包括調閱帳冊、查核文件等，不須經過B、C之同意。B、C之反對不影響A之權限行使。公司亦不得以「多數監察人反對」為由拒絕A之調查要求。

**結論：** 監察權之行使採獨任制，每位監察人均有完整且獨立之監察權限，不受其他監察人意見之拘束。
