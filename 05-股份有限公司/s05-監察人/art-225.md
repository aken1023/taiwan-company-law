# 第 225 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 股東會決議，對於監察人提起訴訟時，公司應自決議之日起三十日內提起之。

## 白話解說

本條規定了公司對監察人提起訴訟的程序與期限。當監察人因違反法令、章程或怠忽職務致公司受有損害時（如第224條之情形），公司得對監察人提起損害賠償訴訟。但此項訴訟須先經股東會決議，始得提起。

經股東會決議後，公司應在決議之日起三十日內提起訴訟。設定此期限的目的，在於避免公司拖延提訴，讓責任追究流於空談。若公司逾三十日期限仍未提起訴訟，股東得依第214條或第215條之規定，自行為公司提起訴訟（代位訴訟）。

由於監察人通常是代表公司對外進行訴訟之人，但在對監察人提起訴訟時，顯然不能由監察人代表公司。此時應由董事會代表公司提起訴訟，或由股東會另選代表人為之。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司對其監察人 |
| 核心規定 | 股東會決議後，公司應於三十日內對監察人提起訴訟 |
| 違反效果 | 逾期未提訴，股東得依法代位提起訴訟 |
| 注意事項 | 須先有股東會決議；由董事代表公司進行訴訟 |

## 實務案例

**情境：** 辰公司股東會通過決議，對監察人C提起損害賠償訴訟，金額為五百萬元。但決議後四十天，公司仍未向法院提起訴訟。

**分析：** 依本條規定，公司應自股東會決議之日起三十日內提起訴訟。辰公司已逾三十日期限仍未提訴，持有股份達一定比例之股東得依第214條規定，以書面請求公司提訴。若公司仍不提訴，股東得為公司利益代位提起訴訟。

**結論：** 三十日期限為法定不變期間，公司應在期限內完成起訴程序。逾期雖不影響訴權本身（仍可提訴），但股東取得代位訴訟之權利。
