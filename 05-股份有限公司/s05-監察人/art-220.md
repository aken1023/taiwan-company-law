# 第 220 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人除董事會不為召集或不能召集股東會外，得為公司利益，於必要時，召集股東會。

## 白話解說

本條賦予監察人在特定情況下召集股東會的權限。一般而言，股東會的召集權屬於董事會。但在以下兩種情形下，監察人得自行召集股東會：

第一，董事會「不為召集」，亦即董事會有能力召集股東會但拒絕或消極不召集。例如董事會明知有重大事項需提交股東會決議，但因董事長考量個人利益而不願召集。

第二，董事會「不能召集」，亦即董事會客觀上已無法召集股東會。例如董事全體辭職、死亡或因故無法行使職權。

除了以上情形，監察人亦得於「必要時」為公司利益召集股東會。此處「必要時」應由監察人依其專業判斷，認為確有召集股東會之必要，以保護公司及股東之利益。

此規定確保公司在董事會功能失靈時，仍有替代機制可以啟動股東會之召集程序，為公司治理提供安全閥。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人 |
| 核心規定 | 董事會不為或不能召集股東會時，監察人得為公司利益召集之 |
| 違反效果 | 無正當理由召集可能導致決議效力有爭議 |
| 注意事項 | 須以「為公司利益」且「必要時」為前提 |

## 實務案例

**情境：** 癸公司董事長涉嫌掏空公司，其他董事亦配合不願召集股東會。監察人認為應召集股東會改選董事會以保護公司利益。

**分析：** 本案董事會雖有能力召集股東會，但因董事長涉嫌掏空而「不為召集」。此時監察人為保護公司利益，有權依本條規定自行召集股東會。監察人應發出合法之召集通知，載明開會日期、地點及議案（如改選全體董事），依法定期間通知全體股東。

**結論：** 監察人在董事會不為召集的情況下，得以自身名義召集股東會，此為法律賦予之特別召集權，不須經董事會同意。召集之股東會決議效力與董事會召集者相同。
