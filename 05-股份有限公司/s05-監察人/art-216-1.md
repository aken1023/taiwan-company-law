# 第 216-1 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 公司監察人選舉，依章程規定採候選人提名制度者，準用第一百九十二條之一規定。

## 白話解說

本條允許公司在章程中訂明監察人選舉採用候選人提名制度。所謂候選人提名制度，是指監察人候選人須先經過提名程序，再由股東會就提名之候選人名單中選舉產生。這與傳統上由出席股東當場提名的方式不同，可以使選舉更有秩序並事先審查候選人資格。

採用候選人提名制度時，須準用第192條之1有關董事候選人提名制度的各項規定，包括：受理候選人提名期間及公告方式、持有一定比例股份之股東得提出候選人名單、董事會或其他召集權人審查候選人資格等程序。

此制度的目的在於使監察人選舉更加透明、有效率，尤其對於公開發行公司而言，可避免選舉當日臨時提名造成的混亂。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 章程訂明採候選人提名制度之股份有限公司 |
| 核心規定 | 監察人選舉得依章程規定採候選人提名制度 |
| 違反效果 | 未依法定程序提名者不得列入候選名單 |
| 注意事項 | 須準用第192-1條全部程序規定 |

## 實務案例

**情境：** 乙公司章程明定監察人選舉採候選人提名制度，持有已發行股份1%以上之股東甲提出候選人名單。董事會審查後發現其中一位候選人曾受有期徒刑之宣告尚未執行完畢，屬於消極資格限制。

**分析：** 依本條準用第192條之1規定，董事會應審查候選人是否符合資格。該候選人因有第30條消極資格之事由，董事會應將其從候選人名單中剔除，僅將合格之候選人提交股東會選舉。

**結論：** 候選人提名制度的優點在於可以事先篩選候選人資格，避免不適任者當選後再生爭議，但須嚴格遵守法定的提名及審查程序。
