# 第 217 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人任期不得逾三年。但得連選連任。
>
> 監察人任期屆滿而不及改選時，延長其任期至改選監察人就任時為止。但主管機關得依職權限期令公司改選；屆期仍不改選者，自限期屆滿時，當然解任。

## 白話解說

本條規定監察人的任期上限為三年，但可以連選連任，並無連任次數之限制。這使得表現良好、受股東信任的監察人得以持續在任，維持監督工作的穩定性與連續性。

為避免監察人任期屆滿後公司因故未及時改選而出現監督真空，本條第二項設有「延任」機制：當任期屆滿而尚未改選時，原監察人的任期自動延長至新任監察人就任時為止。不過，為了防止公司藉此規定無限期拖延改選，主管機關有權依職權限期命令公司辦理改選。如果公司在限期屆滿後仍不改選，原任監察人即當然解任，不再享有監察人之身分與權限。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人 |
| 核心規定 | 任期不得逾三年，得連選連任 |
| 違反效果 | 主管機關限期不改選者，監察人當然解任 |
| 注意事項 | 任期屆滿不及改選時自動延任至新任就任時 |

## 實務案例

**情境：** 丙公司監察人任期已屆滿一年，公司仍未召集股東會改選監察人。主管機關發函限期三個月內辦理改選，但丙公司置之不理。

**分析：** 依本條第二項，監察人任期屆滿時雖可延任，但主管機關已限期命令改選。丙公司逾期仍不改選，則原監察人於限期屆滿時「當然解任」，無須任何其他程序。此後公司即無監察人，將嚴重影響公司治理。

**結論：** 公司應重視監察人改選之期限，若因故不及改選雖有延任機制保障，但不可無限期拖延。主管機關介入後仍不改選，將導致監察人全體當然解任，公司應依第217-1條盡速召集臨時股東會補選。
