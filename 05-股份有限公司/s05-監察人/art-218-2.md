# 第 218-2 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人得列席董事會陳述意見。
>
> 董事會或董事執行業務有違反法令、章程或股東會決議之行為者，監察人應即通知董事會或董事停止其行為。

## 白話解說

本條賦予監察人兩項重要的監督工具。第一項規定監察人有權列席董事會並陳述意見。注意是「列席」而非「出席」，亦即監察人不具有表決權，但可以在董事會上發表看法，對決策提出質疑或建議。此規定使監察人能即時掌握董事會之決策動態，落實事前監督之功能。

第二項則是更積極的監督手段：當董事會或董事個人在執行業務時有違反法令、章程或股東會決議的行為時，監察人「應即」通知董事會或該董事停止違法行為。這不僅是監察人的「權利」，更是其「義務」。監察人發現違法行為卻未即時通知停止者，可能構成怠忽職務，須依第224條負損害賠償責任。

這項停止通知雖不具有強制執行力，但它會產生重要的法律效果：若董事在接到通知後仍繼續違法行為，將更難主張其行為係出於善意或無過失。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人及董事會 |
| 核心規定 | 監察人得列席董事會；違法時應通知停止 |
| 違反效果 | 監察人知情不通知者構成怠忽職務，須負賠償責任 |
| 注意事項 | 監察人列席董事會無表決權，僅得陳述意見 |

## 實務案例

**情境：** 庚公司董事會決議以顯著低於市價之價格，將公司一筆不動產出售予董事長之關係人。監察人列席董事會時發現此交易明顯不合理，但未表示反對，也未通知董事會停止。事後公司因此交易受有重大損失。

**分析：** 依本條第二項，董事會執行業務有違反法令或章程之行為（以不合理低價出售公司資產，可能構成違反董事忠實義務），監察人「應即通知董事會停止其行為」。本案監察人列席董事會卻未發出停止通知，已構成怠忽職務。依第224條規定，監察人應對公司因此所受之損害負賠償責任。

**結論：** 監察人列席董事會不僅是權利，更隱含積極監督之義務。發現違法行為時，監察人必須立即通知停止，否則將與違法之董事共同承擔責任。
