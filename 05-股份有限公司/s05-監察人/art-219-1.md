# 第 219-1 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 公開發行股票之公司，得依證券管理法令規定設置審計委員會替代監察人；其辦法，由證券主管機關定之。
>
> 公司設置審計委員會者，本法、證券交易法及其他法律對於監察人之規定，於審計委員會準用之。

## 白話解說

本條是公司治理制度革新的重要規定，允許公開發行公司以「審計委員會」替代傳統的監察人制度。審計委員會由獨立董事組成，是借鏡國際公司治理趨勢（特別是美國制度）的設計。

設置審計委員會後，原本由監察人行使的各項職權（如查核財務報表、列席董事會、對違法行為通知停止等），全部改由審計委員會行使。公司法、證券交易法及其他法律中有關監察人的規定，均準用於審計委員會。

審計委員會的具體設置辦法、組成要件、運作方式等，授權由證券主管機關（金融監督管理委員會）以行政命令定之。目前上市櫃公司已逐步全面設置審計委員會，取代傳統監察人制度。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 公開發行股票之公司 |
| 核心規定 | 得以審計委員會替代監察人 |
| 違反效果 | 須依證券管理法令規定辦理 |
| 注意事項 | 設置後，所有監察人相關規定準用於審計委員會 |

## 實務案例

**情境：** 壬上市公司原設有三名監察人，經董事會評估後，決議自下屆起改設審計委員會，由三名獨立董事組成。股東A詢問改制後原來監察人的職權如何行使。

**分析：** 依本條規定，公司設置審計委員會後，公司法及證券交易法中所有關於監察人的規定，均準用於審計委員會。因此，原本由監察人行使的查核財務報表（第219條）、列席董事會（第218-2條）、代表公司與董事交易（第223條）等職權，改由審計委員會依法行使。公司不再設置監察人。

**結論：** 審計委員會制度與監察人制度二擇一，不得同時並存。設置審計委員會後，其享有與監察人相同的法定權限與義務。
