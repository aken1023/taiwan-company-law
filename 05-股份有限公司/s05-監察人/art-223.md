# 第 223 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 董事為自己或他人與公司為買賣、借貸或其他法律行為時，由監察人為公司之代表。

## 白話解說

本條解決了董事與公司進行交易時的利益衝突問題。一般情況下，公司之代表人為董事長，但當董事個人（或董事代表的第三人）要與公司進行交易時，由董事長代表公司顯然不妥當，因為董事長本身也是董事會成員，可能與該交易董事有利害關係。

因此，法律規定在此類情形下，改由監察人代表公司。監察人作為獨立的監督機關，可以客觀地評估交易條件是否公平合理，確保公司利益不受侵害。

本條所指的法律行為範圍很廣，包括買賣、借貸及「其他法律行為」，涵蓋一切董事與公司之間的交易。不僅限於董事「為自己」之交易，也包括董事「為他人」（例如董事代理他人）與公司之間的交易。

此規定為強制規定，不得以章程或股東會決議排除。違反本條由非監察人代表公司所為之交易，其效力有爭議，通說認為對公司不生效力（相對無效）。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 董事為自己或他人與公司為交易時 |
| 核心規定 | 由監察人為公司之代表 |
| 違反效果 | 未由監察人代表之交易對公司不生效力 |
| 注意事項 | 涵蓋買賣、借貸及所有法律行為 |

## 實務案例

**情境：** 寅公司董事B擬以個人名義向公司購買一筆不動產。董事長A直接代表公司與B簽約，未經監察人參與。

**分析：** 依本條規定，董事B為自己與公司為買賣行為時，應由監察人為公司之代表。本案由董事長A代表公司簽約，違反本條強制規定。通說認為此交易對公司不生效力，B不能據此取得不動產所有權。即使買賣價格合理公平，仍應由監察人代表公司重新簽約。

**結論：** 董事與公司之間任何交易，必須由監察人代表公司，否則交易效力有瑕疵。實務上建議由監察人審核交易條件後，以公司代表人身分簽署相關契約。
