# 第 227 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 第一百九十六條至第二百條、第二百零八條之一、第二百十四條及第二百十五條之規定，於監察人準用之。

## 白話解說

本條是一項重要的準用規定，將原本適用於董事的多項規定擴大適用於監察人，使監察人在報酬、解任、訴訟等方面有明確的法律依據。

準用的條文及其內容如下：

1. **第196條**（報酬）：監察人之報酬，未經章程訂明者，應由股東會議定。不得事後追認。
2. **第197條**（持股轉讓限制）：監察人之股份轉讓限制及設質規定。
3. **第198條**（選舉方式）：監察人之選舉採累積投票制。
4. **第199條**（解任）：監察人得由股東會決議解任。
5. **第200條**（解任之損害賠償）：監察人被無正當理由解任時之損害賠償。
6. **第208-1條**（缺額補選）：監察人缺額達法定人數時之補選規定。
7. **第214條**（代位訴訟）：股東得為公司對監察人提起代位訴訟。
8. **第215條**（訴訟費用擔保）：代位訴訟之費用擔保規定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人 |
| 核心規定 | 董事相關之報酬、解任、選舉、訴訟等規定準用於監察人 |
| 違反效果 | 依各該準用條文之規定 |
| 注意事項 | 應注意準用時條文內容之適當調整 |

## 實務案例

**情境：** 午公司股東會以臨時動議方式，決議將監察人月薪從五萬元調高為十萬元。監察人D聲稱薪資調整既經股東會決議即為合法。

**分析：** 依本條準用第196條之規定，監察人之報酬未經章程訂明者，應由股東會議定。本案股東會雖有決議調高報酬，但須注意是否符合股東會召集程序（如臨時動議是否合法列入議程等）。若章程已有明訂報酬標準，則應先修改章程。

**結論：** 監察人報酬之訂定準用董事報酬規定，應依法由章程訂明或經股東會議定，以確保報酬之合理性及程序之合法性。
