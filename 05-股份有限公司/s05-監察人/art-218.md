# 第 218 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人應監督公司業務之執行，並得隨時調查公司業務及財務狀況，查核、抄錄或複製簿冊文件，並得請求董事會或經理人提出報告。
>
> 監察人辦理前項事務，得代表公司委託律師、會計師審核之。
>
> 違反第一項規定，規避、妨礙或拒絕監察人檢查行為者，各處新臺幣二萬元以上十萬元以下罰鍰。

## 白話解說

本條是監察人最核心的權限規定，賦予監察人廣泛的監督調查權。監察人的首要職責就是監督公司業務之執行，確保董事會及經理人依法、依章程運作公司。

為了有效履行監督職責，法律賦予監察人「隨時」調查公司業務及財務狀況的權力，不受時間限制。監察人可以查核公司帳冊、文件，並可抄錄或複製相關資料。此外，監察人有權要求董事會或經理人提出業務報告，以瞭解公司營運狀況。

考量到部分調查事項可能需要專業知識，本條允許監察人代表公司委託律師或會計師進行審核。此費用由公司負擔，而非監察人個人支出。

為確保監察人調查權能有效行使，本條第三項設有罰則：凡規避、妨礙或拒絕監察人檢查行為者，將處以新臺幣二萬元以上十萬元以下之罰鍰。此罰則適用於公司內部任何阻礙監察人行使職權之人。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人、董事、經理人 |
| 核心規定 | 監察人有權隨時調查公司業務及財務，查核簿冊文件，並請求董事會報告 |
| 違反效果 | 規避、妨礙或拒絕者處新臺幣二萬元以上十萬元以下罰鍰 |
| 注意事項 | 監察人得委託律師、會計師代為審核，費用由公司負擔 |

## 實務案例

**情境：** 戊公司監察人懷疑公司有不當關係人交易，要求財務部門提供近三年所有關係人交易之明細帳冊。財務經理以「資料涉及營業秘密」為由拒絕提供。

**分析：** 依本條規定，監察人有權隨時調查公司業務及財務狀況，並得查核、抄錄或複製簿冊文件。財務經理不得以「營業秘密」為由拒絕監察人之檢查。監察人係公司內部監督機關，本有知悉公司一切業務之權限。財務經理之拒絕行為構成本條第三項之「拒絕監察人檢查行為」，可處新臺幣二萬元以上十萬元以下罰鍰。

**結論：** 公司內部人員不得以任何理由阻礙監察人行使調查權。監察人亦可委託會計師進行更專業的審核，以釐清關係人交易是否合法。
