# 第 224 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人因忽視任務，致公司受有損害者，對公司負賠償之責。

## 白話解說

本條規定監察人因怠忽職務而致公司受損害時之賠償責任。監察人依法負有監督公司業務執行之義務，若監察人未善盡其監督職責，導致公司遭受損害，監察人應對公司負損害賠償責任。

所謂「忽視任務」，是指監察人未盡其法定之注意義務而言，包括但不限於：未依第218條行使調查權而未發現公司重大弊端、依第219條查核財務報表時未盡注意義務、發現董事違法行為未依第218-2條通知停止等。

此責任為過失責任，而非無過失責任。監察人必須有怠忽職務之過失，且公司確實因此受有損害，並有因果關係存在，監察人始須負賠償責任。若監察人已盡善良管理人之注意義務而仍無法防止損害發生者，則不負賠償責任。

賠償範圍依民法規定，以公司所受之損害及所失之利益為限。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人 |
| 核心規定 | 監察人因忽視任務致公司受損害者，對公司負賠償責任 |
| 違反效果 | 負損害賠償責任 |
| 注意事項 | 須有怠忽職務之過失、公司受損害及因果關係 |

## 實務案例

**情境：** 卯公司監察人連續數年未查核公司財務報表，也未行使任何監察權。期間公司財務長挪用公司資金達數千萬元。

**分析：** 依本條規定，監察人之法定義務包括查核財務報表、監督公司業務等。本案監察人連年未行使任何監察權，屬於嚴重「忽視任務」。若監察人有適當行使查核權，應可及早發現財務長挪用公款之行為。監察人之怠忽與公司損害間具有因果關係，監察人應對公司負賠償責任。

**結論：** 監察人不可僅掛名而不行使任何監察權。即使公司有專業會計師進行簽證，監察人仍有獨立查核之義務。完全不作為的監察人，在公司發生損害時將難逃賠償責任。
