# 第 219 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第五節 監察人

## 條文原文

> 監察人對於董事會編造提出股東會之各項表冊，應予查核，並報告意見於股東會。
>
> 監察人辦理前項事務，得委託會計師審核之。
>
> 監察人違反第一項規定而為虛偽之報告者，各處新臺幣六萬元以下罰金。

## 白話解說

本條規定監察人對公司財務報表的查核義務。每個會計年度終了，董事會依第228條編造營業報告書、財務報表及盈餘分派或虧損撥補議案後，這些表冊在提交股東會承認之前，必須先經監察人查核。監察人查核後應將其查核意見向股東會報告，讓股東在表決是否承認這些表冊時有參考依據。

由於財務報表的查核需要相當的專業知識，本條第二項允許監察人委託會計師來協助審核工作，相關費用由公司負擔。

為確保監察人確實善盡查核義務而非只是橡皮圖章，本條第三項設有刑事罰則：監察人若對表冊查核結果做出虛偽報告（例如明知財務報表有不實情事卻報告正常），將處以新臺幣六萬元以下之罰金。此為刑事罰金而非行政罰鍰，屬於較嚴厲之制裁。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之監察人 |
| 核心規定 | 監察人應查核董事會編造之表冊，並向股東會報告意見 |
| 違反效果 | 虛偽報告者處新臺幣六萬元以下罰金（刑事責任） |
| 注意事項 | 得委託會計師審核；此為法定義務非任意行為 |

## 實務案例

**情境：** 辛公司監察人在查核年度財務報表時，發現公司虛增營業收入達兩千萬元。但因監察人與董事長私交甚好，在股東會上仍報告「查核無異常」。事後公司財務造假案爆發。

**分析：** 依本條第三項，監察人明知財務報表有虛增營業收入之不實情形，卻在股東會做出「查核無異常」之虛偽報告，已觸犯本條刑事處罰規定，可處新臺幣六萬元以下罰金。此外，監察人可能另外面臨第224條之損害賠償責任，以及證券交易法之相關刑事責任（若為公開發行公司）。

**結論：** 監察人對財務報表之查核是法定義務，不得流於形式。明知表冊有虛偽而做出不實報告，不僅有刑事責任，更須對公司及股東負損害賠償責任。
