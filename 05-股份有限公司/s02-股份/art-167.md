# 第 167 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 公司除依第一百五十八條、第一百六十七條之一、第一百八十六條及第三百十七條規定外，不得自將股份收回、收買或收為質物。但於股東清算或受破產之宣告時，得按市價收回其股份，抵償其於清算或破產宣告前結欠公司之債務。
>
> 公司依前項但書規定，收回或收買之股份，應於六個月內，按市價將其出售，届期未經出售者，視為公司未發行股份，並為變更登記。
>
> 被持有已發行有表決權之股份總數或資本總額超過半數之從屬公司，不得將控制公司之股份收買或收為質物。
>
> 前項控制公司及其從屬公司直接或間接持有他公司已發行有表決權之股份總數或資本總額合計超過半數者，他公司亦不得將控制公司及其從屬公司之股份收買或收為質物。

## 白話解說

這條規定了公司收回自己股份的嚴格限制，俗稱「庫藏股限制」：

1. **原則禁止**：公司原則上不得收回、收買自己的股份，也不得以自己的股份作為質物。這是為了維護資本充實原則
2. **例外允許**：只有在以下法定情形下才能收回：
   - 收回特別股（第 158 條）
   - 買回庫藏股轉讓員工（第 167-1 條）
   - 股東行使股份收買請求權（第 186 條）
   - 合併時反對股東之股份收買（第 317 條）
3. **股東清算/破產**：股東個人進入清算或破產時，公司得按市價收回其股份抵償債務
4. **六個月處分義務**：收回的股份必須在六個月內按市價出售，否則視為未發行股份
5. **從屬公司限制**：從屬公司（子公司）不得收買控制公司（母公司）的股份，以防止循環持股

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有股份有限公司；控制公司與從屬公司 |
| 核心規定 | 原則禁止收回自己股份；例外情形法定列舉；從屬公司不得收買控制公司股份 |
| 違反效果 | 違反規定收回之股份無效；從屬公司違法收買者，其收買行為亦屬違法 |
| 注意事項 | 收回之股份應於六個月內出售；從屬公司之限制延伸至間接控制關係 |

## 實務案例

**情境：** X 公司（母公司）持有 Y 公司（子公司）60% 股份。Y 公司擬在公開市場買入 X 公司股票作為投資。

**分析：** X 公司持有 Y 公司超過 50% 的有表決權股份，Y 公司為 X 公司的從屬公司。依第 167 條第 3 項，Y 公司不得收買 X 公司（控制公司）的股份。即使 Y 公司認為 X 公司股票有投資價值，法律也禁止此行為，因為這實質上等於公司間接收買自己的股份。

**結論：** 從屬公司收買控制公司股份將構成違法行為，公司負責人可能面臨法律責任。
