# 第 159 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 公司已發行特別股者，其章程之變更如有損害特別股股東之權利時，除應有代表已發行股份總數三分之二以上股東出席之股東會，以出席股東過半數之決議為之外，並應經特別股股東會之決議。
>
> 前項特別股股東會之決議，準用第一百七十五條之規定。

## 白話解說

這條是保護特別股股東的重要規定：

1. **雙重決議保護**：當公司要修改章程，而修改內容會損害特別股股東的權利時，必須經過兩道關卡：
   - **第一關**：股東會特別決議（2/3 以上股東出席、出席過半數同意）
   - **第二關**：特別股股東會決議（單獨由特別股股東開會表決）
2. **特別股股東會**：特別股股東可以獨立召開自己的股東會，對損害其權利的章程變更案進行表決，準用一般股東會的決議規定
3. **權利保護**：如果特別股股東會不同意，即使普通股股東會通過了，章程變更仍然無法生效

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 已發行特別股之股份有限公司 |
| 核心規定 | 章程變更損害特別股權利時，須經股東會特別決議及特別股股東會決議 |
| 違反效果 | 未經特別股股東會決議之章程變更，對特別股股東不生效力 |
| 注意事項 | 特別股股東會準用第 175 條一般股東會決議規定；須判斷章程變更是否「損害」特別股權利 |

## 實務案例

**情境：** J 公司已發行優先分派股利的特別股（年息 8%），現因經營困難，擬修改章程將特別股股利降為 5%。

**分析：** 將特別股股利從 8% 降為 5% 明顯損害特別股股東的權利，須依第 159 條進行雙重決議。首先需召開股東會，以特別決議通過章程變更案；其次需單獨召開特別股股東會，由特別股股東表決同意。若特別股股東會不通過，則章程不得變更。

**結論：** 公司不能單方面降低特別股的股利，必須取得特別股股東的同意。這是法律對特別股股東的重要保護機制。
