# 第 161 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 公司非經設立登記或發行新股變更登記後，不得發行股票。但公開發行股票之公司，證券管理機關另有規定者，不在此限。
>
> 違反前項規定發行股票者，其股票無效。但持有人得向發行股票人請求損害賠償。

## 白話解說

這條規範股票發行的時間限制：

1. **登記前不得發行**：公司必須先完成設立登記（新設公司）或發行新股變更登記（增資公司）之後，才能正式發行股票。這是為了確保公司的合法性和資本的真實性
2. **公開發行例外**：公開發行公司由於受到證券主管機關（金管會）的特別監理，金管會另有規定者不受此限
3. **違反效果**：未經登記就發行的股票是「無效」的，但善意的持有人可以向違法發行股票的人請求損害賠償

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有股份有限公司 |
| 核心規定 | 非經設立登記或發行新股變更登記後，不得發行股票 |
| 違反效果 | 股票無效；持有人得向發行人請求損害賠償 |
| 注意事項 | 公開發行公司得依證券管理機關規定辦理；違法發行者應負損害賠償責任 |

## 實務案例

**情境：** L 公司正在辦理設立登記程序中，創辦人為了加速向投資人募資，在登記完成前就先印製股票發給投資人。

**分析：** 依第 161 條，公司非經設立登記後不得發行股票。L 公司在設立登記完成前即發行股票，違反本條規定，該等股票無效。但投資人（持有人）得向創辦人（發行股票人）請求損害賠償。

**結論：** 創辦人急於募資的心情可以理解，但必須等設立登記完成後才能發行股票，否則股票無效，還要承擔損害賠償責任。
