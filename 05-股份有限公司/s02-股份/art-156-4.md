# 第 156-4 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 公司為改善財務結構或回復正常營運，而參與主管機關主導之紓困方案時，得經董事會三分之二以上董事出席，以出席董事過半數決議，發行新股轉讓於政府，作為接受政府財務上協助之對價，不受第二百六十七條第一項至第三項、第二百七十條及前條第二項之限制。
>
> 前項發行新股應洽特定人認購，不得公開招募。

## 白話解說

這條是在經濟危機或企業困難時的特殊救濟規定：

1. **紓困條款**：當公司財務狀況惡化，為了改善財務結構或恢復正常營運，參與政府主導的紓困方案時，可以發行新股給政府
2. **決議門檻**：需經董事會特別決議（2/3 以上董事出席、出席過半數同意），效率較高，不需經股東會
3. **排除限制**：不受員工優先認購權、公開招募限制，以及股份交換上限（100%）的限制，因為緊急紓困需要快速執行
4. **洽特定人**：新股只能由政府認購，不得對外公開招募，避免紓困過程中的複雜性

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 參與政府紓困方案之股份有限公司 |
| 核心規定 | 得經董事會特別決議發行新股轉讓於政府，作為接受財務協助之對價 |
| 違反效果 | 未經法定決議程序或未符合紓困要件者，發行無效 |
| 注意事項 | 僅限政府紓困情境；須洽特定人認購、不得公開招募；排除多項限制以利緊急處理 |

## 實務案例

**情境：** F 航空公司因全球疫情導致營收大幅下滑，資金週轉困難，政府推出航空業紓困方案。F 公司希望參與紓困方案，以發行新股方式取得政府的財務協助。

**分析：** F 公司符合「改善財務結構或回復正常營運」的要件，且紓困方案係由主管機關主導。F 公司可經董事會特別決議，發行新股轉讓予政府。此時不受員工優先認購、發行上限等限制，以利快速取得紓困資金。

**結論：** F 公司可合法發行新股給政府，但需注意政府因此取得股份後可能影響公司經營決策，待公司恢復正常後，政府通常會出售持股退場。
