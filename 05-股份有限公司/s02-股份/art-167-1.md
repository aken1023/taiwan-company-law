# 第 167-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 公司除法律另有規定者外，得經董事會以董事三分之二以上之出席及出席董事過半數同意之決議，於不超過該公司已發行股份總數百分之五之範圍內，收買其股份；收買股份之總金額，不得逾保留盈餘加已實現之資本公積之金額。
>
> 公司依前項規定收買之股份，應於三年內轉讓於員工，届期未轉讓者，視為公司未發行股份，並為變更登記。
>
> 公司依第一項規定收買之股份，不得享有股東權利。

## 白話解說

這條建立了「庫藏股轉讓員工」制度：

1. **收買條件**：
   - 需經董事會特別決議（2/3 以上董事出席、出席過半數同意）
   - 收買數量不得超過已發行股份總數的 5%
   - 收買總金額不得超過保留盈餘加已實現資本公積
2. **轉讓期限**：收買的股份必須在三年內轉讓給員工，逾期未轉讓者視為未發行股份，要做變更登記（等於註銷）
3. **無股東權利**：公司持有自己的庫藏股期間，這些股份不享有任何股東權利（不能投票、不能領股利等）
4. **目的**：讓公司能以低於市價的方式，將庫藏股轉讓給員工作為獎勵，提升員工向心力

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有股份有限公司 |
| 核心規定 | 得經董事會特別決議收買不超過 5% 股份轉讓予員工 |
| 違反效果 | 超過 5% 上限或超過金額限制者，超出部分之收買無效 |
| 注意事項 | 三年內須轉讓於員工；庫藏股期間不享有股東權利；收買金額有上限 |

## 實務案例

**情境：** Z 公司已發行 1,000 萬股，保留盈餘 5,000 萬元、已實現資本公積 2,000 萬元。公司擬以每股 20 元收買 200 萬股作為員工獎勵。

**分析：**
- 數量檢查：200 萬股 = 已發行 1,000 萬股的 20%，超過 5% 上限（50 萬股），不合規
- 金額檢查：200 萬股 x 20 元 = 4,000 萬元，未超過保留盈餘 + 已實現資本公積（7,000 萬元）
- Z 公司最多只能收買 50 萬股（1,000 萬股 x 5%）

**結論：** Z 公司應調降收買股數至 50 萬股以內，才符合法定上限。
