# 第 167-3 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 公司依第一百六十七條之一或其他法律規定收買自己之股份轉讓於員工者，得限制員工在一定期間內不得轉讓。但其期間最長不得超過二年。

## 白話解說

這條允許公司對員工取得庫藏股後的轉讓行為加以限制：

1. **轉讓限制**：公司將庫藏股轉讓給員工時，可以約定員工在一定期間內不能轉賣這些股份
2. **最長二年**：限制轉讓的期間最長不得超過二年
3. **立法目的**：如果員工取得庫藏股後馬上賣掉，就無法達到激勵員工長期留任的目的。適度的轉讓限制可以確保員工與公司利益長期綁定
4. **強制上限**：為了保護員工的財產權，法律設定最長二年的限制，公司不得約定超過此期限

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 依第 167-1 條或其他法律規定取得庫藏股之員工 |
| 核心規定 | 得限制員工在最長二年內不得轉讓所取得之股份 |
| 違反效果 | 超過二年之轉讓限制約定，超過部分無效 |
| 注意事項 | 限制期間最長二年為強制規定；限制應於轉讓時明確約定 |

## 實務案例

**情境：** B1 公司將庫藏股以優惠價格轉讓予員工，並約定員工三年內不得轉讓。

**分析：** 依第 167-3 條，轉讓限制最長不得超過二年。B1 公司約定三年，超過法定上限。超過二年的部分無效，即員工在取得股份滿二年後即可自由轉讓，不受第三年限制之拘束。

**結論：** 公司設定轉讓限制期間時，應注意不得超過法定二年上限。若希望更長期綁定員工，應考慮其他機制（如分期給予、績效條件等）。
