# 第 169 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 股東名簿應編號記載下列事項：
>
> 一、各股東之姓名或名稱、住所或居所。
>
> 二、各股東之股數及其股票號數。
>
> 三、發給股票之年、月、日。
>
> 四、發行無記名股票者，應記載其股數、號數及發給之年、月、日。
>
> 五、發行特別股者，並應註明特別股之種類。
>
> 採行無票面金額股之公司，股東名簿應記載所繳納股款之金額。

## 白話解說

這條規定了股東名簿上必須記載的法定事項：

1. **股東基本資料**：每位股東的姓名（自然人）或名稱（法人）、住所或居所
2. **持股資料**：每位股東持有的股數及對應的股票號碼
3. **發給日期**：股票發給的年月日
4. **無記名股票**：如有發行無記名股票，應記載其股數、號碼及發給日期
5. **特別股註記**：如有發行特別股，應在名簿上註明特別股的種類
6. **無票面金額股**：如果公司採行無票面金額股，還要記載每位股東所繳納的股款金額

股東名簿是公司最重要的文件之一，它是公司辨識股東身分、行使股東權利的基本依據。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有股份有限公司 |
| 核心規定 | 股東名簿應編號記載股東姓名住所、股數號數、發給日期、特別股種類等事項 |
| 違反效果 | 未依法記載者，可能影響股東權利之認定及公司治理之運作 |
| 注意事項 | 無票面金額股應額外記載繳納股款金額；2018 年修法配合新制度增訂 |

## 實務案例

**情境：** E1 公司的股東名簿因管理疏失，部分股東的住所資料已過期未更新，導致股東會開會通知無法送達。

**分析：** 依第 169 條，股東名簿應記載各股東的住所或居所。如果資料不正確，可能導致股東會通知無法送達，影響股東出席權利，甚至可能導致股東會決議因程序瑕疵而被撤銷。公司應定期要求股東更新資料，並主動維護股東名簿的正確性。

**結論：** 股東名簿的正確維護是公司治理的基礎工作。公司應建立機制定期更新股東資料，股東也有義務在資料變更時主動通知公司。
