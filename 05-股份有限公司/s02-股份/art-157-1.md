# 第 157-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 非公開發行股票之公司發行複數表決權特別股或對於特定事項具否決權特別股者，應符合下列規定：
>
> 一、該特別股股東，以對公司有重大貢獻或為維持公司信用所必要之人為限。
>
> 二、公司章程應載明特別股之權利義務及其行使之條件。
>
> 三、複數表決權特別股或特定事項否決權特別股之股東，對於監察人選舉，與普通股股東之表決權同。

## 白話解說

這條是對非公開發行公司發行「超級表決權」特別股的規範：

1. **僅限非公開發行公司**：公開發行公司不能發行複數表決權或否決權特別股，因為會損害公眾投資人的權益
2. **對象限制**：只有對公司有重大貢獻（如核心技術創辦人）或為維持公司信用所必要的人（如關鍵投資人），才能持有這種特別股
3. **章程載明**：權利義務和行使條件必須在章程中明確記載
4. **監察人選舉平等**：即使持有複數表決權特別股，在選舉監察人時，表決權與普通股相同（一股一票），這是為了確保監察人能真正發揮監督功能

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 非公開發行之股份有限公司 |
| 核心規定 | 複數表決權及否決權特別股僅限對公司有重大貢獻或維持信用必要之人持有 |
| 違反效果 | 不符資格者持有複數表決權特別股，其額外表決權部分無效 |
| 注意事項 | 選舉監察人時複數表決權不適用，回歸一股一票；公開發行公司不適用 |

## 實務案例

**情境：** H 科技公司（非公開發行）由兩位核心技術創辦人成立，後引進財務投資人。創辦人擔心經營權被稀釋，希望透過複數表決權特別股維持經營控制。

**分析：** 兩位創辦人為公司核心技術提供者，屬於「對公司有重大貢獻」之人，符合持有複數表決權特別股的資格。公司章程應明確記載複數表決權的倍數（例如每股 3 票）及行使條件。但在選舉監察人時，創辦人的特別股只能一股一票，與普通股相同。

**結論：** H 公司可合法發行複數表決權特別股給創辦人，但設計時應注意監察人選舉的限制，並確保章程記載完整。
