# 第 156-2 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 公司得依董事會之決議，向證券管理機關申請辦理公開發行程序。
>
> 公開發行股票之公司已不宜繼續公開發行時，得經股東會特別決議，向證券管理機關申請停止公開發行。

## 白話解說

這條規定了公司「公開發行」的進入與退出機制：

1. **申請公開發行**：公司想要讓股票在公開市場流通，只需要經過董事會決議，就可以向金管會證期局申請公開發行。門檻相對較低，因為公開發行有助於公司籌資
2. **停止公開發行**：如果公司認為繼續公開發行已不合適（例如股東人數太少、營運規模縮小等），可以申請「下市」或停止公開發行，但這需要股東會特別決議（2/3 出席、過半同意），門檻較高，以保護投資大眾

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有股份有限公司（申請公開發行）；公開發行公司（停止公開發行） |
| 核心規定 | 公開發行由董事會決議即可；停止公開發行須股東會特別決議 |
| 違反效果 | 未經法定程序擅自公開發行或停止公開發行者，該行為不生效力 |
| 注意事項 | 進入門檻低（董事會）、退出門檻高（股東會特別決議），以保障投資人 |

## 實務案例

**情境：** C 公司為公開發行公司，因經營策略調整，大股東希望將公司「私有化」，停止公開發行以減少法遵成本。

**分析：** 依第 156-2 條第 2 項，停止公開發行須經股東會特別決議。C 公司必須召開股東會，取得已發行股份總數三分之二以上股東出席、出席股東過半數同意，才能向金管會申請停止公開發行。大股東不能單方面透過董事會決議就停止公開發行。

**結論：** 法律刻意提高停止公開發行的門檻，是為了保護廣大的小股東權益，避免大股東任意將公司私有化而損害小股東利益。
