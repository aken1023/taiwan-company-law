# 第 161-2 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 發行股票之公司，其發行之股份得免印製股票，應洽證券集中保管事業機構登錄。
>
> 前項股份之轉讓，不適用第一百六十四條規定。
>
> 前二項規定，於公司登記後，尚未印製股票者，適用之。

## 白話解說

這條建立了「無實體股票」（帳簿劃撥）制度：

1. **免印製股票**：公司發行股份時，不一定要印製實體紙本股票，可以改由證券集中保管事業（即臺灣集中保管結算所）以電子登錄方式管理
2. **轉讓方式改變**：無實體股票的轉讓不需要用背書方式（第 164 條），而是透過帳簿劃撥的方式辦理，更加便利安全
3. **尚未印製者適用**：如果公司已登記但尚未印製實體股票，同樣可以適用無實體登錄制度

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 發行股票之股份有限公司（含已登記但未印製股票者） |
| 核心規定 | 得免印製股票，改由證券集中保管事業機構登錄 |
| 違反效果 | 不涉及違反問題，屬公司可選擇之制度 |
| 注意事項 | 無實體股票之轉讓不適用背書轉讓規定；上市櫃公司實務上均已採行無實體制度 |

## 實務案例

**情境：** N 公司原本有印製實體股票，現決定改採無實體股票制度以節省成本及提升管理效率。

**分析：** N 公司可依第 161-2 條，洽臺灣集中保管結算所辦理股份登錄。改採無實體制度後，股份轉讓以帳簿劃撥方式進行，不再需要實體股票的背書交付。原有的實體股票應予以回收註銷。

**結論：** 無實體股票制度大幅降低了印製成本、遺失風險及偽造可能性，是現代化公司治理的重要工具。目前臺灣上市櫃公司已全面實施無實體股票。
