# 第 162 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第二節 股份

## 條文原文

> 股票應編號，載明下列事項，由代表公司之董事簽名或蓋章，並經依法得擔任股票發行簽證人之銀行簽證後發行之：
>
> 一、公司名稱。
>
> 二、設立登記或發行新股變更登記之年、月、日。
>
> 三、採行票面金額股者，股份總數及每股金額；採行無票面金額股者，股份總數。
>
> 四、本次發行股份數。
>
> 五、公司採行無票面金額股者，應載明無票面金額。
>
> 六、發起人股票應標明發起人股票之字樣。
>
> 七、特別股票應標明其特別種類之字樣。
>
> 八、公司依第一百六十一條之二規定免印製股票者，免標明前二款事項。

## 白話解說

這條規定了實體股票上必須記載的內容：

1. **編號**：每張股票都要有唯一的編號，便於管理追蹤
2. **法定記載事項**：
   - 公司名稱
   - 登記日期
   - 股份總數及面額（票面金額股）或股份總數（無票面金額股）
   - 本次發行股數
   - 無票面金額股應註明
   - 發起人股票要特別標示
   - 特別股票要標明種類
3. **簽名蓋章**：需要代表公司的董事簽名或蓋章
4. **銀行簽證**：必須經過合格的銀行簽證後才能發行，這是防止偽造的機制
5. **免印製例外**：採用無實體股票制度者，免標明發起人股票及特別股字樣

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 印製實體股票之股份有限公司 |
| 核心規定 | 股票應編號、載明法定事項、經董事簽章及銀行簽證 |
| 違反效果 | 未依法定格式記載者，股票之效力可能產生疑義 |
| 注意事項 | 2018 年修法配合無票面金額股制度增訂相關記載事項；免印製股票者部分免除 |

## 實務案例

**情境：** O 公司印製實體股票時，漏未經銀行簽證即交付給股東。

**分析：** 依第 162 條，股票必須經依法得擔任股票發行簽證人之銀行簽證後方能發行。未經銀行簽證的股票，欠缺法定要件，其效力有疑慮。實務上認為銀行簽證是股票的「形式要件」，未經簽證者可能被認定為不合法的股票。

**結論：** 公司發行實體股票時，務必確保完成銀行簽證程序，否則股票的有效性將受到質疑。
