# 第 352 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算進行中，有下列情事之一者，法院得因清算人、監察人、股東或債權人之聲請或依職權，宣告公司破產：
> 一、特別清算有顯著困難時。
> 二、公司負債超過資產有不實之嫌疑時。

## 白話解說

在特別清算進行過程中，如果出現以下兩種情形之一：特別清算的進行遇到明顯困難（例如債權人之間對協定始終無法達成共識），或者有嫌疑顯示公司的負債超過資產的情況不實（例如帳目有虛偽之嫌），法院可以依據清算人、監察人、股東或債權人的聲請，或主動依職權，宣告公司破產。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之公司 |
| 核心規定 | 特別清算有顯著困難或負債超過資產有不實嫌疑時，得宣告破產 |
| 違反效果 | 轉入破產程序 |
| 注意事項 | 法院得依職權宣告；聲請人包括清算人、監察人、股東或債權人 |

## 實務案例

**情境：** 甲公司特別清算中，債權人會議已召開三次均無法通過協定，且清算人發現公司帳冊有多處不實記載。

**分析：** 依第352條，特別清算有顯著困難（協定屢次無法通過）且公司負債超過資產有不實嫌疑（帳冊不實），符合宣告破產之要件。

**結論：** 法院得因利害關係人之聲請或依職權，宣告甲公司破產。
