# 第 322 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 公司之清算，以董事為清算人。但本法或章程另有規定或股東會另選清算人時，不在此限。
>
> 前項清算人，應於就任後十五日內，將其姓名、住所或居所及就任日期，向法院聲報。
>
> 清算人之解任，應由股東會之決議為之，但法院亦得因利害關係人之聲請，將清算人解任。

## 白話解說

公司解散後進入清算程序時，原則上由公司的董事擔任清算人來負責處理清算事務。但如果公司法另有規定、章程有特別約定、或股東會另外選任了清算人，就不一定由董事擔任。清算人就任後15天內要向法院報備其姓名、住所和就任日期。清算人的解任，可以由股東會決議，或者由利害關係人向法院聲請解任。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 解散之股份有限公司 |
| 核心規定 | 以董事為法定清算人，章程或股東會得另選；清算人就任15日內向法院聲報 |
| 違反效果 | 未向法院聲報者，依第387條得處罰鍰 |
| 注意事項 | 清算人解任可由股東會決議或法院因利害關係人聲請為之 |

## 實務案例

**情境：** 甲公司股東會決議解散，但全體董事均辭職不願擔任清算人。少數股東擔心公司資產遭侵害，向法院聲請選任清算人。

**分析：** 依第322條第1項，以董事為清算人。董事辭職拒絕擔任時，股東會得另選清算人。如無法順利選任，利害關係人得向法院聲請選任清算人。

**結論：** 法院得依利害關係人之聲請選任適當之人為清算人，以確保清算程序順利進行。
