# 第 330 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 賸餘財產之分派，除章程另有規定外，按各股東股份比例分派之。

## 白話解說

公司清算完畢後，清償所有債務還有剩餘的財產，稱為「賸餘財產」。這些賸餘財產原則上要按照各股東所持有的股份比例來分配。但如果公司章程另有不同的規定（例如特別股有優先分配權），就依章程辦理。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算中公司之股東 |
| 核心規定 | 賸餘財產按股份比例分派，章程另有規定者從其規定 |
| 違反效果 | 不適用 |
| 注意事項 | 特別股之賸餘財產分配優先順序得依章程另定 |

## 實務案例

**情境：** 甲公司清算後賸餘財產為1000萬元。股東A持有60%股份、B持有40%股份。但章程規定B持有的特別股就賸餘財產有優先分配300萬元之權利。

**分析：** 依第330條，章程另有規定者從其規定。B之特別股有優先分配300萬元之權利，優先分配後剩餘700萬元再按股份比例分配。

**結論：** B先獲300萬元，剩餘700萬元按比例分配：A得420萬元（700萬 x 60%），B再得280萬元（700萬 x 40%），合計B獲580萬元。
