# 第 340 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算開始後，清算人於每三個月應將清算狀況報告法院一次。

## 白話解說

進入特別清算後，清算人必須每三個月向法院報告一次清算的進度和狀況。這是定期報告制度，讓法院持續掌握清算進展，確保清算程序按時完成。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之清算人 |
| 核心規定 | 每三個月向法院報告清算狀況 |
| 違反效果 | 未定期報告者，清算人違反法定義務 |
| 注意事項 | 報告頻率為每三個月一次 |

## 實務案例

**情境：** 甲公司特別清算已進行九個月，但清算人僅於第一個三個月報告過一次，其後未再報告。

**分析：** 依第340條，清算人應每三個月向法院報告清算狀況。九個月內至少應報告三次，清算人僅報告一次，違反定期報告義務。

**結論：** 法院得命清算人補提報告，情節嚴重者得依職權將清算人解任。
