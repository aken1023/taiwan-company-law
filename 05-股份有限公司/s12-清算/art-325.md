# 第 325 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 清算人於執行前條職務之範圍內，除本節有規定外，其權利義務與董事同。

## 白話解說

清算人在執行清算職務的範圍內，享有與董事相同的權利並負擔相同的義務。也就是說，清算人可以代表公司、處理清算事務，就像董事在公司正常營運時一樣。但清算人的權限僅限於清算事務範圍，不包括新的營業行為。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算人 |
| 核心規定 | 清算人之權利義務與董事同（限於清算職務範圍內） |
| 違反效果 | 清算人違反義務者，比照董事之責任規定 |
| 注意事項 | 僅在清算職務範圍內準用董事之權利義務；清算人不得從事新營業行為 |

## 實務案例

**情境：** 甲公司清算人為了結現務，以公司名義與客戶簽訂最後一批出貨合約。有股東質疑清算人無權對外簽約。

**分析：** 依第325條，清算人於執行清算職務範圍內，權利義務與董事同。了結現務為第324條所定清算人職務之一，簽訂合約以完成最後出貨屬於了結現務之範圍。

**結論：** 清算人有權為了結現務而簽訂合約，股東之質疑無理由。
