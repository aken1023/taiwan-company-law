# 第 339 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算開始後，清算人應將特別清算開始前公司業務及財產之狀況，向法院報告之。

## 白話解說

特別清算開始後，清算人有義務向法院報告公司在特別清算開始前的業務狀況和財產狀況。這是為了讓法院充分了解公司的實際情形，以便有效監督清算程序。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之清算人 |
| 核心規定 | 清算人應向法院報告公司業務及財產狀況 |
| 違反效果 | 未報告者，清算人違反法定義務 |
| 注意事項 | 報告內容為特別清算開始「前」之公司狀況 |

## 實務案例

**情境：** 甲公司進入特別清算，清算人向法院提出之報告未包含公司近三年之財務變動情形及重大交易。法院認為報告不完整。

**分析：** 依第339條，清算人應向法院報告公司業務及財產之狀況。報告應涵蓋足以讓法院了解公司實際情形之內容。

**結論：** 法院得要求清算人補充報告，提供更完整之公司業務及財產資訊。
