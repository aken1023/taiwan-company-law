# 第 323 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 不能依前條之規定定清算人時，法院得因利害關係人之聲請，選派清算人。

## 白話解說

如果依照第322條的規定無法確定清算人（例如董事全部缺位、章程未約定、股東會也無法選出清算人），法院可以依據利害關係人（如股東、債權人等）的聲請，由法院來選派適當的人擔任清算人，以確保清算程序能夠進行。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 無法依第322條確定清算人之公司 |
| 核心規定 | 法院得因利害關係人聲請選派清算人 |
| 違反效果 | 不適用 |
| 注意事項 | 利害關係人包括股東、債權人等；法院有裁量權選派適當人選 |

## 實務案例

**情境：** 甲公司因股東不足二人而解散，唯一董事因涉及掏空公司資產遭羈押，無法執行清算人職務。債權人乙銀行向法院聲請選派清算人。

**分析：** 依第323條，因無法依第322條定清算人，法院得因利害關係人（乙銀行為債權人）之聲請選派清算人。法院可選派會計師或律師等專業人士擔任。

**結論：** 法院得受理乙銀行之聲請，選派獨立之專業人士為清算人。
