# 第 347 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 協定經法院認可後，對於全體債權人有效力。

## 白話解說

當清償協定經過債權人會議決議通過，並且獲得法院的認可後，該協定就對所有的債權人產生拘束力，包括那些沒有出席債權人會議或投反對票的債權人。這是為了避免少數債權人阻撓清算程序的進行。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 全體債權人 |
| 核心規定 | 經法院認可之協定，對全體債權人有效力 |
| 違反效果 | 不適用 |
| 注意事項 | 效力及於未出席或投反對票之債權人 |

## 實務案例

**情境：** 甲公司特別清算之協定經債權人會議通過並獲法院認可。債權人丙未出席會議，主張協定不拘束其債權。

**分析：** 依第347條，協定經法院認可後，對全體債權人有效力。丙雖未出席會議，仍受協定拘束。

**結論：** 丙之主張無理由，應依協定之內容受償。
