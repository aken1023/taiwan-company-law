# 第 326 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 清算人就任後，應即檢查公司財產情形，造具資產負債表及財產目錄，送經監察人審查，提請股東會承認後，並即報法院。
>
> 清算人應於六個月內完結清算；不能於六個月內完結清算時，清算人得申敘理由，聲請法院展期。
>
> 清算人不於前項規定期限內清算完結者，各處新臺幣一萬元以上五萬元以下罰鍰。

## 白話解說

清算人上任後，第一件事就是要徹底盤點公司的財產狀況，做出資產負債表和財產目錄，交給監察人審查後，再提交股東會承認，並向法院報備。清算工作應該在六個月內完成。如果六個月內無法完成，清算人可以向法院說明理由申請展期。如果逾期不完成又沒有申請展期，每位清算人會被處一萬到五萬元的罰鍰。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算人 |
| 核心規定 | 就任後應造具資產負債表及財產目錄，經監察人審查、股東會承認後報法院 |
| 違反效果 | 逾六個月未完結清算者，各處新臺幣一萬元以上五萬元以下罰鍰 |
| 注意事項 | 清算期限為六個月，得向法院聲請展期 |

## 實務案例

**情境：** 甲公司清算人就任已七個月，因公司債權債務關係複雜尚未完結清算，也未向法院聲請展期。

**分析：** 依第326條第2項，清算應於六個月內完結。不能於六個月內完結者，清算人應申敘理由聲請法院展期。甲公司清算人逾期且未聲請展期，依第3項應受罰鍰處分。

**結論：** 清算人應儘速向法院聲請展期，並加速清算程序。逾期部分將被處新臺幣一萬元以上五萬元以下罰鍰。
