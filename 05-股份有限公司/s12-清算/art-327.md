# 第 327 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 清算人應於就任後十五日內，以公告方式催告債權人報明債權，對於明知之債權人，應分別通知。
>
> 前項公告應指定一個月以上之期限。

## 白話解說

清算人上任後的15天內，必須公告催告所有債權人來申報債權。如果清算人知道某位債權人的存在，還要個別通知他。公告中必須指定至少一個月的期限，讓債權人在期限內來報明其債權的金額和內容。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算人 |
| 核心規定 | 就任15日內公告催告債權人報明債權；已知債權人應分別通知 |
| 違反效果 | 未通知已知債權人者，清算人應負損害賠償責任 |
| 注意事項 | 公告期限至少一個月 |

## 實務案例

**情境：** 甲公司清算人於就任後第20日才刊登催告公告，且未個別通知已知之主要往來銀行。

**分析：** 依第327條，清算人應於就任後15日內以公告催告債權人，且對明知之債權人應分別通知。甲公司清算人逾期公告且未個別通知已知債權人，均違反法定義務。

**結論：** 清算人違反法定義務，如因此造成債權人損害，應負損害賠償責任。
