# 第 351 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 清算人怠於履行前條之義務時，對公司債權人因此所受之損害，應負賠償之責。

## 白話解說

如果清算人怠慢不履行第350條的義務（即發現公司資不抵債時不聲請破產），因為拖延而導致債權人受到損害（例如公司資產在拖延期間進一步減少），清算人個人要對債權人負損害賠償責任。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 怠於聲請破產之清算人 |
| 核心規定 | 清算人怠於聲請破產致債權人受損害者，應負賠償責任 |
| 違反效果 | 清算人個人負損害賠償責任 |
| 注意事項 | 損害賠償範圍為因怠於聲請破產所造成之損害 |

## 實務案例

**情境：** 甲公司清算人發現公司資不抵債後，仍拖延三個月才聲請破產。在拖延期間，公司之機器設備因未妥善保管而毀損，價值減少500萬元。

**分析：** 依第351條，清算人怠於履行聲請破產之義務，對債權人因此所受之損害應負賠償責任。500萬元的設備損失係因清算人拖延所致。

**結論：** 債權人得向清算人個人請求賠償500萬元之損害。
