# 第 344 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 第三百四十二條之協定，應經債權人會議之決議，其決議以出席債權人過半數之同意，並經出席債權人所代表債權額占債權總額四分之三以上之同意為之。

## 白話解說

清算人提出的清償協定草案，必須經過債權人會議的決議通過。決議的門檻是雙重的：第一，出席的債權人人數要有過半數同意；第二，同意的債權人所代表的債權金額要達到全部債權總額的四分之三以上。這個雙重門檻確保協定既有多數債權人的支持，也有代表多數金額的債權人的同意。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之債權人會議 |
| 核心規定 | 協定決議須出席債權人過半數同意，且同意之債權額占總額3/4以上 |
| 違反效果 | 未達門檻者，協定不成立 |
| 注意事項 | 採雙重門檻：人頭數過半且金額達3/4 |

## 實務案例

**情境：** 甲公司特別清算之債權人會議，共有10位債權人出席（總債權額1億元）。6位債權人投贊成票，但其代表之債權額僅6千萬元。

**分析：** 依第344條，決議須出席債權人過半數同意（6/10 > 1/2，符合）且同意之債權額占總額3/4以上（6千萬/1億 = 60%，未達75%）。

**結論：** 雖然人頭數過半同意，但同意之債權額未達3/4門檻，協定決議不成立。
