# 第 335 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 清算之執行，有顯著困難時，法院得因債權人或清算人之聲請或依職權，命令開始特別清算。
>
> 清算人就任後，有不能清償債務之虞時，法院亦得命令開始特別清算。

## 白話解說

當普通清算的進行遇到明顯的困難（例如債權人眾多且意見分歧、公司財務狀況極度複雜等），法院可以依據債權人或清算人的聲請，或自己主動決定，命令開始「特別清算」程序。此外，如果清算人就任後發現公司的債務可能清償不了（資不抵債的風險），法院也可以命令開始特別清算。特別清算比普通清算有更嚴格的法院監督。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算中之股份有限公司 |
| 核心規定 | 清算執行有顯著困難或有不能清償債務之虞時，得開始特別清算 |
| 違反效果 | 不適用 |
| 注意事項 | 得由債權人、清算人聲請或法院依職權命令；特別清算受法院較嚴格監督 |

## 實務案例

**情境：** 甲公司進行普通清算中，發現公司帳面資產嚴重低估，實際負債遠超資產。清算人認為普通清算無法順利進行，向法院聲請特別清算。

**分析：** 依第335條第2項，清算人就任後有不能清償債務之虞時，法院得命令開始特別清算。甲公司資不抵債，符合特別清算之要件。

**結論：** 法院得依清算人之聲請命令開始特別清算，由法院監督清算程序之進行。
