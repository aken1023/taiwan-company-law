# 第 342 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算開始後，清算人得就公司財產之清償及其方法，擬定協定草案，提請債權人會議決議。

## 白話解說

進入特別清算後，清算人可以針對如何用公司財產清償債務（包括清償的比例和方式），擬定一份「協定草案」，然後提交給債權人會議來表決。這是特別清算中的「協定」制度，讓清算人可以和債權人協商出一個雙方都能接受的清償方案。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之清算人及債權人 |
| 核心規定 | 清算人得擬定協定草案提請債權人會議決議 |
| 違反效果 | 不適用 |
| 注意事項 | 協定草案內容為公司財產清償及方法 |

## 實務案例

**情境：** 甲公司特別清算中，公司資產僅能清償債務之60%。清算人擬定協定草案，建議以60%之比例清償所有債權人。

**分析：** 依第342條，清算人得擬定協定草案提請債權人會議決議。如債權人會議同意此60%清償方案，即可依協定進行清償。

**結論：** 清算人應召開債權人會議，說明公司財產狀況並提出60%清償之協定草案供表決。
