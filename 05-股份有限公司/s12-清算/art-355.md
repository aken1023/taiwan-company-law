# 第 355 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算，除本款另有規定外，準用普通清算之規定。

## 白話解說

特別清算除了第335條至第354條的專門規定以外，其他事項都準用普通清算（第322條至第334條）的規定。也就是說，特別清算的基本框架仍然以普通清算為基礎，只是在法院監督、債權人會議、協定制度等方面加上更嚴格的特別規定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之公司 |
| 核心規定 | 特別清算準用普通清算之規定 |
| 違反效果 | 依各被準用條文之效果處理 |
| 注意事項 | 特別清算之專門規定優先適用；未規定者準用普通清算 |

## 實務案例

**情境：** 甲公司進入特別清算後，清算人欲了解其職務範圍及權限。特別清算相關規定未另外規範清算人之職務。

**分析：** 依第355條，特別清算準用普通清算之規定。關於清算人之職務，依第324條（普通清算）辦理：了結現務、收取債權清償債務、分派盈餘或虧損、分派賸餘財產。

**結論：** 特別清算中清算人之職務範圍，準用第324條普通清算之規定。
