# 第 343 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 債權人會議由清算人召集，並以清算人為主席。
>
> 法院認為必要時，亦得召集債權人會議。

## 白話解說

特別清算中的債權人會議，由清算人負責召集，清算人同時擔任會議主席。另外，如果法院認為有必要，法院也可以主動召集債權人會議。這確保了債權人在特別清算程序中有參與討論和表決的機會。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之清算人及債權人 |
| 核心規定 | 債權人會議由清算人召集並擔任主席；法院亦得召集 |
| 違反效果 | 不適用 |
| 注意事項 | 法院亦有主動召集之權限 |

## 實務案例

**情境：** 甲公司特別清算中，清算人遲遲不召集債權人會議。主要債權人乙銀行向法院反映，要求召開債權人會議討論清償方案。

**分析：** 依第343條第2項，法院認為必要時亦得召集債權人會議。清算人怠於召集，法院得依職權或債權人之請求召集。

**結論：** 法院得依職權召集債權人會議，以推動清算程序之進行。
