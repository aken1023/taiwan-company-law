# 第 334 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 第八十三條第二項至第四項、第八十四條第二項、第八十五條至第八十九條及第九十一條之規定，於股份有限公司之清算準用之。

## 白話解說

股份有限公司的清算，準用無限公司清算的部分規定，包括：清算人變更時之聲報（第83條第2項至第4項）、清算人不得執行清算外業務（第84條第2項）、清算人就任後之催告及通知債權人（第85條）、債權人申報期限內不得清償（第86條）、清算後賸餘財產之分派（第87條）、聲報法院（第88條）、清算人之報酬（第89條）及清算時效（第91條）等。透過準用規定統一清算制度。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算中之股份有限公司 |
| 核心規定 | 準用無限公司清算之相關規定 |
| 違反效果 | 依各被準用條文之效果處理 |
| 注意事項 | 準用範圍涵蓋清算人變更聲報、職務限制、債權人催告、賸餘財產分派、法院聲報、報酬及時效等 |

## 實務案例

**情境：** 甲公司清算中，清算人以公司名義投資股票，遭監察人質疑。

**分析：** 依第334條準用第84條第2項，清算人不得執行清算目的範圍外之業務。投資股票非屬了結現務、收取債權清償債務等清算職務範圍，清算人之行為逾越權限。

**結論：** 清算人之投資行為違法，應立即停止並承擔因此造成之損害賠償責任。
