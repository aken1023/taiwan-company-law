# 第 331 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 清算完結，清算人應於十五日內，造具結算表冊，送經監察人審查，並提請股東會承認。
>
> 股東會對前項結算表冊為承認後，即為對清算人責任之解除。但清算人有不法行為者，不在此限。
>
> 第一項結算表冊之承認，股東會應於三十日內向法院聲報。
>
> 清算人應於清算完結，經股東會承認後十五日內，向主管機關申請解散登記。

## 白話解說

清算工作全部完成後，清算人要在15天內做好結算報告（結算表冊），交給監察人審查後，提交股東會承認。股東會承認結算表冊後，就等於解除了清算人的責任（除非清算人有違法行為）。承認後30天內要向法院報備，清算人也要在股東會承認後15天內向主管機關（經濟部或所在地直轄市/縣市政府）辦理解散登記。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算完結之公司及清算人 |
| 核心規定 | 清算完結後造具結算表冊，經監察人審查、股東會承認後向法院聲報及辦理解散登記 |
| 違反效果 | 未依限辦理者，依法處罰鍰 |
| 注意事項 | 股東會承認即解除清算人責任，但不法行為除外；承認後30日內向法院聲報、15日內申請解散登記 |

## 實務案例

**情境：** 甲公司清算完結後，清算人造具結算表冊經股東會承認。事後發現清算人在清算期間曾私自低價處分公司不動產圖利自己。

**分析：** 依第331條第2項，股東會承認結算表冊即解除清算人責任，但清算人有不法行為者不在此限。清算人低價處分公司財產圖利自己屬於不法行為，其責任不因股東會承認而解除。

**結論：** 股東得另行追究清算人之不法行為責任，不受股東會承認結算之影響。
