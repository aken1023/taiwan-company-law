# 第 328 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 在前條第一項所定期限內，清算人不得對債權人為清償。但對債權人有清償之必要者，清算人得聲請法院許可後清償之。

## 白話解說

在第327條所定的債權人申報債權的公告期間內（至少一個月），清算人原則上不能先對任何一個債權人進行清償。這是為了保障所有債權人公平受償的機會。但如果有特別緊急或必要的情形（例如員工薪資、避免資產損失的必要支出），清算人可以向法院聲請許可後進行清償。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算人 |
| 核心規定 | 催告期間內不得對債權人為清償，但經法院許可者除外 |
| 違反效果 | 違反禁止規定而清償者，清算人應負損害賠償責任 |
| 注意事項 | 必要之清償須經法院許可 |

## 實務案例

**情境：** 甲公司清算人在催告期間內，未經法院許可即優先清償對A供應商之貨款200萬元，導致其他債權人分配不足。

**分析：** 依第328條，催告期間內清算人不得對債權人清償，除非經法院許可。甲公司清算人未經許可即清償A供應商，違反本條規定。

**結論：** 其他債權人得向清算人主張損害賠償。清算人應先取得法院許可後方得於催告期間內為清償。
