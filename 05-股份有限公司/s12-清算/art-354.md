# 第 354 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算開始之原因消滅時，法院得因清算人之聲請，裁定撤銷特別清算。

## 白話解說

如果原本導致開始特別清算的原因已經不存在了（例如原本有清算執行顯著困難的情形，但困難已排除；或原本有資不抵債之虞，但後來發現資產足以清償），法院可以依清算人的聲請，裁定撤銷特別清算，改回普通清算程序繼續進行。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之公司 |
| 核心規定 | 特別清算原因消滅時，法院得因清算人聲請裁定撤銷特別清算 |
| 違反效果 | 不適用 |
| 注意事項 | 撤銷後回復普通清算；須由清算人聲請 |

## 實務案例

**情境：** 甲公司因清算有顯著困難而開始特別清算。後因主要債權人達成和解，清算困難已排除。清算人向法院聲請撤銷特別清算。

**分析：** 依第354條，特別清算開始之原因消滅時，法院得因清算人聲請裁定撤銷。甲公司之顯著困難已因和解而排除，原因已消滅。

**結論：** 法院得裁定撤銷特別清算，回復普通清算程序。
