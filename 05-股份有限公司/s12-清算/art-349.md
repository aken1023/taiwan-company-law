# 第 349 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 公司不履行協定時，法院因債權人之聲請，得命令開始破產程序。

## 白話解說

如果公司未能按照已獲法院認可的協定內容進行清償（也就是「不履行協定」），債權人可以向法院聲請，法院得命令開始破產程序。這是保障債權人的重要手段，防止公司在協定認可後仍不清償。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 不履行協定之公司 |
| 核心規定 | 公司不履行協定時，法院得因債權人聲請命令開始破產 |
| 違反效果 | 轉入破產程序 |
| 注意事項 | 須由債權人聲請 |

## 實務案例

**情境：** 甲公司之清償協定經法院認可後，約定於三個月內以60%比例清償所有債權人。但三個月期限屆至，甲公司僅清償約20%，其餘未履行。

**分析：** 依第349條，公司不履行協定時，法院因債權人聲請得命令開始破產程序。甲公司明顯未依協定履行。

**結論：** 債權人得向法院聲請，法院得命令甲公司開始破產程序。
