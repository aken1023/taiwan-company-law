# 第 348 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 協定經法院認可後，清算人應即執行之。

## 白話解說

協定一旦獲得法院認可，清算人就應該立即依照協定的內容執行清償。清算人沒有拖延的理由，必須按照協定約定的清償比例和方式，向各債權人進行清償。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 清算人 |
| 核心規定 | 協定經法院認可後，清算人應即執行 |
| 違反效果 | 怠於執行者，清算人應負法律責任 |
| 注意事項 | 「即」字表示不得無故拖延 |

## 實務案例

**情境：** 甲公司協定經法院認可後已三個月，清算人仍未開始依協定清償債權人，理由是公司資產尚待處分。

**分析：** 依第348條，協定經法院認可後清算人應即執行。三個月未開始執行，除非有正當理由（如資產確實需要時間處分），否則清算人有怠於執行之責任。

**結論：** 債權人得向法院反映，法院得命清算人儘速執行或依第341條將清算人解任。
