# 第 338 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 法院於特別清算開始後，得隨時就清算事務為必要之調查，並得選任監督人監督清算之進行。

## 白話解說

特別清算開始後，法院有權隨時對清算事務進行必要的調查，以確保清算程序正確進行。此外，法院也可以選任監督人來全程監督清算的進行，加強對清算人行為的監控。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之公司 |
| 核心規定 | 法院得隨時調查清算事務並得選任監督人 |
| 違反效果 | 不適用（法院權限） |
| 注意事項 | 監督人之選任為法院裁量權 |

## 實務案例

**情境：** 甲公司特別清算中，債權人反映清算人有隱匿財產之嫌疑。法院決定選任會計師為監督人，全程監督清算進行。

**分析：** 依第338條，法院得隨時就清算事務進行調查，並得選任監督人。法院選任會計師為監督人，屬於法定權限之行使。

**結論：** 法院選任之監督人有權監督清算人之清算行為，清算人應配合監督人之工作。
