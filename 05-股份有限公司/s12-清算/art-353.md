# 第 353 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算完結後，法院應為終結特別清算之裁定。

## 白話解說

當特別清算的所有事務都處理完畢後（例如協定的清償已全部履行），法院應該做出一個正式的裁定，終結特別清算程序。這是特別清算程序的最後一步，代表整個清算程序正式結束。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算已完結之公司 |
| 核心規定 | 特別清算完結後，法院應為終結特別清算之裁定 |
| 違反效果 | 不適用（法院義務） |
| 注意事項 | 此為特別清算之最終程序 |

## 實務案例

**情境：** 甲公司特別清算已依協定完成對所有債權人之清償，清算人向法院報告清算完結。

**分析：** 依第353條，特別清算完結後，法院應為終結特別清算之裁定。甲公司已完成清償，法院應裁定終結。

**結論：** 法院應做出終結特別清算之裁定，甲公司之特別清算程序正式結束。
