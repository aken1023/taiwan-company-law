# 第 346 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 協定之內容，對於債權人為不公平者，法院得不予認可。

## 白話解說

即使清償協定已經在債權人會議中通過表決，如果協定的內容對某些債權人明顯不公平（例如對某些債權人的清償比例遠低於其他債權人而沒有合理的理由），法院仍然可以不予認可。這是法院對協定內容實質審查的權限，確保協定的公平性。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 已經債權人會議決議之協定 |
| 核心規定 | 協定內容對債權人不公平者，法院得不予認可 |
| 違反效果 | 法院不予認可後，依第345條得宣告破產 |
| 注意事項 | 法院對協定內容有實質審查權 |

## 實務案例

**情境：** 甲公司特別清算之協定草案規定對A類債權人清償80%，但對B類債權人僅清償20%，兩類債權人之順位並無不同。

**分析：** 依第346條，協定內容對債權人不公平者，法院得不予認可。在沒有法定優先順位差異的情況下，對同順位債權人給予懸殊之清償比例，明顯不公平。

**結論：** 法院得以協定內容不公平為由，不予認可該協定。
