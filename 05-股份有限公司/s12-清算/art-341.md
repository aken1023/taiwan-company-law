# 第 341 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 特別清算開始後，法院認為必要時，得將清算人解任，另行選派清算人。

## 白話解說

在特別清算進行中，如果法院認為有必要（例如清算人不適任、有不正當行為、或怠於執行職務），法院可以主動將現任清算人解任，並另外選派適當的人擔任清算人。這是法院在特別清算中加強監督的重要權限之一。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 特別清算中之清算人 |
| 核心規定 | 法院得將清算人解任並另行選派 |
| 違反效果 | 不適用（法院權限） |
| 注意事項 | 法院有裁量權決定是否解任及另行選派 |

## 實務案例

**情境：** 甲公司特別清算中，清算人被發現涉嫌侵占公司資產。債權人向法院報告此情事。

**分析：** 依第341條，法院認為必要時得將清算人解任，另行選派清算人。清算人涉嫌侵占公司資產，已不適宜繼續擔任清算人。

**結論：** 法院得將該清算人解任，另行選派適當之人（如律師或會計師）為清算人。
