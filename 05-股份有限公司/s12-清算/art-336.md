# 第 336 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 法院為前條之裁定後，應即通知清算人，並以公告方式通知債權人及股東。

## 白話解說

法院裁定開始特別清算後，應立即將裁定結果通知清算人，並以公告方式通知所有債權人和股東。讓相關人士知道公司已經進入特別清算程序，以便行使相關權利。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 法院、清算人、債權人及股東 |
| 核心規定 | 法院裁定特別清算後應通知清算人並公告通知債權人及股東 |
| 違反效果 | 不適用（法院義務） |
| 注意事項 | 通知對象包括清算人、債權人及股東 |

## 實務案例

**情境：** 法院裁定甲公司開始特別清算，但僅通知清算人而未進行公告。部分債權人不知公司已進入特別清算程序。

**分析：** 依第336條，法院應以公告方式通知債權人及股東。未進行公告將導致債權人無法及時行使權利。

**結論：** 法院應依法進行公告，確保所有利害關係人知悉特別清算程序之開始。
