# 第 345 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十二節 清算

## 條文原文

> 協定不成立或不認可時，法院得因清算人、監察人或債權人之聲請，宣告公司破產。

## 白話解說

如果清算人所提出的協定草案在債權人會議中未能通過（協定不成立），或者法院不認可該協定，代表清償方案無法獲得同意。此時，法院可以依據清算人、監察人或債權人的聲請，宣告公司破產，轉入破產程序處理。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 協定未成立或未獲認可之公司 |
| 核心規定 | 協定不成立或不認可時，法院得因聲請宣告破產 |
| 違反效果 | 不適用 |
| 注意事項 | 聲請人包括清算人、監察人或債權人 |

## 實務案例

**情境：** 甲公司特別清算中，清算人提出之協定草案遭債權人會議否決。清算人認為公司已無其他清償方案，向法院聲請宣告破產。

**分析：** 依第345條，協定不成立時，法院得因清算人之聲請宣告公司破產。本案協定未通過且無替代方案，符合聲請破產之要件。

**結論：** 法院得受理清算人之聲請，宣告甲公司破產，轉入破產程序。
