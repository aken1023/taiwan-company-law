# 第 270 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第八節 發行新股

## 條文原文

> 公司有下列情形之一者，不得公開發行新股：
> 一、最近連續二年有虧損者。但依其事業性質，須有較長準備期間或具有健全之營業計畫，確能改善營利能力者，不在此限。
> 二、資產不足抵償債務者。

## 白話解說

本條規定了公司不得公開發行新股的兩種情形，目的在於保護社會投資大眾。

第一種情形：公司最近連續兩年有虧損。持續虧損的公司向大眾募集資金，投資人承受的風險過高。但如果公司因事業性質而需要較長的準備期間（例如生技公司的研發期較長），或者有健全的營業計畫確能改善獲利能力者，則可例外允許公開發行。

第二種情形：公司的資產不足以抵償債務（即資不抵債、淨資產為負值）。這種公司已處於技術性破產狀態，不應再向大眾募集資金。此款無例外規定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 擬公開發行新股之公司 |
| 核心規定 | 連續二年虧損者或資不抵債者不得公開發行 |
| 違反效果 | 違反者主管機關得撤銷核准 |
| 注意事項 | 連續二年虧損有但書例外；資不抵債無例外；僅限「公開」發行 |

## 實務案例

**情境：** 乙生技公司成立五年，前四年因研發投入而連續虧損，第五年開始有營收但仍未轉盈。公司擬公開發行新股籌集研發第三期臨床試驗資金。

**分析：** 乙公司最近連續二年有虧損，原則上不得公開發行新股。但依本條第 1 款但書，如事業性質需較長準備期間，且有健全營業計畫確能改善營利能力者，可以例外公開發行。生技公司研發期較長為業界常態，如能提出具體營業計畫，有機會獲准。

**結論：** 乙公司可向主管機關申請核准，提出完整的營業計畫及第三期臨床試驗之前景分析，說明其事業性質確需較長準備期間。由主管機關審酌是否核准。
