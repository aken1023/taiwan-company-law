# 第 274 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第八節 發行新股

## 條文原文

> 公司發行新股時，應就前條各款事項，公告並通知原有股東。

## 白話解說

本條規定公司在發行新股時，必須將第 273 條所列各款事項（包括章程事項、股份總數、公開發行申請事項、章程修正內容及股款繳納日期）公告，並通知原有股東。

「公告」是指向社會大眾公開揭示，通常刊登於新聞紙或公司網站；「通知」是指對每位原有股東個別發送通知。公告與通知的目的有二：一是讓原有股東知悉發行新股的資訊，以便行使第 267 條所定的優先認購權；二是讓社會大眾了解公司的增資狀況。

如未依法公告及通知，原有股東可能主張其認購權受到侵害。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 發行新股之股份有限公司 |
| 核心規定 | 須將第 273 條各款事項公告並通知原有股東 |
| 違反效果 | 未公告或通知，原股東得主張認購權受侵害 |
| 注意事項 | 公告與通知並行；須於合理期間前為之，使股東有足夠時間決定是否認購 |

## 實務案例

**情境：** 甲公司發行新股 300 萬股，僅在公司網站公告，但未個別通知原有股東。多位股東因此未能在期限內行使優先認購權。

**分析：** 依本條規定，公司發行新股時應「公告並通知」原有股東。甲公司僅公告未通知，程序有瑕疵。未獲通知的股東若因此未能行使認購權，得向公司主張權利。

**結論：** 甲公司應補行個別通知原有股東，並給予合理的認購期間。如已有股東因未獲通知而喪失認購權，可向公司請求損害賠償或主張該次發行程序有瑕疵。
