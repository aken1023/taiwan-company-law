# 第 272 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第八節 發行新股

## 條文原文

> 公司公開發行新股時，應以現金為股款。但由原有股東認購或由特定人協議認購，而不公開發行者，得以公司事業所需之財產為出資。

## 白話解說

本條規定了發行新股時股款繳納方式的區分。公司如果是公開發行新股（向社會大眾募集），股款必須以現金方式繳納，不得以實物出資。這是為了保護一般投資人，避免實物出資估價不實的風險。

但如果新股不是公開發行，而是由原有股東認購或由特定人協議認購，則可以用公司事業所需的財產（如不動產、設備、技術、專利等）來出資。這是因為在非公開發行的情形下，認購人與公司之間較為熟悉，且可透過協議方式議定實物出資的價值，風險相對較低。

實務上，非現金出資的估價仍須合理公允，否則可能構成虛增資本，損害其他股東及債權人的權益。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 發行新股之股份有限公司 |
| 核心規定 | 公開發行以現金為股款；非公開發行得以財產出資 |
| 違反效果 | 公開發行以非現金出資，程序違法 |
| 注意事項 | 財產出資限於「公司事業所需之財產」；估價須合理 |

## 實務案例

**情境：** 甲公司不公開發行新股 200 萬股，由特定人乙以一筆市值約 2,000 萬元的土地作為出資認購。

**分析：** 依本條但書，由特定人協議認購而不公開發行者，得以公司事業所需之財產為出資。乙以土地出資，如該土地為甲公司事業所需（如作為廠房用地），且經合理鑑價，則屬合法。

**結論：** 甲公司得接受乙以土地作為出資，但應確認：(1) 該土地為公司事業所需，(2) 土地估價合理公允，(3) 依法辦理產權移轉。
