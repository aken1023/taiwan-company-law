# 第 271 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第八節 發行新股

## 條文原文

> 證券管理機關於審核發行新股案件時，如發現有違反法令或不實之情事，得撤銷其核准。

## 白話解說

本條賦予證券管理機關（金管會）對發行新股案件的事後撤銷權。當主管機關在審核或事後查核過程中，發現公司發行新股有違反法令（如未保留員工份額、未通知原股東優先認購等）或申報資料有不實情事（如虛增營利、隱匿虧損等），可以撤銷原先的核准。

本條的規範對象為公開發行新股之案件（需經主管機關核准者），與第 251 條（公司債發行之撤銷核准）為平行規定。撤銷核准後，公司即不得繼續募集新股；已收到的股款應返還予認購人。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 已獲核准公開發行新股之公司 |
| 核心規定 | 主管機關發現違法或不實得撤銷核准 |
| 違反效果 | 核准被撤銷，不得繼續募集；已收股款應返還 |
| 注意事項 | 屬行政處分；公司負責人另可能負民刑事責任 |

## 實務案例

**情境：** 丙公司申請公開發行新股時，在財務報表中虛增營收 5 億元，使其看似未連續虧損而符合第 270 條要件。事後經主管機關查核發現。

**分析：** 丙公司申報資料不實（虛增營收），且因此規避了第 270 條之發行限制，屬違反法令。主管機關依本條得撤銷核准。

**結論：** 主管機關得撤銷丙公司之發行新股核准。丙公司負責人可能因財務報表不實而負刑事責任，投資人亦得請求損害賠償。
