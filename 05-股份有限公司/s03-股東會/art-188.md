# 第 188 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 第一百八十五條所列之行為，於決議後尚未實行者，經股東會以第一百八十五條第一項之決議撤銷前項行為之決議時，行使第一百八十六條之請求權者，失其效力。

## 白話解說

本條處理一種特殊情況：當公司股東會已經依第185條作成重大營業行為的決議，部分反對股東也已經依第186條提出股份收買請求，但後來公司又召開股東會以同樣的特別決議門檻撤銷了原來的重大行為決議，且該重大行為尚未實際執行。

在這種情況下，既然公司已撤銷原來的重大行為決議，反對股東當初行使股份收買請求權的前提（公司要進行重大行為）已經不存在了，因此其收買請求權失其效力。

此規定合理地讓公司在改變決策時，不必繼續負擔收買股份的義務。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 已行使§186收買請求權之反對股東 |
| 核心規定 | §185行為尚未實行而股東會撤銷原決議時，收買請求權失效 |
| 違反效果 | 收買請求失效，公司無須收買股份 |
| 注意事項 | 須以§185同等門檻之特別決議撤銷原決議；且原行為尚未實行 |

## 實務案例

**情境：** Z公司股東會決議讓與全部營業。反對股東依第186條請求公司收買其股份。嗣後Z公司因故決定不再進行營業讓與，再次召開股東會以特別決議撤銷原讓與決議。

**分析：** 依本條規定，Z公司之營業讓與尚未實行，且股東會已以§185第一項之特別決議門檻撤銷原決議。此時反對股東之股份收買請求權失其效力，Z公司無須收買該股東之股份。

**結論：** 公司撤銷重大行為之決議時，反對股東之收買請求權隨之失效，回復原狀。
