# 第 186 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 股東於股東會為前條決議前，已以書面通知公司反對該項行為之意思表示，並於股東會已為反對者，得請求公司以當時公平價格，收買其所有之股份。

## 白話解說

本條規定「異議股東之股份收買請求權」，是保護反對重大營業行為之少數股東的重要制度。

當公司依第185條為出租全部營業、讓與營業或財產、受讓他人全部營業等重大行為時，反對該行為的股東可以要求公司以「當時公平價格」買回其所持有的全部股份，使其得以退出公司。

但行使此權利有嚴格的程序要求：股東必須在股東會決議前，以書面通知公司反對該項行為的意思表示；並且在股東會上已為反對（投反對票）。兩個要件缺一不可。

所謂「當時公平價格」通常以股東會決議前之公平市場價格為基準，如有爭議可由法院裁定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 反對§185重大行為之股東 |
| 核心規定 | 反對股東得請求公司以當時公平價格收買其股份 |
| 違反效果 | 未依程序（書面通知+會議反對）表示反對者，不得行使收買請求權 |
| 注意事項 | 須事先以書面通知反對，且在股東會上投反對票 |

## 實務案例

**情境：** Y公司股東會決議讓與全部營業。股東陳某在會前未以書面通知反對，但在會議上投了反對票，隨後向公司請求以公平價格收買其股份。

**分析：** 依本條規定，行使股份收買請求權須同時具備兩個要件：一是在股東會決議前以書面通知公司反對之意思表示；二是在股東會上投反對票。陳某雖在會上投了反對票，但未於會前以書面通知反對，欠缺第一個要件，不得行使股份收買請求權。

**結論：** 反對股東應注意在股東會前以書面通知反對，否則將喪失股份收買請求權。

## 立法沿革

- 本條確立反對股東之股份收買請求權，保障少數股東退出公司之權利。
- 歷次修正維持基本架構。
