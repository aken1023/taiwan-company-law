# 第 177-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 公司召開股東會時，得採行以書面或電子方式行使其表決權；其以書面或電子方式行使表決權時，其行使方法應載明於股東會召集通知。但公開發行股票之公司，符合證券主管機關依公司規模、股東人數與結構及其他必要情況所定之條件者，應將電子方式列為表決權行使管道之一。

## 白話解說

本條規定股東得以書面或電子方式行使表決權，這是除了親自出席和委託代理之外的第三種表決權行使方式。公司如果決定採行此制度，必須在股東會召集通知中載明行使方法。

對於公開發行公司，如果符合證券主管機關所定之一定條件（如公司規模、股東人數等），則「必須」將電子投票列為表決權行使管道之一，這是強制性的規定。

此制度的目的是便利股東參與公司治理，特別是對於人數眾多、分散各地的公開發行公司股東而言，電子投票可大幅降低參與門檻，提高股東出席率。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司（公開發行公司有強制義務） |
| 核心規定 | 得採書面或電子方式行使表決權；公開發行公司符合一定條件者應提供電子投票 |
| 違反效果 | 未依規定提供電子投票者，可能構成召集程序違法 |
| 注意事項 | 行使方法應載明於召集通知；以書面或電子方式行使者視為親自出席 |

## 實務案例

**情境：** K公司為公開發行公司，股東人數達2萬人，符合證券主管機關規定之條件。K公司召開股東常會時僅提供紙本委託書之方式，未提供電子投票。

**分析：** 依本條但書規定，K公司符合主管機關所定條件，應將電子方式列為表決權行使管道之一。未提供電子投票違反本條規定，可能構成召集程序之瑕疵。

**結論：** 符合條件之公開發行公司應主動提供電子投票管道，以符合法令要求並保障股東參與權。

- 民國94年增訂本條，建立書面及電子投票制度。
