# 第 189-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 法院對於前條撤銷決議之訴，認為其違反之事實非屬重大且於決議無影響者，得駁回其請求。

## 白話解說

本條是第189條的補充規定，賦予法院「裁量駁回權」。即使股東會的召集程序或決議方法確實存在違法瑕疵，但如果法院認為該違法的情形不嚴重（非屬重大），而且對決議結果沒有影響，法院可以駁回撤銷之訴。

本條的立法目的在於維護法律安定性及交易安全。如果每個輕微的程序瑕疵都導致決議被撤銷，將嚴重影響公司經營的穩定性。因此，法院在審理時應衡量瑕疵的嚴重程度及其對決議結果的實際影響。

例如：股東會通知晚了一天寄發，但出席率仍達90%，且決議以壓倒性多數通過，此種情形下通知晚一天的瑕疵「非屬重大且於決議無影響」，法院得駁回撤銷之請求。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 法院審理§189撤銷決議之訴時 |
| 核心規定 | 違反之事實非屬重大且於決議無影響者，法院得駁回 |
| 違反效果 | 不適用（本條賦予法院裁量權） |
| 注意事項 | 「非屬重大」與「於決議無影響」二要件須同時具備；法院為「得」駁回，非「應」駁回 |

## 實務案例

**情境：** B公司非公開發行公司，股東常會通知於開會18日前寄發（應於20日前），出席率達85%，某議案以80%出席表決權同意通過。少數股東主張召集程序違法，提起撤銷之訴。

**分析：** 通知期限僅短缺2日，屬輕微瑕疵。出席率高達85%，遠超過半數門檻，表示通知短缺2日並未影響股東出席。且議案以80%之高比例通過，即使少數未受通知之股東全數出席反對，亦不影響結果。法院得依本條認定瑕疵非屬重大且於決議無影響，駁回撤銷之請求。

**結論：** 法院對輕微程序瑕疵有裁量駁回之空間，但重大瑕疵仍應撤銷決議。
