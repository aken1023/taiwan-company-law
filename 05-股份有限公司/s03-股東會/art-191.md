# 第 191 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 股東會決議之內容，違反法令或章程者無效。

## 白話解說

本條規定股東會決議之「內容無效」。與第189條處理「程序瑕疵」（召集程序或決議方法違法）不同，本條處理的是「內容瑕疵」，即決議的實質內容違反法令或章程。

例如：股東會決議以低於法定最低薪資的標準支付員工薪水（違反勞動法令）、決議將公司財產無償贈與董事個人（違反公司法相關規定）等。

本條規定此類決議「無效」，與第189條的「得撤銷」有重大區別：

一、無效是自始、當然、確定無效，不待法院判決即為無效。
二、無須在30日內提起訴訟，任何人、任何時候均得主張無效。
三、法院為確認判決，確認決議自始無效，而非形成判決。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 決議內容違反法令或章程之股東會決議 |
| 核心規定 | 決議內容違反法令或章程者，自始無效 |
| 違反效果 | 決議自始、當然、確定無效 |
| 注意事項 | 與§189程序瑕疵不同；無30日除斥期間限制；任何人均得主張 |

## 實務案例

**情境：** D公司股東會決議，將公司盈餘全數分配給董事長個人作為特別獎金，不分配予股東。

**分析：** 公司盈餘之分配應依法令及章程規定，由全體股東依持股比例分配。股東會決議將盈餘全數分配給董事長一人，違反公司法關於盈餘分配之規定，屬決議內容違反法令，依本條規定該決議自始無效。任何股東、債權人或利害關係人均得隨時主張該決議無效，不受30日期間之限制。

**結論：** 內容違法之決議自始無效，公司及相關人員均不得據以執行。

## 立法沿革

- 民國18年公司法制定時即有決議內容違法無效之規定。
- 本條為公司法最基本的決議效力規範之一，歷次修正均維持不變。
