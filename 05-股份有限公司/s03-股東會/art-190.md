# 第 190 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 股東會決議經依前條撤銷確定後，如該決議已為登記者，主管機關應依職權或依利害關係人之聲請撤銷其登記。

## 白話解說

本條規定決議撤銷確定後的行政配套措施。當法院依第189條判決撤銷股東會決議確定後，如果該決議事項（例如變更章程、選任董事等）已經向主管機關辦理登記，主管機關應撤銷該登記。

撤銷登記可以由主管機關依職權主動為之，也可以由利害關係人（如原告股東）向主管機關聲請。

這是為了確保法院判決的效力能夠落實在行政登記層面，使第三人能夠從公司登記中獲得正確的資訊。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 經法院撤銷確定之股東會決議所涉之登記事項 |
| 核心規定 | 主管機關應依職權或依聲請撤銷已為之登記 |
| 違反效果 | 主管機關未撤銷登記者，利害關係人得聲請之 |
| 注意事項 | 須法院撤銷判決確定後始得撤銷登記 |

## 實務案例

**情境：** C公司股東會決議變更章程並選任新董事，已向主管機關完成登記。嗣後該決議經法院撤銷確定。

**分析：** 依本條規定，主管機關應撤銷該變更章程及新董事選任之登記，回復至原來的登記狀態。利害關係人（如原告股東或原任董事）得持法院確定判決向主管機關聲請撤銷登記。

**結論：** 決議撤銷確定後，相關登記應一併撤銷，以維護登記之正確性。
