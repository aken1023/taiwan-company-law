# 第 173-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 繼續三個月以上持有已發行股份總數過半數股份之股東，得自行召集股東臨時會。

## 白話解說

本條是民國107年公司法修正時新增的重要規定。相較於第173條要求少數股東先請求董事會召集、董事會不為召集時再報經主管機關許可的繁複程序，本條提供了一個更直接的途徑：只要繼續三個月以上持有已發行股份總數過半數的股東，就可以直接自行召集股東臨時會，不需要先請求董事會召集，也不需要經主管機關許可。

本條的立法背景在於實務上曾發生經營權爭奪時，董事會拒絕召集股東會，而主管機關許可程序又曠日廢時，導致持有過半數股份之股東反而無法有效行使其權利的不合理情況。因此增訂本條，讓持有過半數股份的多數股東有自行召集的權限。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 繼續3個月以上持有已發行股份總數過半數之股東 |
| 核心規定 | 過半數股東得直接自行召集股東臨時會，無須經董事會或主管機關 |
| 違反效果 | 未符合持股條件而自行召集者，決議效力有疑義 |
| 注意事項 | 持股須繼續3個月以上；可為單一股東或數位股東合計 |

## 實務案例

**情境：** 甲公司之大股東A持有55%股份已逾半年。甲公司董事會長期未召開股東會，且拒絕少數股東依第173條之請求。A遂決定依本條自行召集臨時會。

**分析：** A繼續三個月以上持有已發行股份總數過半數（55%），符合本條規定之要件。A得直接自行召集股東臨時會，無須先請求董事會召集，亦無須報經主管機關許可。A應依第172條規定之通知期限及方式發出召集通知，所召集之股東臨時會具有合法效力。

**結論：** 本條為過半數股東提供了快速有效的召集途徑，在公司經營權爭奪或董事會失能時特別重要。
