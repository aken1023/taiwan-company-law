# 第 185-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 公司與他公司依前條為營業或財產之讓與或受讓時，其債權讓與之通知，得以公告方式代之。

## 白話解說

本條為配合第185條重大營業行為所設之便利規定。當公司依第185條進行營業或財產之讓與或受讓時，通常會涉及大量債權的移轉。依民法規定，債權讓與須逐一通知債務人。但在大規模營業讓與的情況下，債權數量可能非常龐大，逐一通知在實務上有相當的困難。

因此，本條規定公司可以用「公告」的方式代替個別通知。這大幅簡化了大規模營業讓與時的債權移轉程序，使交易更為便利可行。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 依§185進行營業或財產讓與/受讓之公司 |
| 核心規定 | 債權讓與之通知得以公告方式代替個別通知 |
| 違反效果 | 不適用本條而未為通知者，債權讓與對債務人不生效力 |
| 注意事項 | 僅適用於§185之營業或財產讓與/受讓情形 |

## 實務案例

**情境：** W公司將全部營業讓與X公司，涉及3,000筆應收帳款之債權移轉。W公司以報紙公告方式通知全部債務人。

**分析：** 依本條規定，W公司依第185條為營業讓與時，其債權讓與之通知得以公告方式代之。以報紙公告方式通知符合本條規定，不須逐一對3,000位債務人個別通知。

**結論：** 本條大幅簡化營業讓與時之債權移轉程序，符合商業效率之需求。
