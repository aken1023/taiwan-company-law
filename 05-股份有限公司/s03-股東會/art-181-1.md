# 第 181-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 公司得依章程規定，對持有一定數量股份以上之股東，就其超過部分之股份，限制其表決權。

## 白話解說

本條允許公司在章程中設定表決權之上限限制，針對持有超過一定數量股份的大股東，就其超過部分的股份限制表決權。此即「表決權上限制度」。

例如，章程可訂明「股東持有股份超過已發行股份總數10%者，其超過10%部分之表決權減半計算」。此制度的目的在於防止單一大股東過度集中控制權，保護中小股東的權益。

但此規定僅為授權規定，公司得自行決定是否採用，以及如何設定上限門檻。如章程未訂明，則仍依第179條每股一表決權之原則行使。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 持有一定數量股份以上之大股東 |
| 核心規定 | 得依章程限制超過一定持股數量之表決權 |
| 違反效果 | 章程未訂明者不適用本條限制 |
| 注意事項 | 為授權規定，非強制規定；須在章程中明確訂明 |

## 實務案例

**情境：** O公司章程規定，股東持有股份超過已發行股份總數5%者，超過5%之部分每股僅享有半數表決權。大股東王某持有O公司8%股份。

**分析：** 依章程規定，王某持股中5%部分享有完整表決權，超過5%之3%部分每股僅享有0.5個表決權。假設已發行1,000萬股，王某持有80萬股，其中50萬股享有完整表決權（50萬個），另30萬股享有半數表決權（15萬個），合計王某之表決權為65萬個（而非80萬個）。

**結論：** 表決權上限限制制度有助於平衡大小股東之權益，但須在章程中明確規定。
