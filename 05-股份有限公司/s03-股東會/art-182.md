# 第 182 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 股東會得決議延期或續行集會，其延期或續行集會，不適用第一百七十二條之通知規定。

## 白話解說

本條規定股東會的「延期」或「續行集會」制度。如果股東會當天因故無法完成所有議程（例如議案太多、討論時間不足、出席人數不足等），可以決議延期到另一天繼續開會，或者休會後在之後的某一天續行集會。

延期或續行集會是原來股東會的延續，不是一個全新的會議，因此不需要重新依照第172條的通知期限來通知股東。這樣的規定是為了簡化程序，避免因重新通知而浪費時間。

但仍應通知股東延期或續行集會的日期及地點，以使股東知悉。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 無法於當日完成議程之股東會 |
| 核心規定 | 股東會得決議延期或續行集會，不適用§172之通知規定 |
| 違反效果 | 未經股東會決議而逕行延期者，可能構成程序瑕疵 |
| 注意事項 | 延期或續行集會為原會議之延續，非新召集之會議；仍應通知股東時間地點 |

## 實務案例

**情境：** Q公司股東常會議程包含10項議案，會議進行至晚間9時仍有5項議案未完成討論。主席宣布延期至隔週六同一地點續行集會。

**分析：** 依本條規定，股東會得決議延期或續行集會。Q公司之續行集會係原會議之延續，不需重新發出20日前之通知。但主席之宣布應經出席股東同意或依議事規則處理，且應通知全體股東續行集會之時間地點。

**結論：** 延期或續行集會為原會議之延續，程序上較為簡便，但仍應適當通知股東。
