# 第 176-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 股東對於會議之事項，有自身利害關係致有害於公司利益之虞時，該股東不得加入表決，並不得代理他股東行使其表決權。但關於推選該次會議之主席不在此限。

## 白話解說

本條規定股東表決權之利害關係迴避制度。當股東會討論之議案與某位股東個人有直接的利害關係，且該利害關係可能損害公司利益時，該股東必須迴避，不得參與該議案之表決。不僅自己不能投票，也不得代理其他股東行使表決權。

但如果是推選當次會議主席的程序，則不受迴避限制，因為主席之選定屬於會議程序事項，與利害關係迴避之目的無直接關聯。

此迴避制度的目的是防止有利害關係的股東利用其表決權影響決議結果，損害公司及其他股東之權益。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 對會議事項有自身利害關係之股東 |
| 核心規定 | 有利害關係且有害公司利益之虞時，不得加入表決，亦不得代理他股東行使表決權 |
| 違反效果 | 未迴避而參與表決之決議，可能構成決議方法違法，得依§189撤銷 |
| 注意事項 | 推選會議主席不在迴避之列 |

## 實務案例

**情境：** I公司股東會討論是否以高價向股東陳某購買其所有之不動產。陳某持有I公司10%股份，在該議案表決時投下贊成票。

**分析：** 陳某對於該議案有直接的自身利害關係（他是不動產出售人），且以高價購買可能損害公司利益。依本條規定，陳某不得加入該議案之表決。若陳某仍參與表決，該決議之決議方法違法，其他股東得依第189條訴請法院撤銷。

**結論：** 有利害關係之股東應自行迴避，否則決議效力可能受影響。
