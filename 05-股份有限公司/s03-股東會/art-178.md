# 第 178 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> 股東對於會議之事項，有自身利害關係致有害於公司利益之虞時，不得加入表決，並不得代理他股東行使其表決權。

## 白話解說

本條規定股東表決權之利害關係迴避原則。當股東會討論的議案涉及某位股東自身的利益，且這種利害關係可能有損於公司利益時，該股東不僅不能就該議案投票表決，也不能代替其他股東行使表決權。

要適用本條的迴避規定，需同時具備兩個要件：第一，股東對議案有「自身利害關係」；第二，該利害關係「致有害於公司利益之虞」。兩者缺一不可。如果股東的利益與公司利益方向一致，並無害於公司利益之虞者，則不必迴避。

迴避的股東其所持有的股份在該議案表決時不計入出席股東之表決權數，但仍計入出席之股份總數。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 對議案有自身利害關係且有害公司利益之虞之股東 |
| 核心規定 | 不得加入表決，亦不得代理他股東行使表決權 |
| 違反效果 | 未迴避之決議構成決議方法違法，得依§189訴請撤銷 |
| 注意事項 | 須同時具備「自身利害關係」及「有害公司利益之虞」兩要件 |

## 實務案例

**情境：** L公司股東會討論是否以市價二倍之價格向大股東趙某承租辦公室。趙某持有L公司30%股份，於該議案表決時投下贊成票。

**分析：** 趙某身為出租人，對於承租議案有直接之自身利害關係，且以市價二倍之高價承租，顯有害於公司利益之虞。依本條規定，趙某不得加入該議案之表決。若趙某仍加入表決，其表決權不應計入，且該決議之決議方法違法，其他股東得依第189條於30日內訴請法院撤銷。

**結論：** 利害關係迴避制度旨在防止股東以表決權圖謀私利，股東應主動迴避或由會議主席要求迴避。

## 立法沿革

- 民國18年公司法制定時即有利害關係迴避之規定。
- 歷次修正維持本條基本架構。
