# 第 177 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> I 股東得於每次股東會，出具公司印發之委託書，載明授權範圍，委託代理人，出席股東會。
>
> II 除信託事業或經證券主管機關核准之股務代理機構外，一人同時受二人以上股東委託時，其代理之表決權不得超過已發行股份總數表決權之百分之三，超過時其超過之表決權，不予計算。
>
> III 一股東以出具一委託書，並以委託一人為限，應於股東會開會五日前送達公司，委託書有重複時，以最先送達者為準。但聲明撤銷前委託者，不在此限。

## 白話解說

本條規定股東出席股東會的委託代理制度。股東如果無法親自出席股東會，可以使用公司印發的委託書，載明授權範圍，委託代理人代為出席。

為防止少數人蒐集大量委託書以控制股東會，本條設有限制：除了信託事業或經核准之股務代理機構外，任何一個人如果同時受到兩位以上股東的委託，其代理行使的表決權最多不得超過公司已發行股份總數表決權的百分之三。超過的部分不予計算。

此外，每位股東只能出具一張委託書、委託一位代理人，且須在開會五日前送達公司。如有重複委託（同一股東出具多張委託書），以最先送達的為準，但股東聲明撤銷前一份委託的除外。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 無法親自出席股東會之股東 |
| 核心規定 | 得以委託書委託代理人出席；一人受二人以上委託時代理表決權不超過3% |
| 違反效果 | 超過3%部分不予計算；重複委託以最先送達者為準 |
| 注意事項 | 須使用公司印發之委託書；開會5日前送達；一人一委託書 |

## 實務案例

**情境：** 甲持有J公司1%股份，乙持有2%，丙持有3%。三人均委託張三代理出席股東會。J公司已發行股份總數為1,000萬股。

**分析：** 張三同時受三人委託，合計代理6%表決權。依本條第二項規定，張三非信託事業亦非股務代理機構，其代理之表決權不得超過已發行股份總數之3%（30萬股）。超過部分不予計算，故張三最多僅能代理30萬股之表決權，其餘30萬股之表決權不予計算。

**結論：** 委託書制度在保障股東參與權的同時，也防止委託書的過度集中。代理人應注意3%之上限限制。

- 民國18年公司法制定時即有委託書制度之規定。
- 民國90年修正，增訂信託事業及股務代理機構之例外。
- 民國107年修正，配合實務需要調整文字。
