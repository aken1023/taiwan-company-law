# 第 181 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> I 政府或法人為股東時，其代表人不限於一人。但其表決權之行使，仍以其所持有之股份綜合計算。
>
> II 前項之代表人有二人以上時，其代表人行使表決權應一致。

## 白話解說

本條規定政府機關或法人擔任公司股東時，其出席股東會之代表人可以不限於一人。例如政府持有某公司20%股份，可以指派多位代表人出席股東會。但這些代表人的表決權仍以該政府或法人所持有的股份數綜合計算，不因代表人人數增加而增加表決權。

如果有兩位以上代表人出席，他們行使表決權時必須一致，不能對同一議案有人投贊成有人投反對。這是因為他們代表的是同一個股東（政府或法人），該股東對同一議案應該只有一個意思表示。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 以政府或法人身分為股份有限公司之股東 |
| 核心規定 | 代表人不限一人，但表決權綜合計算；多位代表人行使表決權應一致 |
| 違反效果 | 代表人意見不一致時，該表決權之行使有效性存疑 |
| 注意事項 | 代表人人數不影響表決權數；各代表人須協調一致立場 |

## 實務案例

**情境：** 某縣政府持有N公司15%股份，指派甲、乙兩位代表人出席股東會。就「是否增資」議案，甲投贊成票、乙投反對票。

**分析：** 依本條第二項規定，代表人有二人以上時行使表決權應一致。甲、乙意見相左，無法形成一致之意思表示，該15%股份之表決權行使將產生疑義。實務上通常以無法有效行使表決權論，不予計入贊成或反對票數。

**結論：** 政府或法人指派多位代表人出席時，應事先協調立場以確保表決權之有效行使。
