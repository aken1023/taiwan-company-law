# 第 184 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第三節 股東會

## 條文原文

> I 股東會得查核董事會造具之表冊，並得選任檢查人，檢查之。
>
> II 對前項之檢查，有妨礙、拒絕或規避之行為者，各處新臺幣二萬元以上十萬元以下罰鍰。

## 白話解說

本條賦予股東會兩項監督權利：

第一，股東會可以查核董事會所編造的各種表冊（如營業報告書、財務報表、盈餘分配表等），確認其內容是否正確。

第二，股東會可以選任「檢查人」來進行更深入的檢查。檢查人由股東會選任，受股東會之委託，對公司的帳簿、文件進行專業查核。

如果有人對檢查人的檢查工作有妨礙、拒絕或規避的行為（例如拒絕提供帳簿資料、隱匿文件等），每次將被處以新臺幣二萬元以上十萬元以下的罰鍰。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之股東會 |
| 核心規定 | 股東會得查核董事會表冊並選任檢查人檢查 |
| 違反效果 | 妨礙、拒絕或規避檢查者，處新臺幣2萬至10萬元罰鍰 |
| 注意事項 | 檢查人由股東會選任；罰鍰對象為妨礙檢查之人 |

## 實務案例

**情境：** T公司股東會選任會計師丁為檢查人，檢查公司近三年之帳務。丁要求公司提供相關帳簿及傳票，但公司財務長以「涉及營業秘密」為由拒絕提供。

**分析：** 依本條規定，檢查人受股東會委託進行檢查，公司負有配合之義務。財務長拒絕提供帳簿係「拒絕」檢查之行為，依第二項規定應處新臺幣二萬元以上十萬元以下罰鍰。營業秘密不構成拒絕配合股東會選任之檢查人的正當理由。

**結論：** 公司內部人員不得以任何理由妨礙或拒絕股東會選任之檢查人進行檢查。

- 民國18年公司法制定時即有股東會查核表冊及選任檢查人之規定。
- 歷次修正配合罰鍰金額之調整。
