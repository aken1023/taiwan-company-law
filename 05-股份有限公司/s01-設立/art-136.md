# 第 136 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第一節 設立

## 條文原文

> 前條撤銷核准，未招募者，停止招募；已招募者，應募人得依股份原發行金額，加算法定利息，請求返還。

## 白話解說

本條規定了第 135 條撤銷核准後的法律效果，分成兩種情形處理。如果核准被撤銷時尚未開始招募，那就直接停止不再招募。如果已經開始招募且有投資人認股繳款了，則必須保護這些已繳款的認股人。

已繳款的認股人（應募人）有權要求返還其所繳的股款，而且不只是返還原繳金額，還要加上法定利息。法定利息依民法規定，目前為年利率 5%。這樣的設計是為了彌補認股人資金被占用期間的時間價值損失。

本條體現了對投資大眾的保護精神。當公開招募的核准因違法或虛偽而被撤銷時，認股人是無辜的受害者，法律賦予他們取回投資本金加利息的權利。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 因撤銷核准而受影響之應募人（認股人） |
| 核心規定 | 未招募者停止招募；已招募者，應募人得請求返還股款加法定利息 |
| 違反效果 | 發起人若不返還，應募人得依法請求，發起人負連帶責任（第 150 條） |
| 注意事項 | 法定利息依民法第 203 條為年利率 5% |

## 實務案例

**情境：** 發起人甲等公開招募股份經金管會核准，已有 200 位認股人合計繳入 5,000 萬元。三個月後金管會發現營業計畫書有虛偽，撤銷核准。

**分析：** 依第 136 條，核准被撤銷後，已繳款的 200 位認股人得請求返還其繳納的股款，並加算法定利息（年利率 5%）。假設某認股人三個月前繳入 50 萬元，可請求返還 50 萬元加上三個月的法定利息約 6,250 元。

**結論：** 發起人應儘速將代收股款專戶中的資金連同利息返還全體認股人。若發起人無力返還，依第 150 條應負連帶責任。
