# 第 142 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第一節 設立

## 條文原文

> 認股人延欠前條應繳之股款時，發起人應定一個月以上之期限催告該認股人照繳，並聲明逾期不繳失其權利。
>
> 發起人已為前項之催告，認股人不照繳者，即失其權利，所認股份另行募集。
>
> 前項情形，如有損害，仍得向認股人請求賠償。

## 白話解說

本條建立了認股人延欠股款時的處理程序，也就是「催告—失權—求償」三步驟。

第一步是催告：當認股人收到催繳通知後卻不繳款時，發起人必須給予認股人至少一個月的寬限期，正式催告其繳款，並且在催告中聲明「逾期不繳將喪失認股權利」。這個程序保障認股人有充分的時間籌措資金。

第二步是失權：如果認股人在催告期限屆滿後仍然不繳款，就自動喪失認股權利。喪失的股份由發起人另行向其他人募集。

第三步是求償：即使認股人喪失了權利，如果因為他的延欠造成公司損害（例如延遲設立造成的損失、重新募集的費用等），發起人仍然可以向該認股人請求損害賠償。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 延欠股款之認股人 |
| 核心規定 | 催告期至少一個月；逾期不繳喪失權利；所認股份另行募集 |
| 違反效果 | 認股人喪失認股權利且可能須賠償損害 |
| 注意事項 | 催告必須明確聲明失權效果；未聲明者催告可能無效 |

## 實務案例

**情境：** 陳先生認購某新設公司 50 萬股，但收到催繳通知後以資金周轉困難為由遲遲未繳。發起人隨即口頭告知陳先生失去認股權利，並將這 50 萬股轉給其他投資人。

**分析：** 依第 142 條第一項，發起人必須「定一個月以上之期限催告」且「聲明逾期不繳失其權利」。口頭告知不符合催告的法定形式，且未給予至少一個月的寬限期。發起人的處理不合法，陳先生可能主張其認股權利仍然存在。

**結論：** 發起人應以書面方式正式催告，給予至少一個月期限，並載明失權聲明。程序合法後，認股人不繳始喪失權利。
