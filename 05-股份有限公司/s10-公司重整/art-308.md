# 第 308 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 重整之終止或完成，法院應為裁定及公告，並通知已知之利害關係人。
>
> 重整之終止或完成，應向主管機關為變更之登記。

## 白話解說

本條規定重整終止或完成時的程序要求。

不論重整是因失敗而終止（如依第 307 條各款事由），還是因成功而完成（如依第 310 條重整計畫執行完畢），法院都必須做出裁定並進行公告，同時通知所有已知的利害關係人。

此外，公司應向主管機關辦理變更登記，使公司的登記狀態反映重整已終止或完成的事實。這是基於登記公示原則，讓第三人透過查閱登記資料得知公司不再處於重整狀態。

這些程序要求確保了重整狀態的終結有明確的時間點，各方權利義務關係得以明確界定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 重整終止或完成時 |
| 核心規定 | 法院應為裁定及公告，通知利害關係人；應辦理變更登記 |
| 違反效果 | 未辦變更登記不影響裁定效力，但不得對抗善意第三人 |
| 注意事項 | 不論終止或完成均適用相同程序 |

## 實務案例

**情境：** 某公司重整計畫執行完畢，法院裁定重整完成並公告。但公司遲遲未向主管機關辦理變更登記，導致登記資料仍顯示「重整中」。

**分析：** 依第 308 條第 2 項，重整完成應向主管機關為變更登記。公司未辦理登記，使第三人查閱登記時仍認為公司在重整中，可能影響公司的交易信用。

**結論：** 公司應儘速辦理變更登記。主管機關亦得依法通知公司限期辦理。
