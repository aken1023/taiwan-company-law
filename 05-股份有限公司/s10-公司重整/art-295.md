# 第 295 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 重整裁定前所為之保全處分，於重整裁定後仍有效力。其因保全處分所生之程序，於重整裁定後已無必要者，重整人得聲請法院撤銷之。

## 白話解說

本條處理重整裁定前（依第 286 條）所做的保全處分，在重整裁定正式生效後的效力問題。

原則上，裁定前的保全處分在裁定後繼續有效。這是合理的，因為重整裁定後公司仍需要保護措施，先前的保全處分（如凍結公司財產、停止強制執行等）在重整期間仍有維持的必要。

但如果某些保全處分在重整裁定後已無必要（例如，第 294 條已規定強制執行當然停止，則先前針對特定強制執行案件的保全處分即屬多餘），重整人可以聲請法院撤銷這些不再需要的處分，以減少不必要的法律限制。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 重整裁定前已做出保全處分之情形 |
| 核心規定 | 裁定前之保全處分於裁定後仍有效力；已無必要者得聲請撤銷 |
| 違反效果 | 不適用 |
| 注意事項 | 重整人得主動聲請撤銷已無必要之保全處分 |

## 實務案例

**情境：** 法院在重整裁定前，依第 286 條裁定禁止公司記名股票轉讓及停止對公司之強制執行。重整裁定生效後，重整人認為禁止股票轉讓之處分已無必要。

**分析：** 依第 295 條，兩項保全處分原則上仍有效。但重整人認為禁止股票轉讓已無必要時，得聲請法院撤銷該項處分。至於停止強制執行之處分，因第 294 條已有當然停止之規定，該保全處分雖仍有效但實質上已與法律效果重疊。

**結論：** 重整人得聲請法院撤銷禁止股票轉讓之保全處分。法院應審酌撤銷是否影響重整程序之進行後為裁定。
