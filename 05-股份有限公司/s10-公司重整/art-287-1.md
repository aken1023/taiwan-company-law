# 第 287-1 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 對於第二百八十六條之裁定，得為抗告。但抗告中不停止執行。

## 白話解說

本條規定對第 286 條保全處分裁定的救濟途徑。

當法院依第 286 條對公司做出保全處分裁定時（如凍結財產、限制業務、停止強制執行等），受影響的利害關係人如果認為該裁定不當，有權向上級法院提起抗告（即對裁定不服的救濟途徑）。

但本條特別規定「抗告中不停止執行」，也就是說，即使當事人提起抗告，原保全處分裁定的效力仍然繼續維持，不會因為抗告而暫時停止。這是因為保全處分的目的在於緊急保護公司資產及利害關係人的權益，如果允許抗告停止執行，可能導致公司資產在抗告期間遭到轉移或流失，喪失保全的意義。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 受第 286 條保全處分裁定影響之利害關係人 |
| 核心規定 | 對保全處分裁定得為抗告，但抗告不停止執行 |
| 違反效果 | 不適用 |
| 注意事項 | 保全處分於抗告期間仍繼續有效 |

## 實務案例

**情境：** 法院依第 286 條裁定凍結公司全部銀行帳戶。公司認為凍結範圍過廣影響日常營運，提起抗告。

**分析：** 依第 287-1 條，公司得對保全處分裁定提起抗告，但抗告不停止執行。因此，在抗告法院做出裁定前，公司銀行帳戶仍維持凍結狀態。

**結論：** 公司於抗告期間，帳戶仍被凍結。如確有急迫需要，公司得依第 287 條聲請法院變更保全處分（例如解凍部分帳戶供日常營運使用），或請求抗告法院加速審理。
