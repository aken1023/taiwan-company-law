# 第 291 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 裁定重整後，法院應就第二百八十九條所列事項，為公告，並通知公司、已知之公司債權人及股東。
>
> 前項公告應揭示於法院牌示處。

## 白話解說

本條規定重整裁定後的公告及通知程序，確保所有利害關係人都能及時知悉重整的相關資訊。

法院裁定准許重整後，必須將第 289 條所列的重要事項（包括重整監督人的選任、債權及股東權的申報期日與場所、第一次關係人會議的期日與場所等）進行公告，同時通知公司本身、法院已知的公司債權人及股東。

公告應揭示於法院的牌示處（布告欄），使一般公眾得以知悉。通知則以個別送達方式為之，確保已知的利害關係人確實收到資訊。

此雙軌制的通知方式（公告加個別通知），是為了在保障已知利害關係人權益的同時，也讓可能被遺漏的債權人或股東有機會得知重整資訊。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 裁定重整後之法院 |
| 核心規定 | 法院應公告第 289 條事項，並通知公司、已知債權人及股東 |
| 公告方式 | 揭示於法院牌示處 |
| 注意事項 | 採公告及個別通知雙軌制，確保利害關係人知悉 |

## 實務案例

**情境：** 法院裁定某公司重整後進行公告及通知。但某小額債權人丙未收到個別通知，因不知重整裁定之事，未於期限內申報債權。

**分析：** 依第 291 條，法院應通知「已知之」公司債權人。如丙為已知之債權人但法院漏未通知，可能影響其權利行使。如丙為法院不知之債權人，則法院以公告方式告知已符合法律要求。

**結論：** 債權人應主動關注公司動態。如因法院漏未通知致權利受損，丙得依法主張救濟。法院應盡力查明所有已知債權人並逐一通知。
