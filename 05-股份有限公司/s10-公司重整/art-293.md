# 第 293 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 裁定重整後，公司之業務經營權及財產之管理處分權移屬於重整人。
>
> 公司之董事、監察人、經理人或其他職員，應將有關公司業務及財務之一切帳冊、文件及財產，移交重整人。
>
> 公司之董事、監察人、經理人或其他職員，非經重整人同意，不得離開其住居所所在地之法院管轄區域。

## 白話解說

本條規定重整裁定後，公司經營權和財產管理權的歸屬變更，以及原公司管理階層的義務。

重整裁定生效後，公司的業務經營權及財產管理處分權全面移轉給重整人。原有的董事會、監察人等公司機關雖然存續，但其權限已被凍結，由重整人全權負責公司的營運管理。

原公司的董事、監察人、經理人及其他職員有義務將所有業務及財務相關的帳冊、文件及財產移交給重整人，以確保重整人能夠完整掌握公司狀況並順利執行重整任務。

此外，為配合重整程序的進行，公司的董事、監察人、經理人及其他職員非經重整人同意，不得離開其住居所所在地之法院管轄區域，以確保重整人隨時可以詢問調查相關事項。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 經裁定重整之公司及其董事、監察人、經理人 |
| 核心規定 | 公司經營權及財產管理權移屬重整人；原管理階層應移交帳冊文件 |
| 違反效果 | 拒絕移交者可能構成妨害重整；未經同意離開管轄區域者負法律責任 |
| 注意事項 | 原董事、監察人、經理人非經重整人同意不得離開住居所地法院管轄區域 |

## 實務案例

**情境：** 某公司被裁定重整後，原董事長拒絕將公司大小章、銀行帳戶存摺及重要合約文件移交給重整人，並計畫出國。

**分析：** 依第 293 條第 2 項，原董事長有法定義務移交所有帳冊文件及財產。依第 3 項，非經重整人同意不得離開管轄區域。

**結論：** 重整人得報請法院強制原董事長移交相關文件財產，並限制其出境。原董事長拒絕移交及擅自出國的行為，可能構成妨害重整程序之違法行為。
