# 第 304 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 重整計畫未得關係人會議各組之可決時，重整監督人應即報告法院。法院得命關係人會議在一個月內再為審查，經再次審查仍未獲各組之可決時，法院得依公正合理之原則，指示變更方針，命重整人於一定期間內提出新重整計畫。
>
> 前項新重整計畫，仍未獲關係人會議各組之可決時，法院得以裁定認可之。但法院為前述裁定前，應使關係人有陳述意見之機會。

## 白話解說

本條處理重整計畫未獲關係人會議通過時的補救程序，為重整提供多次嘗試的機會。

當重整計畫第一次未獲各組可決時，重整監督人應報告法院。法院可以命關係人會議在一個月內重新審查，給予雙方溝通協商的時間。如果第二次仍未通過，法院可以依據公正合理的原則，指示重整人調整計畫方向，並在一定期間內提出新的重整計畫。

即使新的重整計畫仍未獲各組可決，法院仍有最後手段——可以以裁定方式逕行認可重整計畫。但法院在做此裁定前，必須給予所有關係人陳述意見的機會，以保障其正當程序權利。

此規定體現了法院在重整程序中的積極角色，避免因個別組別的反對而導致有重整價值的公司無法獲得重整機會。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 重整計畫未獲關係人會議通過時 |
| 核心規定 | 法院得命再審查、指示變更方針或逕行認可 |
| 違反效果 | 法院逕行認可前須給予關係人陳述意見之機會 |
| 注意事項 | 法院指示應依公正合理原則；新計畫仍未通過時法院得逕行裁定認可 |

## 實務案例

**情境：** 某公司重整計畫經三次關係人會議，均因有擔保債權人組不同意而未通過。法院依公正合理原則指示修改，但有擔保債權人組仍堅持反對。

**分析：** 依第 304 條第 2 項，新重整計畫仍未獲各組可決時，法院得以裁定逕行認可。但法院在裁定前應讓所有關係人（包括有擔保債權人組）有陳述意見之機會。法院應審酌計畫內容是否符合公正合理原則。

**結論：** 如法院認為重整計畫對有擔保債權人之清償條件已屬公正合理，得以裁定逕行認可。有擔保債權人如不服，得對該裁定提起抗告。
