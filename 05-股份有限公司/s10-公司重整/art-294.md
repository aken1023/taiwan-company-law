# 第 294 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 裁定重整後，對公司之破產、和解、強制執行及因財產關係所生之訴訟等程序，當然停止。

## 白話解說

本條規定重整裁定後最重要的法律效果之一：所有針對公司的破產程序、和解程序、強制執行程序，以及因財產關係所生的訴訟程序，都當然停止（即自動停止，無需另行聲請或裁定）。

這項規定的目的在於為重整程序創造一個「保護傘」環境，使重整人能夠在不受外部追索壓力的情況下，專注於公司的重整工作。如果在重整期間仍允許債權人個別追索，可能導致公司資產被瓜分而無法完成重整。

「當然停止」意味著停止效力是法律直接規定的，無需法院另行裁定。所有相關的執行法院和訴訟法院，在知悉重整裁定後，即應依法停止相關程序。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 經裁定重整之公司 |
| 核心規定 | 破產、和解、強制執行及財產關係訴訟程序當然停止 |
| 違反效果 | 停止後所為之程序行為無效 |
| 注意事項 | 「當然停止」無需另行裁定，法律直接生效 |

## 實務案例

**情境：** 某公司被裁定重整。裁定前，債權人甲已對公司名下廠房聲請強制執行拍賣，且拍賣程序已進行至鑑價階段。重整裁定送達後，執行法院是否應停止拍賣程序？

**分析：** 依第 294 條，裁定重整後，對公司之強制執行程序「當然停止」。不論強制執行程序進行到何階段，均應停止。

**結論：** 執行法院應立即停止對公司廠房之拍賣程序。待重整程序結束後，視重整結果再決定是否續行強制執行。
