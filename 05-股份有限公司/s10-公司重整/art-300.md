# 第 300 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 關係人會議，為重整債權人及股東組成之會議，以審議及表決重整計畫。
>
> 關係人會議由重整監督人為主席。
>
> 關係人會議之關係人，依其權利之性質分組行使其表決權。
>
> 關係人會議開會時，重整人及公司負責人應列席備詢。

## 白話解說

本條規定關係人會議的組成、功能及運作方式。

關係人會議是公司重整程序中最重要的決策機構，由所有重整債權人及股東組成。其主要功能是審議及表決重整人所提出的重整計畫，決定公司未來的命運。

會議由重整監督人擔任主席，負責主持議事。參加會議的關係人依其權利性質分組行使表決權，例如有擔保債權人一組、無擔保債權人一組、股東一組等，各組獨立表決，避免不同性質的利害關係人在表決時發生利益衝突。

在會議進行時，重整人及公司負責人應列席備詢，隨時回答關係人的提問，使關係人充分了解公司狀況及重整計畫的內容後再做出決定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 重整債權人及股東 |
| 核心規定 | 關係人會議審議及表決重整計畫 |
| 會議主席 | 重整監督人 |
| 注意事項 | 關係人依權利性質分組表決；重整人及公司負責人應列席備詢 |

## 實務案例

**情境：** 某公司重整關係人會議召開，關係人分為三組：有擔保債權人組（5 人）、無擔保債權人組（30 人）、股東組（200 人）。重整人提出重整計畫，但有擔保債權人組對計畫中的清償比例不滿。

**分析：** 依第 300 條第 3 項，關係人依權利性質分組表決。各組獨立進行表決，任何一組未通過都可能影響重整計畫的成立。有擔保債權人對清償比例有疑慮，得在會議中要求重整人說明。

**結論：** 重整人應列席說明計畫內容及清償比例的合理性。如有擔保債權人組不同意，重整人可能需要修改計畫或由法院依第 305 條處理。
