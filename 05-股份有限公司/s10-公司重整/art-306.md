# 第 306 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 法院未認可重整計畫時，應裁定終止重整。但其未認可係因重整計畫有不當者，得裁定命重整人限期補正。
>
> 前項但書，以一次為限。

## 白話解說

本條規定法院不認可重整計畫時的處理方式。

如果法院審查後認為重整計畫無法認可，原則上應裁定終止重整程序。重整終止後，公司回復原來的狀態，先前暫停的各種程序（如強制執行）也會恢復進行。

但如果法院不認可的原因僅是計畫內容有不當之處（例如某些條件不合理，但非根本無法重整），法院可以不直接終止重整，而是裁定命重整人在限期內補正修改計畫。

這項補正機會只有一次，避免程序無限拖延。如果補正後的計畫仍不獲認可，法院即應裁定終止重整。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 法院審查重整計畫後不予認可時 |
| 核心規定 | 原則上裁定終止重整；計畫有不當者得命限期補正（以一次為限） |
| 違反效果 | 重整終止後，公司回復原狀態，暫停之程序恢復 |
| 注意事項 | 補正機會僅限一次 |

## 實務案例

**情境：** 法院審查某公司之重整計畫，認為計畫中對無擔保債權人之清償比例過低（僅 5%），有失公正，擬不予認可。

**分析：** 依第 306 條但書，法院認為不認可的原因係計畫「有不當」（清償比例過低），而非根本無重整可能。法院得裁定命重整人限期補正，調高清償比例。

**結論：** 法院命重整人於 30 日內補正，提出合理的清償比例。如重整人補正後的計畫仍不獲認可，法院即裁定終止重整。
