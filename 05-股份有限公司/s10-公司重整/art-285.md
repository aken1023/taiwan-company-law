# 第 285 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 法院於裁定重整前，得就對公司業務具有專門學識、經營經驗而非利害關係人者，選任為檢查人，就下列事項於選任後三十日內調查完畢報告法院：
>
> 一、公司業務及財務狀況。
>
> 二、公司有無重整之可能及其理由。
>
> 三、其他有關重整之方案。
>
> 前項檢查人，得向公司之董事、監察人、經理人及其他職員為必要之調查及詢問。
>
> 公司之董事、監察人、經理人或其他職員，對於檢查人之調查詢問，無正當理由不得拒絕。
>
> 法院對於第一項檢查人之報告，認為必要時，得命其補充調查。

## 白話解說

本條賦予法院在正式裁定是否准許重整之前，選任專業檢查人進行調查的權力，以協助法院做出正確判斷。

檢查人的資格要求是：對公司業務具有專門學識及經營經驗，且非該公司之利害關係人（以確保客觀公正）。實務上通常選任會計師、律師或企管顧問等專業人士。

檢查人的調查範圍包括三大項：公司業務及財務狀況的全面了解、公司有無重整可能的專業評估及理由說明、以及其他有關重整方案的建議。檢查人必須在選任後三十日內完成調查並向法院報告。

為確保調查順利進行，檢查人有權向公司的董事、監察人、經理人及其他職員進行必要的調查詢問，而公司人員無正當理由不得拒絕。如果法院認為報告內容不夠完整，還可以命檢查人補充調查。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 法院審查重整聲請期間 |
| 核心規定 | 法院得選任檢查人調查公司狀況，30 日內報告 |
| 檢查人資格 | 具專門學識及經營經驗之非利害關係人 |
| 注意事項 | 公司人員不得無正當理由拒絕調查詢問；法院得命補充調查 |

## 實務案例

**情境：** 法院受理某公司之重整聲請後，選任甲會計師為檢查人。甲於調查期間請公司財務長提供近三年財務報表及相關帳冊，財務長以「商業機密」為由拒絕提供。

**分析：** 依第 285 條第 3 項，公司職員對檢查人之調查詢問，無正當理由不得拒絕。「商業機密」並非拒絕配合法院所選任檢查人調查的正當理由，因為檢查人有保密義務。

**結論：** 財務長無正當理由拒絕提供資料，違反本條第 3 項規定。法院得對其為適當之處分。建議公司全面配合檢查人調查，以利重整程序順利進行。
