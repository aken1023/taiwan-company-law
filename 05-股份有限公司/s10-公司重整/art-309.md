# 第 309 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 公司在重整期間內，除依重整計畫之規定外，公司之破產、解散及股息紅利之分派等，均不適用之。

## 白話解說

本條規定公司在重整期間內的法律限制，排除若干在正常狀態下適用的法律規定。

公司在重整期間，除非重整計畫另有規定，否則不得進入破產程序、不得解散，也不得分派股息紅利。

排除破產程序是因為重整的目的就是避免公司走向破產，在重整期間不應同時進行破產。排除解散是因為重整的前提是公司繼續存續並恢復營運。排除股息紅利之分派則是為了保護債權人的利益——在公司尚未完成重整、債務尚未清償前，不應將公司資產以股息紅利方式分配給股東。

但如果重整計畫中有特別規定（例如計畫中安排在一定條件下可以分派股息），則從其規定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 重整期間之公司 |
| 核心規定 | 除依重整計畫外，破產、解散及股息紅利分派均不適用 |
| 違反效果 | 在重整期間進行上述行為屬違法 |
| 注意事項 | 重整計畫有特別規定者從其規定 |

## 實務案例

**情境：** 某公司在重整期間，公司有一筆營業利潤。部分股東要求依據章程規定分派股息。

**分析：** 依第 309 條，公司在重整期間，除依重整計畫之規定外，股息紅利之分派不適用。即使公司有盈餘，在重整未完成前，該盈餘應優先用於清償債務及執行重整計畫。

**結論：** 除非重整計畫中有允許分派股息的特別安排，否則公司不得在重整期間分派股息紅利。股東的請求應予駁回。
