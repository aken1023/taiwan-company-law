# 第 302 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 關係人會議對重整計畫之決議，應分下列各組行之：
>
> 一、有擔保重整債權人組。
>
> 二、無擔保重整債權人組。
>
> 三、股東組。
>
> 前項決議，各組均應有出席權額三分之二以上之同意行之。

## 白話解說

本條規定重整計畫在關係人會議中的分組表決方式及通過門檻。

重整計畫的表決分為三組進行：有擔保重整債權人組（即其債權有擔保物作保障的債權人）、無擔保重整債權人組（普通債權人），以及股東組。分組的目的在於保護不同性質利害關係人的利益，避免一方利用多數優勢壓迫另一方。

每一組的通過門檻都是出席權額三分之二以上同意。「權額」是指出席者所代表的債權金額或股份數額。換言之，不是單純計算人頭數，而是按照各人所代表的債權金額或股份比例來計算表決結果。

三組必須全部通過，重整計畫才算在關係人會議中獲得決議。任何一組未達到三分之二同意門檻，重整計畫即無法通過。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 關係人會議表決重整計畫 |
| 核心規定 | 分三組表決，各組須出席權額 2/3 以上同意 |
| 三組分類 | 有擔保債權人組、無擔保債權人組、股東組 |
| 注意事項 | 以權額計算（非人頭數）；三組均須通過 |

## 實務案例

**情境：** 某公司重整之關係人會議中，有擔保債權人組出席權額 80%，其中 70% 同意（達 2/3）；無擔保債權人組出席權額 90%，其中 60% 同意（未達 2/3）；股東組出席權額 75%，其中 80% 同意（達 2/3）。

**分析：** 依第 302 條，三組均須出席權額 2/3 以上同意。有擔保債權人組（70%）及股東組（80%）均達門檻，但無擔保債權人組（60%）未達 2/3，重整計畫在關係人會議中未獲通過。

**結論：** 重整計畫未獲關係人會議通過。重整人得修改計畫（特別是針對無擔保債權人之清償條件），再次提交表決。或依第 305 條第 2 項，由法院審酌是否逕行認可。
