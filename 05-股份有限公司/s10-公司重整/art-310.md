# 第 310 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 重整計畫之執行，除依其規定外，由重整人為之。
>
> 重整人執行重整計畫，應受重整監督人之監督。
>
> 重整計畫之執行完畢後，重整人應即報告法院。法院經調查認為確已完成時，應為重整完成之裁定。

## 白話解說

本條規定重整計畫的執行主體、監督機制及重整完成的確認程序。

重整計畫經法院認可後，原則上由重整人負責執行。重整計畫中如有特別規定某些事項由其他人執行者，從其規定。

在整個執行過程中，重整人必須接受重整監督人的監督。重整監督人有權查核重整人是否確實按照計畫執行，有無偏離計畫或有不當行為。

當重整計畫執行完畢時，重整人應向法院報告。法院會進行調查，確認重整計畫是否確實完成。經調查確認後，法院應作成「重整完成」的裁定，正式宣告重整程序圓滿結束。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 重整計畫經認可後之執行階段 |
| 核心規定 | 由重整人執行計畫，受重整監督人監督；執行完畢後法院裁定重整完成 |
| 違反效果 | 重整人未依計畫執行者，得依第 307 條終止重整 |
| 注意事項 | 法院須調查確認後始得為重整完成之裁定 |

## 實務案例

**情境：** 某公司重整計畫經認可後，重整人花費 2 年完成所有計畫內容：清償債務至約定比例、完成組織調整、引進新投資人並恢復正常營運。重整人向法院報告計畫執行完畢。

**分析：** 依第 310 條第 3 項，重整人報告後，法院應進行調查確認。法院可以查閱重整人的執行報告、徵詢重整監督人意見、審酌債務清償情形等。

**結論：** 如法院調查認為重整計畫確已完成，應為重整完成之裁定。重整完成後，公司回復正常運作，重整人及重整監督人的職務終了。
