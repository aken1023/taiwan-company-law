# 第 299 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 公司債權人或股東對於前條清冊所載各事項有異議者，應於第一次關係人會議前，向重整監督人提出之。
>
> 重整監督人應將異議之各事項，於第一次關係人會議中報告之。
>
> 經重整監督人調查後仍有爭議者，應報請法院裁定之。

## 白話解說

本條規定對債權清冊及股東清冊內容有異議時的處理程序。

如果債權人或股東認為清冊所載的債權金額、順位、擔保內容或股東權等事項有錯誤，應在第一次關係人會議召開前，向重整監督人提出異議。這個時限的設定是為了讓異議能夠在關係人會議中一併處理。

重整監督人收到異議後，應在第一次關係人會議中報告這些異議事項，讓所有利害關係人知悉。重整監督人會先行調查異議的事實，如果調查後異議仍無法解決（雙方仍有爭議），則應報請法院以裁定方式解決爭議。

法院的裁定具有確定力，可以終局解決債權金額及順位的爭議，確保重整程序能夠順利進行。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 對債權清冊或股東清冊有異議之債權人或股東 |
| 核心規定 | 應於第一次關係人會議前向重整監督人提出異議 |
| 爭議解決 | 經調查仍有爭議者，報請法院裁定 |
| 注意事項 | 異議期限為第一次關係人會議前；重整監督人應於會議中報告異議事項 |

## 實務案例

**情境：** 甲債權人閱覽清冊後發現，其債權金額被記載為 300 萬元，但實際應為 500 萬元。甲在第一次關係人會議前向重整監督人提出異議，但重整監督人認為甲提出之證據不足。

**分析：** 依第 299 條，甲已依法在期限內提出異議。重整監督人應在關係人會議中報告此異議。因調查後仍有爭議，應報請法院裁定甲的債權金額究竟為 300 萬元或 500 萬元。

**結論：** 法院將審酌甲及公司雙方所提出的證據，以裁定確認甲的正確債權金額。此裁定將作為重整計畫中清償甲債權的依據。
