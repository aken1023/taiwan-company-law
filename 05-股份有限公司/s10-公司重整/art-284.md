# 第 284 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 法院除為前條裁定外，應將聲請書副本，送達公司之主管機關及目的事業中央主管機關，並徵詢其關於應否重整之意見；其意見應於十日內送達法院。

## 白話解說

本條規定法院在審查重整聲請時，除了可能依前條裁定駁回外，應將聲請書副本送達公司的主管機關（經濟部或地方主管機關）及公司所屬產業的目的事業中央主管機關（如金融業為金管會、通訊業為 NCC 等），並徵詢他們對於公司是否應該進行重整的意見。

這些主管機關對公司的營運狀況、產業前景有較深入的了解，其意見可作為法院判斷重整可行性的重要參考。主管機關應於收到徵詢後十日內將意見送達法院，以避免程序遲延。

這項規定體現了公司重整不僅是私法上的程序，更涉及公共利益（如員工就業、產業發展），因此需要行政機關參與提供專業意見。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 受理重整聲請之法院 |
| 核心規定 | 法院應將聲請書副本送主管機關及目的事業中央主管機關，徵詢意見 |
| 回覆期限 | 主管機關意見應於十日內送達法院 |
| 注意事項 | 主管機關意見為法院審查之重要參考，但不拘束法院裁定 |

## 實務案例

**情境：** 某上市電子公司聲請重整，法院將聲請書副本送達經濟部商業司及經濟部工業局。工業局回覆意見認為該公司所屬產業仍有發展前景，重整有利於維持產業供應鏈完整性。

**分析：** 依第 284 條，法院徵詢主管機關意見為法定程序。目的事業主管機關的正面意見，有助於法院認定公司有重整之可能及必要。

**結論：** 法院在裁定是否准許重整時，將參考主管機關的意見，但仍應綜合考量公司整體財務狀況及其他因素後作成裁定。
