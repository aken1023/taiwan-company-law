# 第 307 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十節 公司重整

## 條文原文

> 有下列情事之一者，法院得依職權或因重整監督人、目的事業中央主管機關或利害關係人之聲請，以裁定終止重整：
>
> 一、重整人執行職務有重大違法或不當情事，經法院解任後，無適當之人可資選任者。
>
> 二、公司有隱匿或毀損財產，或為其他不利於債權人之行為者。
>
> 三、公司經重整而顯無重整之可能者。
>
> 四、公司無正當理由對關係人為差別待遇者。
>
> 五、重整計畫所定期限屆滿而未完成重整者。

## 白話解說

本條規定法院得裁定終止重整的五種情形。即使重整程序已經開始，甚至重整計畫已經認可，如果出現本條所列情事，法院仍得終止重整。

第一種是重整人執行職務嚴重不當，被法院解任後又找不到合適的替代人選。第二種是公司本身有惡意行為，如隱匿或毀損財產、做出不利債權人之行為。第三種是重整後發現公司已經不可能恢復營運。第四種是公司對某些關係人給予差別待遇（如偏厚某債權人而虧待其他人）。第五種是重整計畫所定的執行期限已屆滿，但重整仍未完成。

法院可以依職權或因重整監督人、主管機關、利害關係人的聲請而終止重整。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 重整程序進行中之公司 |
| 核心規定 | 列舉五款法院得裁定終止重整之情事 |
| 聲請權人 | 法院依職權、重整監督人、目的事業主管機關、利害關係人 |
| 注意事項 | 重整終止後公司回復原狀態，可能進入破產清算程序 |

## 實務案例

**情境：** 某公司重整計畫經法院認可後開始執行，執行期限為 3 年。但 3 年屆滿後，公司僅完成計畫內容的 40%，仍有大量債務未依計畫清償。

**分析：** 依第 307 條第 5 款，重整計畫所定期限屆滿而未完成重整者，法院得裁定終止重整。本案計畫執行率僅 40%，顯然未能在期限內完成。

**結論：** 法院得依職權或因利害關係人聲請，裁定終止重整。重整終止後，債權人得就未獲清償部分繼續向公司追索，公司可能面臨破產清算。
