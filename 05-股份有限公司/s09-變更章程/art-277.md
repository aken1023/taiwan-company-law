# 第 277 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第九節 變更章程

## 條文原文

> 公司非經股東會代表已發行股份總數三分之二以上股東之出席，以出席股東表決權過半數之同意，不得變更章程。
>
> 公開發行股票之公司，出席股東之股份總數不足前項定額者，得以有代表已發行股份總數過半數股東之出席，出席股東表決權三分之二以上之同意行之。
>
> 前二項出席股東股份總數及表決權數，章程有較高之規定者，從其規定。

## 白話解說

本條規定了股份有限公司變更章程的表決門檻，屬於「特別決議」事項，其要求比一般決議更為嚴格。

對於一般（非公開發行）公司而言，變更章程必須有代表已發行股份總數三分之二以上的股東出席股東會，且出席股東表決權過半數同意，才能通過。這是為了確保章程這種公司根本大法的修改，必須獲得足夠多數股東的支持。

對於公開發行股票的公司，由於股東人數眾多、出席率通常較低，法律設有較寬鬆的替代門檻：只要有代表已發行股份總數過半數的股東出席，出席股東表決權三分之二以上同意即可。這是考量公開發行公司股權分散的實際狀況所做的彈性規定。

此外，公司章程可以訂定比法律規定更高的出席及表決門檻，但不得低於法定最低標準。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 所有股份有限公司 |
| 核心規定 | 變更章程須經股東會特別決議通過 |
| 一般公司門檻 | 代表已發行股份總數 2/3 以上出席，出席股東表決權過半數同意 |
| 公開發行公司門檻 | 代表已發行股份總數過半數出席，出席股東表決權 2/3 以上同意 |
| 注意事項 | 章程得訂定較高門檻，但不得低於法定標準 |

## 實務案例

**情境：** 某股份有限公司（非公開發行）已發行股份總數為 1,000 萬股，擬於股東會中修改章程增設特別股條款。股東會當日出席股東代表 600 萬股，其中 350 萬股同意修改。

**分析：** 依第 277 條第 1 項，變更章程需代表已發行股份總數 2/3 以上（即約 667 萬股）之股東出席。本案僅 600 萬股出席，未達法定出席門檻。

**結論：** 因出席股份數不足法定門檻，該次章程修改決議不成立，公司應另行召集股東會，並積極徵求委託書以提高出席率。
