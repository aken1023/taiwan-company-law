# 第 280 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第九節 變更章程

## 條文原文

> 前條之規定，於股份合併時準用之。

## 白話解說

本條為準用條文，規定當公司進行股份合併時，應準用第 279 條關於減資換發新股票的程序規定。

所謂股份合併，是指公司將數股合併為一股，例如將兩股合併為一股，使股份總數減少但每股面額或價值提高。這種操作在實質上類似於減資後換發新股票，因此法律規定準用相同程序。

準用的結果是：公司進行股份合併時，也必須給股東至少六個月的換取期限，通知各股東前來換取合併後的新股票，逾期未換取者同樣喪失其股東權利，公司得拍賣其股份並以賣得金額給付該股東。公司負責人如未依限通知或公告，也會受到罰鍰處分。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 進行股份合併之股份有限公司 |
| 核心規定 | 股份合併時準用第 279 條減資換發新股票之程序 |
| 違反效果 | 準用第 279 條之罰則，逾期不換取者喪失股東權利 |
| 注意事項 | 須通知股東並給予至少六個月之換取期限 |

## 實務案例

**情境：** 某公司股東會決議將每兩股合併為一股。公司通知股東於六個月內持舊股票前來換取新股票。股東乙持有奇數股（101 股），合併後應得 50.5 股。

**分析：** 股份合併時準用第 279 條程序，六個月換取期限合法。至於合併後產生的畸零股（0.5 股），實務上公司通常會以現金補償畸零股部分，或安排股東間相互搓合湊足整股。

**結論：** 公司應妥善處理畸零股問題，可於股東會決議中一併處理畸零股之處理方式，如以現金按比例退還。股東應於期限內完成換取，以保障自身權利。
