# 第 356-5 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十三節 閉鎖性股份有限公司

## 條文原文

> 閉鎖性股份有限公司之章程，應訂明股份轉讓之限制。
>
> 前項股份轉讓之限制，應於股票以明顯文字註記，並於股東名簿記載之。

## 白話解說

閉鎖性公司的章程中必須明確訂定股份轉讓的限制條件。例如可以規定股份轉讓須經董事會同意、或其他股東有優先承購權等。這些轉讓限制必須在股票上以明顯文字註記，讓取得股票的人清楚知道有轉讓限制，並且要在股東名簿中記載這些限制。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 閉鎖性股份有限公司 |
| 核心規定 | 章程應訂明股份轉讓限制，並於股票註記及股東名簿記載 |
| 違反效果 | 未訂明轉讓限制者，不符閉鎖性公司要件 |
| 注意事項 | 轉讓限制為閉鎖性公司之核心特徵；須於股票明顯標示 |

## 實務案例

**情境：** 甲閉鎖性公司章程規定「股東轉讓股份須經董事會同意」。股東乙未經董事會同意即將股份轉讓給公司外部之丙。

**分析：** 依第356-5條，章程訂明之股份轉讓限制具有效力。乙未經董事會同意即轉讓股份，違反章程之限制規定。

**結論：** 該轉讓不生效力，丙不能取得甲公司之股東身分。乙應先取得董事會同意後方得轉讓。
