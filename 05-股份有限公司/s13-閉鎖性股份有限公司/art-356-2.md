# 第 356-2 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第十三節 閉鎖性股份有限公司

## 條文原文

> 閉鎖性股份有限公司之章程，應記載閉鎖性之文字。

## 白話解說

閉鎖性股份有限公司的章程中，必須明確記載公司是「閉鎖性」的文字，讓所有人清楚知道這是一家閉鎖性公司。這是一項強制性的章程記載事項，公司名稱或章程中必須有明確標示。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 閉鎖性股份有限公司 |
| 核心規定 | 章程應記載「閉鎖性」之文字 |
| 違反效果 | 章程欠缺此記載者，可能影響閉鎖性公司之認定 |
| 注意事項 | 為強制記載事項 |

## 實務案例

**情境：** 甲公司設立登記時章程記載「甲閉鎖性股份有限公司」，明確表明其閉鎖性質。

**分析：** 依第356-2條，甲公司章程已記載「閉鎖性」文字，符合法定要求。

**結論：** 甲公司章程之記載符合規定，得適用閉鎖性公司之相關特別規定。
