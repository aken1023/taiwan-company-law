# 第 261 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第七節 公司債

## 條文原文

> 無記名公司債之持有人，得隨時請求改為記名式。

## 白話解說

本條賦予無記名公司債持有人單方面請求將其債券改為記名式的權利。無記名公司債的特性是債券上不記載持有人姓名，交付即可轉讓，流通性較高。但無記名債券也有遺失、被盜的風險較大的缺點。

因此法律允許持有人隨時請求公司將其無記名債券改為記名式，以獲得較高的安全保障。改為記名式後，轉讓須依第 260 條的規定辦理登記，安全性提高但流通性相對降低。

公司不得拒絕持有人的此項請求，且應在存根簿上登記新的記名債權人資料。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 無記名公司債之持有人 |
| 核心規定 | 得隨時請求改為記名式 |
| 違反效果 | 公司不得拒絕，否則持有人得訴請履行 |
| 注意事項 | 改為記名式後適用第 260 條之轉讓規定 |

## 實務案例

**情境：** 丁持有甲公司之無記名公司債 500 萬元，因擔心債券遺失，向甲公司請求改為記名式。甲公司以「公司內部行政作業繁忙」為由拒絕。

**分析：** 依本條規定，無記名公司債持有人得「隨時」請求改為記名式，這是持有人的法定權利。甲公司不得以行政作業繁忙為由拒絕。

**結論：** 甲公司應配合辦理，將丁之姓名記載於債券及存根簿上，完成記名式之轉換。如甲公司持續拒絕，丁得訴請法院強制甲公司履行。
