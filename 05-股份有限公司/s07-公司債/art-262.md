# 第 262 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第七節 公司債

## 條文原文

> 公司發行轉換公司債者，於發行後至轉換完成前，其已發行股份之總數加計可轉換股份之總數，如超過公司章程所定股份總數時，董事會應即召開股東會，決議辦理變更章程增加資本額。但公開發行股票之公司，證券管理機關另有規定者，從其規定。
>
> 前項規定，於發行附認股權公司債者，準用之。

## 白話解說

本條處理的是轉換公司債和附認股權公司債可能導致公司發行股份超過章程所定股份總數的問題。當公司發行了可轉換為股份的公司債後，如果將來所有債券都轉換成股份，加上已發行的股份，可能會超過公司章程所定的資本額上限。

遇到這種情形，董事會應即召開股東會，決議修改章程以增加公司的資本額，確保有足夠的股份額度來因應轉換需求。這是為了避免屆時債權人要求轉換時，公司卻無法發行新股的窘境。

對於公開發行股票的公司，證券管理機關可能另有較為彈性的規定。附認股權公司債也準用相同的規定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 發行轉換公司債或附認股權公司債之公司 |
| 核心規定 | 如轉換後股數超過章程所定總數，董事會應即召開股東會修章增資 |
| 違反效果 | 未及時辦理增資，可能導致無法履行轉換義務 |
| 注意事項 | 公開發行公司得依證券管理機關規定辦理；附認股權公司債準用本條 |

## 實務案例

**情境：** 甲公司章程所定股份總數為 1 億股，已發行 8,000 萬股。現發行可轉換公司債，轉換後可取得 3,000 萬股。

**分析：** 已發行 8,000 萬股 + 可轉換 3,000 萬股 = 1.1 億股，超過章程所定的 1 億股。依本條規定，董事會應即召開股東會決議修改章程，將股份總數增加至至少 1.1 億股。

**結論：** 甲公司董事會應盡速召開股東會辦理變更章程增加資本額。如未及時修章，屆時債券持有人請求轉換時，公司將面臨無法履行轉換義務的違約風險。
