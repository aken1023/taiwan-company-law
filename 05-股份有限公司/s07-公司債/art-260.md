# 第 260 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第七節 公司債

## 條文原文

> 記名式之公司債，其轉讓，非將受讓人之姓名或名稱記載於債券，並將受讓人之姓名或名稱及住所或居所記載於公司債存根簿，不得以其轉讓對抗公司。

## 白話解說

本條規定記名式公司債轉讓的對抗要件。記名公司債的轉讓在當事人之間（即讓與人和受讓人之間）只要雙方合意、交付債券即可生效。但如果要讓這個轉讓對公司也發生效力（即「對抗公司」），必須完成兩個步驟：

第一，將受讓人的姓名或名稱記載於債券上。第二，將受讓人的姓名或名稱及住所記載於公司的公司債存根簿上。

只有完成這兩個步驟後，受讓人才能直接向公司主張其公司債債權人的地位，請求公司支付利息和本金。否則公司只認存根簿上登記的原債權人。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 記名式公司債之轉讓 |
| 核心規定 | 須將受讓人記載於債券及存根簿，方得對抗公司 |
| 違反效果 | 未完成登記，不得以轉讓對抗公司（公司仍對原登記之債權人負責） |
| 注意事項 | 轉讓在當事人間仍有效，僅不得對抗公司；無記名公司債之轉讓見第 261 條 |

## 實務案例

**情境：** 甲持有乙公司之記名公司債 1,000 萬元，將之轉讓予丙。丙取得債券但未在債券及存根簿上辦理登記。利息支付日到期時，丙向乙公司請求支付利息。

**分析：** 依本條規定，丙雖已取得債券，但未將姓名記載於債券及存根簿上，不得以轉讓對抗乙公司。乙公司得拒絕向丙支付利息，而仍向存根簿上登記之甲支付。

**結論：** 丙應儘速辦理轉讓登記（於債券及存根簿上記載），否則無法直接向乙公司主張債權。在完成登記前，丙僅得向甲請求轉交利息款項。
