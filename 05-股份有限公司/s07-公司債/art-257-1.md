# 第 257-1 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第七節 公司債

## 條文原文

> 公司債得免印製債券，應洽證券集中保管事業機構登載於其帳簿或以其他方式交付之。

## 白話解說

本條允許公司債發行時不必印製實體債券，改以在證券集中保管事業機構（如臺灣集中保管結算所）的帳簿上登載的方式來發行。這就是所謂的「無實體發行」或「帳簿劃撥」制度。

採用帳簿劃撥方式發行公司債，有多項優點：減少印製實體債券的成本、避免債券遺失或偽造的風險、提高交易效率，且符合現代證券市場電子化的趨勢。目前實務上，公開發行公司所發行的公司債幾乎都採取無實體方式發行。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 發行公司債之股份有限公司 |
| 核心規定 | 公司債得免印製實體債券，以帳簿劃撥方式發行 |
| 違反效果 | 無特定違反效果，本條為授權規定 |
| 注意事項 | 應洽證券集中保管事業機構辦理登載；無實體發行已成實務主流 |

## 實務案例

**情境：** 丙公司擬發行 20 億元公司債，考慮到印製實體債券費用高昂且管理不便，決定採無實體方式發行。

**分析：** 依本條規定，公司債得免印製債券，洽證券集中保管事業機構（臺灣集中保管結算所）以帳簿劃撥方式登載即可。此為法律明文允許。

**結論：** 丙公司得免印製實體債券，向臺灣集中保管結算所申請以帳簿劃撥方式發行。持有人之權利以帳簿記載為準。
