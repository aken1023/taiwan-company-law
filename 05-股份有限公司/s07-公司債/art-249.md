# 第 249 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第七節 公司債

## 條文原文

> 有下列情形之一者，不得發行無擔保公司債：
> 一、對於前已發行之公司債或其他債務，曾有違約或遲延支付本息之事實已了結，自了結之日起三年內。
> 二、最近三年或開業不及三年之開業年度課稅後之平均淨利，未達原定發行之公司債應負擔年息總額之百分之一百五十者。

## 白話解說

本條規定了兩種不得發行無擔保公司債的情形，目的是保護投資人免受信用不佳公司的無擔保債券風險。

第一種情形是：公司過去曾對已發行的公司債或其他債務有違約或遲延支付本息的紀錄。即使該違約已經了結（清償完畢），在了結之日起三年內，仍然不得發行無擔保公司債。這是因為有違約紀錄的公司，信用狀況較差。

第二種情形是：公司最近三年（或開業不滿三年的全部年度）的平均課稅後淨利，不到擬發行無擔保公司債應付年息的 150%。換言之，公司的獲利能力必須足以涵蓋公司債的利息支出，且有相當的安全邊際。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 擬發行無擔保公司債之公司 |
| 核心規定 | 曾違約且了結未滿三年者、平均淨利未達年息 150% 者，不得發行 |
| 違反效果 | 違規發行之無擔保公司債，主管機關得撤銷核准 |
| 注意事項 | 獲利門檻以課稅後平均淨利計算；違約紀錄以了結日起算三年 |

## 實務案例

**情境：** 戊公司擬發行年利率 5% 的無擔保公司債 10 億元，其最近三年的課稅後淨利分別為 8,000 萬、7,000 萬、6,000 萬元。

**分析：** 平均淨利 = (8,000 + 7,000 + 6,000) / 3 = 7,000 萬元。年息總額 = 10 億 x 5% = 5,000 萬元。7,000 / 5,000 = 140%，未達 150% 的門檻。

**結論：** 戊公司不得發行該無擔保公司債。如欲募集，可考慮提供擔保，或降低發行金額至年息總額使淨利達到 150% 以上。
