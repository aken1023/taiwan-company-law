# 第 200-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 經法院裁判解任之董事，自裁判確定日起，三年內不得充任上市、上櫃或興櫃公司之董事、監察人及依第二十七條第一項規定受指定代表行使職務之自然人。

## 白話解說

本條規定經法院裁判解任之董事的「再任禁止」。被法院依第200條裁判解任的董事，自裁判確定之日起三年內，不得擔任上市、上櫃或興櫃公司的董事、監察人，也不得擔任法人代表人行使董事或監察人職務。

此規定的目的是防止違法董事在被裁判解任後，立即轉任其他公開交易公司之董事或監察人，繼續危害投資人及公司利益。三年之禁止期間提供了一定的嚇阻作用。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 經法院裁判解任之董事 |
| 核心規定 | 自裁判確定日起3年內不得充任上市、上櫃或興櫃公司之董事、監察人 |
| 違反效果 | 違反者之就任無效 |
| 注意事項 | 禁止範圍限於上市、上櫃及興櫃公司 |

## 實務案例

**情境：** 甲經法院裁判解任某公司董事，裁判於2024年1月確定。甲於2025年6月受邀擔任另一上市公司董事。

**分析：** 甲自裁判確定日（2024年1月）起3年內（至2027年1月），不得擔任上市公司之董事。2025年6月仍在禁止期間內，甲不得接任該上市公司董事。

**結論：** 經裁判解任之董事應注意3年之再任禁止期間。
