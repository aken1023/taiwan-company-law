# 第 206-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 公開發行股票之公司董事會設有常務董事者，其常務董事會之議事，準用第二百零六條之規定。

## 白話解說

本條規定公開發行公司如有設常務董事會，其議事決議的方式應準用第206條有關董事會決議的規定，即過半數常務董事出席、出席常務董事過半數同意行之。

常務董事會之利害關係揭露義務亦準用第206條之規定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 公開發行公司之常務董事會 |
| 核心規定 | 常務董事會之議事準用§206董事會決議之規定 |
| 違反效果 | 未達法定門檻之決議不成立 |
| 注意事項 | 僅適用於設有常務董事會之公開發行公司 |

## 實務案例

**情境：** 某公開發行公司常務董事會共5名常務董事，某次會議僅2名出席即作成決議。

**分析：** 準用§206規定，常務董事會須過半數出席（至少3名），2名出席未達法定門檻，該決議不成立。

**結論：** 常務董事會之決議門檻與董事會相同，須確保出席及同意人數均達法定要求。
