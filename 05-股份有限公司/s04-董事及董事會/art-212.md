# 第 212 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 股東會決議，對於董事提起訴訟時，公司應自決議之日起三十日內提起之。

## 白話解說

本條規定當股東會決議對董事提起訴訟（例如因董事違法行為致公司受損害而請求賠償），公司必須在股東會決議之日起三十日內提起訴訟。

此規定的目的是避免公司（通常由其他董事控制）在股東會已經決議提起訴訟後故意拖延不提訴，使訴訟因時效或其他原因而喪失。三十日的期限確保了股東會決議能夠被即時落實。

如果公司在三十日內未提起訴訟，股東可能得依第214條之代位訴訟制度自行提起。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股東會決議對董事提起訴訟之公司 |
| 核心規定 | 應自決議日起30日內提起訴訟 |
| 違反效果 | 逾期未提訴可能構成負責人之失職 |
| 注意事項 | 30日為強制期限；公司代表人應積極執行股東會決議 |

## 實務案例

**情境：** JJ公司股東會決議對前任董事長提起損害賠償訴訟。決議後現任董事遲遲不提訴，已超過30日。

**分析：** 依本條，JJ公司應於股東會決議日起30日內提起訴訟。逾期不提訴，現任董事違反本條義務。少數股東得依第214條之代位訴訟程序，以自己名義為公司提起訴訟。

**結論：** 公司應於法定期限內執行股東會之訴訟決議，不得拖延。
