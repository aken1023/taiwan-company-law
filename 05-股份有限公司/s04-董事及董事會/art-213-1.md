# 第 213-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 繼續六個月以上持有已發行股份總數百分之一以上之股東，得以書面請求監察人為公司對董事提起訴訟。監察人自有前項之請求日起，三十日內不提起訴訟時，前項之股東，得為公司提起訴訟。

## 白話解說

本條為第214條少數股東代位訴訟之前置程序。繼續六個月以上持有已發行股份總數百分之一以上的股東，如認為董事有損害公司之行為，得先以書面請求監察人代表公司對董事提起訴訟。

如果監察人收到書面請求後三十日內未提起訴訟，該股東即取得「代位訴訟」的權利，可以自己的名義代表公司對董事提起訴訟。

此程序設計的邏輯是：先讓監察人處理，監察人怠於行使時，少數股東才能介入。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 繼續6個月以上持有已發行股份總數1%以上之股東 |
| 核心規定 | 得書面請求監察人對董事提起訴訟；監察人30日不提訴時股東得代位提訴 |
| 違反效果 | 未經前置程序即提訴者，訴訟可能被駁回 |
| 注意事項 | 須先書面請求監察人；等待30日後方得代位提訴 |

## 實務案例

**情境：** 持有LL公司1.5%股份達一年之股東丁，認為董事長掏空公司，以書面請求監察人提訴。監察人40日仍未提訴。

**分析：** 丁持股1.5%（超過1%）且持有超過6個月，符合本條要件。以書面請求監察人後，監察人超過30日未提訴。丁即取得代位訴訟權，得以自己名義為公司對董事長提起損害賠償訴訟。

**結論：** 少數股東代位訴訟須先踐行書面請求監察人之前置程序。
