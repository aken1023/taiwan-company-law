# 第 201-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 董事辭任，自辭任書送達公司時發生效力。但公司董事缺額達三分之一者，自辭任書送達公司之日起六十日內仍得行使董事職權。

## 白話解說

本條規定董事辭任的生效時點。董事辭任自辭任書送達公司時即發生效力，不需要公司或股東會的同意。這體現了委任關係的特性：受任人得隨時終止委任。

但本條設有例外：如果某位董事辭任後將導致公司董事缺額達三分之一（觸發第201條之補選門檻），該辭任董事雖然辭任已生效，但在辭任書送達後六十日內仍得行使董事職權。此過渡安排是為了避免公司因董事大量辭任而立即陷入運作癱瘓，給予公司足夠時間補選。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 辭任之董事 |
| 核心規定 | 辭任自送達公司時生效；缺額達1/3者，辭任董事60日內仍得行使職權 |
| 違反效果 | 不適用 |
| 注意事項 | 辭任不需公司同意；60日過渡期間僅適用於缺額達1/3之情形 |

## 實務案例

**情境：** EE公司有5名董事，其中第2名董事提出辭任後，又有第3名董事提出辭任。

**分析：** 第2名董事辭任後缺額2/5（40%），已達1/3門檻。依本條但書，第3名辭任之董事在辭任書送達後60日內仍得行使職權，以維持公司基本運作。公司應於此期間內補選董事。

**結論：** 本條兼顧董事辭任自由與公司運作之持續性。
