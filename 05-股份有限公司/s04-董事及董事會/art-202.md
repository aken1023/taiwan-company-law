# 第 202 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 公司業務之執行，除本法或章程規定應由股東會決議之事項外，均應由董事會決議行之。

## 白話解說

本條確立了股份有限公司權限分配的核心原則：董事會是公司業務執行的決策機關。除非法律或章程明確規定某些事項必須由股東會決議，否則公司所有業務的執行都由董事會決議行之。

這意味著董事會享有「概括性」的業務決議權，而股東會的權限則是「列舉性」的。必須由股東會決議的事項包括：選任解任董事、變更章程、合併分割、重大營業行為（§185）等法律明定或章程另定的事項。

此條規定體現了股份有限公司「所有與經營分離」的原則：股東透過股東會處理重大事項，日常業務則由其所選任的董事會負責決策執行。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之董事會 |
| 核心規定 | 公司業務除應由股東會決議者外，均由董事會決議 |
| 違反效果 | 董事會越權決議股東會專屬事項者，該決議無效或得撤銷 |
| 注意事項 | 董事會享概括業務決策權；股東會權限為列舉事項 |

## 實務案例

**情境：** S公司股東會決議，要求董事會每月向股東會報告所有超過100萬元之交易。

**分析：** 依本條規定，公司日常業務之執行屬董事會之權限。股東會固然可以監督董事會，但不得以決議方式干預董事會之日常業務決策權。股東會要求董事會每月報告所有超過100萬元之交易，如係合理之監督機制尚屬可行，但如以此限制董事會之決策權限，則有侵越董事會權限之疑慮。

**結論：** 股東會與董事會各有權限範圍，應相互尊重，不得越權。

## 立法沿革

- 本條確立董事會為公司業務執行之核心機關，歷次修正均維持此原則。
