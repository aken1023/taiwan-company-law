# 第 208-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 董事會不為或不能行使職權，致公司有受損害之虞時，法院因利害關係人或檢察官之聲請，得選任一人以上之臨時管理人，代行董事長及董事會之職權。但不得為不利於公司之行為。

## 白話解說

本條規定「臨時管理人」制度。當董事會因故不為或不能行使職權（例如全體董事解任、辭職，或因內鬥而無法做成任何決議），導致公司有受損害之虞時，法院可以依利害關係人或檢察官之聲請，選任一人以上的臨時管理人。

臨時管理人的職權是代行董事長及董事會之職權，處理公司日常業務。但臨時管理人不得為不利於公司之行為，其職權範圍以維護公司利益為限。

臨時管理人制度是公司治理的最後防線，確保公司在董事會失能時仍能維持基本運作。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 董事會不為或不能行使職權之公司 |
| 核心規定 | 法院得依聲請選任臨時管理人代行董事長及董事會職權 |
| 違反效果 | 臨時管理人不得為不利於公司之行為 |
| 注意事項 | 聲請人為利害關係人或檢察官；臨時管理人職權有限制 |

## 實務案例

**情境：** AA公司全體董事因涉及掏空案被法院裁定收押禁見，無人能行使董事職權，公司營運陷入癱瘓。公司債權人向法院聲請選任臨時管理人。

**分析：** AA公司董事會確實不能行使職權，公司有受損害之虞。債權人為利害關係人，得依本條向法院聲請選任臨時管理人。法院選任之臨時管理人得代行董事長及董事會職權，處理公司日常營運事務，但不得為不利於公司之行為。

**結論：** 臨時管理人制度是確保公司持續營運的緊急機制。
