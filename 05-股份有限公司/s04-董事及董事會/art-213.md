# 第 213 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 公司與董事間訴訟，除法律另有規定外，由監察人代表公司，股東會亦得另選代表公司為訴訟之人。

## 白話解說

本條處理公司與董事之間訴訟時的代表問題。通常公司對外由董事長代表，但當公司與董事發生訴訟時，如果仍由董事長（或其他董事）代表公司，可能會產生利益衝突。

因此，本條規定公司與董事間的訴訟，原則上由監察人代表公司。監察人的職責本就是監督董事，由其代表公司與董事進行訴訟較為適當。

此外，股東會也可以另行選定其他人來代表公司進行訴訟，提供更大的彈性。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 公司與董事間之訴訟 |
| 核心規定 | 由監察人代表公司；股東會亦得另選代表人 |
| 違反效果 | 由有利益衝突之董事代表提訴者，訴訟代表權有瑕疵 |
| 注意事項 | 監察人為法定代表；股東會得另選代表 |

## 實務案例

**情境：** KK公司決議對董事長提起訴訟。公司設有監察人丙。

**分析：** 依本條，公司與董事長間之訴訟應由監察人丙代表公司進行。如股東會認為丙不適合或另有考量，得另選其他人代表公司為訴訟之人。

**結論：** 公司與董事間訴訟的代表權歸屬於監察人，以避免利益衝突。
