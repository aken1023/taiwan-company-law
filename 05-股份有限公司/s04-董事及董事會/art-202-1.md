# 第 202-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 公司依章程規定不設董事會者，由全體董事過半數同意執行公司業務，並由董事互推一人對外代表公司。

## 白話解說

本條配合第192條第二項不設董事會之規定，明確不設董事會但置二名董事之公司的業務執行方式。

當公司不設董事會而置二名董事時，公司業務之執行應由全體董事過半數同意（即二名董事中須有二人同意，實質上是全體同意）。對外代表公司的權限由董事互推一人行使。

如果公司僅置一名董事，則依第192條第二項規定，該董事即為董事長，所有業務由其單獨決定。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 不設董事會之股份有限公司 |
| 核心規定 | 業務由全體董事過半數同意執行；互推一人對外代表公司 |
| 違反效果 | 未經過半數同意之行為效力有疑義 |
| 注意事項 | 僅置一名董事者不適用本條（適用§192第二項） |

## 實務案例

**情境：** FF公司依章程不設董事會，置董事二人（甲、乙），互推甲對外代表公司。乙不同意某項交易，但甲仍以公司名義簽約。

**分析：** 依本條，業務執行應由全體董事過半數同意。二名董事中僅甲同意（1/2），未達過半數。該業務執行不符合法定要件，但甲既為對外代表人，其對第三人之交易行為的效力需視個案判斷。

**結論：** 不設董事會之公司，兩名董事間應充分溝通協調，以避免意見分歧導致業務執行困難。
