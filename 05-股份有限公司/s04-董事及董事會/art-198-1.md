# 第 198-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 經選任為董事之法人，得依其職務關係隨時改派補足原任期。

## 白話解說

本條處理法人擔任董事時，其代表人變更的問題。當法人被選任為董事後，法人可以依其內部職務關係，隨時改派其他代表人來擔任董事，而不需要重新經過股東會選舉，且代表人之任期仍為原來的任期。

此規定是考量法人股東的特殊性：法人的代表人可能因人事異動、組織調整等原因而需更換，如果每次更換都要召開股東會重新選舉，在實務上非常不便。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 經選任為董事之法人 |
| 核心規定 | 法人董事得隨時改派代表人補足原任期 |
| 違反效果 | 不適用 |
| 注意事項 | 改派不須經股東會；新代表人之任期為原任期之剩餘期間 |

## 實務案例

**情境：** 甲法人被選為DD公司董事，指派A先生為代表人。半年後甲法人因內部組織調整，改派B女士為代表人。

**分析：** 依本條規定，甲法人得隨時改派代表人。B女士接替A先生擔任DD公司董事，無須經股東會改選。B女士之任期為A先生原任期之剩餘期間。

**結論：** 法人董事改派代表人是法人之內部權限，不影響該董事席次之存續。
