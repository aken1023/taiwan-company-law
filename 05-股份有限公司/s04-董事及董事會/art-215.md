# 第 215 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 股東提起前條之訴，如有虛構或濫行訴訟之情事，致公司受有損害者，應負損害賠償之責。

## 白話解說

本條是防止股東濫用代位訴訟權的配套規定。如果股東依第214條提起代位訴訟，但其訴訟是虛構的（捏造事實）或濫行提訴的（明知無理由而惡意興訟），導致公司因此受到損害（例如訴訟費用、商譽損害等），該股東應對公司負損害賠償責任。

本條的目的在於平衡：一方面保障少數股東的代位訴訟權，另一方面防止此權利被濫用。虛構訴訟不僅浪費司法資源，也會對公司造成不必要的負擔和損害。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 依§214提起代位訴訟之股東 |
| 核心規定 | 虛構或濫行訴訟致公司受損者，股東應負賠償責任 |
| 違反效果 | 濫訴股東對公司負損害賠償責任 |
| 注意事項 | 須有「虛構」或「濫行」之情事；須致公司受有損害 |

## 實務案例

**情境：** 股東己因與董事個人恩怨，故意捏造董事侵占公司資產之事實，依第214條提起代位訴訟。法院審理後發現事實完全虛構，判決駁回。公司因此支出訴訟費用100萬元並影響商譽。

**分析：** 己虛構事實提訴，屬本條所稱之「虛構訴訟」。公司因此受有訴訟費用及商譽損害，己應依本條對公司負損害賠償責任。

**結論：** 股東提起代位訴訟前應確認事實基礎，虛構或濫行訴訟將承擔賠償責任。

- 本條配合第214條代位訴訟制度，防止權利濫用，歷次修正維持基本架構。
