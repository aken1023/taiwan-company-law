# 第 210 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> I 公司應將章程及歷屆股東會議事錄、財務報表備置於本公司，並將股東名簿及公司債存根簿備置於本公司或股務代理機構。
>
> II 前項章程及簿冊，股東及公司之債權人得檢具利害關係證明文件，指定範圍，隨時請求查閱、抄錄或複製；其備置於股務代理機構者，公司應令股務代理機構提供。
>
> III 代表公司之董事，違反第一項規定，不備置章程、簿冊，或違反前項規定無正當理由而拒絕查閱、抄錄或複製者，各處新臺幣一萬元以上五萬元以下罰鍰。

## 白話解說

本條規定公司簿冊的備置及查閱制度：

公司必須將章程、歷屆股東會議事錄、財務報表備置於公司；股東名簿及公司債存根簿備置於公司或股務代理機構。

股東及公司的債權人有權查閱、抄錄或複製這些文件，但須檢具利害關係的證明文件，並指定查閱範圍。如果文件備置於股務代理機構，公司應要求該機構提供。

違反備置義務或無正當理由拒絕查閱的代表公司董事，將被處以罰鍰。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司 |
| 核心規定 | 應備置章程、議事錄、財報等簿冊；股東及債權人有查閱權 |
| 違反效果 | 不備置或無正當理由拒絕查閱者，處1萬至5萬元罰鍰 |
| 注意事項 | 查閱須檢具利害關係證明並指定範圍 |

## 實務案例

**情境：** HH公司股東要求查閱最近三年之財務報表，公司以「涉及商業機密」為由拒絕。

**分析：** 依本條第二項，股東得隨時請求查閱財務報表。股東如已檢具利害關係證明文件並指定查閱範圍，公司無正當理由不得拒絕。「涉及商業機密」通常不構成拒絕股東查閱財報的正當理由。拒絕者將被處罰鍰。

**結論：** 公司應配合股東及債權人之合理查閱請求，不得任意拒絕。
