# 第 211-1 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 召集權人依第一百七十三條第二項或第一百七十三條之一規定召集股東會者，得請求公司或股務代理機構提供股東名簿。

## 白話解說

本條為配合少數股東或過半數股東自行召集臨時會的規定。當股東依第173條第二項（少數股東經主管機關許可後自行召集）或第173-1條（過半數股東自行召集）召集股東臨時會時，為了能夠合法地通知所有股東，召集權人有權請求公司或股務代理機構提供股東名簿。

如果沒有本條規定，自行召集臨時會的股東將無法取得完整的股東名單及地址，也就無法依法定程序發出召集通知，使得自行召集權形同虛設。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 依§173第二項或§173-1召集股東會之召集權人 |
| 核心規定 | 得請求公司或股務代理機構提供股東名簿 |
| 違反效果 | 公司或股務代理機構拒絕提供者，得循法律途徑救濟 |
| 注意事項 | 僅限於合法之召集權人始得請求 |

## 實務案例

**情境：** 股東甲依第173-1條自行召集臨時會，向公司請求提供股東名簿以便發出通知。公司以「保護股東隱私」為由拒絕。

**分析：** 依本條規定，甲身為合法之召集權人，有權請求公司提供股東名簿。公司不得以保護隱私為由拒絕，因為提供股東名簿是使召集權得以行使的必要條件。

**結論：** 公司應依法配合合法召集權人之股東名簿請求。
