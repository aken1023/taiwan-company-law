# 第 204 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 董事會之召集，應載明事由，於三日前通知各董事及監察人。但有緊急情事時，得隨時召集之。

## 白話解說

本條規定董事會召集的通知程序。召集通知應載明開會事由，並於開會三日前通知每位董事及監察人。監察人雖非董事會成員，但因負有監督職責，故亦應通知其列席。

如果有緊急情事（例如公司面臨重大危機、緊急商業機會等），可以不受三日前通知之限制，得隨時召集董事會。此但書提供了緊急時的彈性處理機制。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 董事會之召集 |
| 核心規定 | 應載明事由、3日前通知各董事及監察人；有緊急情事得隨時召集 |
| 違反效果 | 未依規定通知可能構成召集程序瑕疵，影響決議效力 |
| 注意事項 | 監察人亦應受通知；緊急召集之判斷標準需合理 |

## 實務案例

**情境：** V公司董事長於當日上午以電話通知各董事，下午召開董事會決議重大投資案。並未通知監察人。

**分析：** 依本條，董事會應於3日前通知。除非有緊急情事，否則當日通知違反規定。又未通知監察人，亦有程序瑕疵。如該投資案非屬緊急情事，該次董事會之召集程序違法，決議效力有疑義。

**結論：** 董事會召集應嚴守通知期限及通知對象，除真正緊急情事外不得任意縮短。
