# 第 194 條

> [!info] 章節位置
> 第五章 股份有限公司 › 第四節 董事及董事會

## 條文原文

> 繼續一年以上持有已發行股份總數百分之三以上之股東，得以書面請求董事為公司目的之行為或停止非為公司目的之行為。

## 白話解說

本條賦予少數股東「制止請求權」，即繼續一年以上持有已發行股份總數百分之三以上的股東，得以書面請求董事會停止違法行為。

具體而言，少數股東可以請求董事做兩種事情：一是請求董事「為」公司目的之行為（積極作為義務）；二是請求董事「停止」非為公司目的之行為（消極不作為義務）。

此制度使少數股東在董事有違法或不當行為時，不必等到下次股東會才能處理，可以即時以書面方式要求董事改正。如果董事不理會，股東可進一步循司法途徑救濟。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 繼續1年以上持有已發行股份總數3%以上之股東 |
| 核心規定 | 得以書面請求董事為公司目的之行為或停止非為公司目的之行為 |
| 違反效果 | 董事不從者，股東得循司法途徑救濟 |
| 注意事項 | 須以書面為之；持股須繼續1年以上且達3% |

## 實務案例

**情境：** 持有I公司5%股份達二年之股東林某，發現董事會正進行一項明顯不利公司之關係人交易。林某以書面請求董事會停止該交易。

**分析：** 林某持股比例（5%）及持有期間（2年）均符合本條要件。林某以書面請求董事停止非為公司目的之行為（不利之關係人交易），程序合法。董事會收到請求後應審慎評估，如該交易確實不利於公司，應停止進行。

**結論：** 少數股東制止請求權是即時監督董事行為的重要工具，可有效防止公司利益受損。
