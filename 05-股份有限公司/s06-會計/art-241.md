# 第 241 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 公司無虧損者，得依前條第一項至第三項所定股東會決議之方法，將法定盈餘公積及下列資本公積之全部或一部撥充資本，按股東原有股份之比例發給新股：
>
> 一、超過票面金額發行股票所得之溢額。
>
> 二、受領贈與之所得。
>
> 前項法定盈餘公積撥充資本者，以該項公積超過實收資本額百分之二十五為限。

## 白話解說

本條允許公司在無虧損的情況下，將法定盈餘公積及特定類型的資本公積撥充資本（即轉增資），按股東原有股份比例發給新股。此程序使公司資本額增加，但股東不需額外出資，而是以原有的公積轉入資本。

可用於撥充資本的公積包括：法定盈餘公積、股票溢價之資本公積、受領贈與之資本公積。其中，法定盈餘公積用於撥充資本時，有數額限制：以超過實收資本額百分之二十五的部分為限。此限制確保公司保留足夠的法定公積作為財務安全墊。

例如，公司實收資本額為一億元，法定盈餘公積為五千萬元，則超過25%（二千五百萬元）的部分，即二千五百萬元得撥充資本。

撥充資本的決議程序與第240條相同，須經股東會特別決議。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 無虧損之股份有限公司 |
| 核心規定 | 得將法定盈餘公積及特定資本公積撥充資本，發給新股 |
| 違反效果 | 未經合法決議或有虧損時辦理者，轉增資無效 |
| 注意事項 | 法定公積撥充以超過實收資本額25%部分為限 |

## 實務案例

**情境：** 甲公司實收資本額一億元，法定盈餘公積四千萬元，股票溢價資本公積二千萬元，公司無虧損。董事會擬以法定盈餘公積二千萬元及資本公積二千萬元，共四千萬元撥充資本發行新股。

**分析：** 依本條規定：(1) 法定盈餘公積撥充資本以超過實收資本額25%部分為限。法定公積四千萬元，25%為一千五百萬元（四千萬 - 一億x25% = 一千五百萬元），故法定公積最多可撥充一千五百萬元，而非二千萬元。(2) 股票溢價資本公積二千萬元全額可撥充。(3) 合計最多可撥充三千五百萬元。須經股東會特別決議。

**結論：** 公積撥充資本時應注意法定盈餘公積之25%限額，且公司須無虧損始得辦理。
