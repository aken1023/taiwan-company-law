# 第 230 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 董事會應將其所造具之各項表冊，提出於股東常會請求承認，經股東常會承認後，董事會應將財務報表及盈餘分派或虧損撥補之決議，分發各股東。
>
> 前項財務報表及盈餘分派或虧損撥補決議之分發，公開發行股票之公司，得以公告方式為之。
>
> 第一項表冊及決議，公司得以電子方式分發之。

## 白話解說

本條規定了董事會將表冊提交股東會承認及承認後之分發程序。董事會編造的營業報告書、財務報表及盈餘分派或虧損撥補議案，經監察人查核後，應提出於股東常會請求承認（亦即請求股東會表決通過）。

股東會承認後，董事會應將財務報表及盈餘分派或虧損撥補之決議分發給各股東，讓每位股東都知悉公司的財務狀況及盈餘分配的結果。

對於公開發行公司，因股東人數眾多，逐一分發在實務上有困難，因此允許以公告方式替代個別分發。此外，所有公司均得以電子方式分發相關文件，以符合數位化之趨勢。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之董事會及股東會 |
| 核心規定 | 董事會應將表冊提請股東會承認，承認後分發各股東 |
| 違反效果 | 未提請承認之表冊不生效力 |
| 注意事項 | 公開發行公司得以公告代替分發；得以電子方式為之 |

## 實務案例

**情境：** 亥公司股東常會審議年度財務報表時，部分股東認為董事會編造之盈餘分派議案分派金額過低，拒絕承認。

**分析：** 依本條規定，表冊需經股東常會「承認」。若股東會不承認盈餘分派議案，董事會應修正議案後重新提出。實務上股東會可以修正通過不同的分派方案。未經承認的盈餘分派議案不得執行。

**結論：** 股東會對表冊之承認權是股東監督公司經營的重要權利。股東會有權不承認或修改董事會之提案。
