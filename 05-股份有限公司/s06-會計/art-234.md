# 第 234 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 公司依其業務之性質，自設立登記後，如需二年以上之準備始能開始營業者，經主管機關之許可，得於開始營業前分派股息。
>
> 前項分派股息之金額，應以預付股息列入資產負債表之股東權益項下，公司開始營業後，每屆分派股息及紅利超過實收資本額百分之六時，應以其超過之金額扣抵沖銷之。

## 白話解說

本條規定的是「建設股息」制度，適用於需要長期準備才能開始營業的特殊行業。例如基礎建設公司、大型工程公司等，從設立到開始營業可能需要二年以上的準備期間（如建造工廠、鋪設管線等）。在此期間，公司尚無營業收入，股東無法獲得任何投資回報。

為鼓勵投資人投資此類長期建設事業，法律特別允許此類公司在取得主管機關許可後，於開始營業前即可分派股息給股東。這筆股息稱為「建設股息」，本質上是公司預先支付給股東的一種利息補貼。

建設股息在會計處理上，應以「預付股息」列入資產負債表的股東權益項下（列為減項）。待公司開始營業並有盈餘後，每年分派的股息及紅利超過實收資本額百分之六的部分，應用來扣抵沖銷預付股息，直到完全沖銷為止。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 需二年以上準備始能開始營業之公司 |
| 核心規定 | 經主管機關許可，得於營業前分派建設股息 |
| 違反效果 | 未經許可分派者為違法分派 |
| 注意事項 | 建設股息應以預付股息列帳，日後須扣抵沖銷 |

## 實務案例

**情境：** 丁公司為一大型BOT案之特許公司，從設立到高速公路完工通車預計需要四年。公司依本條向主管機關申請在營業前分派建設股息。

**分析：** 丁公司之業務性質確實需要二年以上準備始能開始營業，符合本條適用要件。經主管機關許可後，得於通車營業前分派股息予股東。分派之金額應以預付股息列帳，待公司通車營業後，每年盈餘分派超過實收資本額6%的部分，用以扣抵建設股息，直到全部沖銷為止。

**結論：** 建設股息制度在實務上較為少見，但對長期建設事業之投資人提供了重要的誘因。申請時須經主管機關嚴格審查。
