# 第 231 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 各項表冊經股東會決議承認後，視為公司已解除董事及監察人之責任。但董事或監察人有不法行為者，不在此限。

## 白話解說

本條規定了股東會承認表冊後的法律效果。當股東會表決通過（承認）董事會所編造之各項表冊後，法律上視為公司已對董事及監察人就該會計年度之責任予以解除。這意味著，股東事後不能再就該年度之經營成果向董事或監察人主張責任。

此規定的立法目的在於給予董事及監察人一定的法律安定性。若每年度的經營決策在表冊經承認後仍可無限期被追究，將使經營者不敢做出任何有風險的商業決策，不利公司發展。

但本條設有重要的例外：若董事或監察人有「不法行為」者，其責任不因股東會承認表冊而解除。所謂不法行為，包括侵占公司財產、偽造財務報表、違反忠實義務等違法行為。此例外規定確保違法行為人不能藉由操控股東會承認來逃避法律責任。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之董事及監察人 |
| 核心規定 | 表冊經股東會承認後，視為解除董事及監察人之責任 |
| 違反效果 | 有不法行為者，責任不因承認而解除 |
| 注意事項 | 僅解除一般經營責任，不法行為之責任不受影響 |

## 實務案例

**情境：** 甲公司董事長透過虛增營收來美化財務報表，經股東會承認。事後被檢調查獲後，董事長主張表冊已經股東會承認，其責任已解除。

**分析：** 依本條但書，董事有不法行為者，其責任不因股東會承認表冊而解除。本案董事長虛增營收係屬不法行為（涉及偽造文書、違反證券交易法等），不在責任解除之範圍。股東會承認表冊的效力僅及於合法的經營決策，不及於違法行為。

**結論：** 董事或監察人不得利用股東會承認表冊之程序來免除其不法行為之責任。不法行為者仍須面對民事損害賠償及刑事追訴。
