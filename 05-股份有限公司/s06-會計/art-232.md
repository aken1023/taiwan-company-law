# 第 232 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 公司非彌補虧損及依本法規定提出法定盈餘公積後，不得分派股息及紅利。公司無盈餘時，不得分派股息及紅利。但法定盈餘公積已超過實收資本額百分之五十時，得以其超過部分派充股息及紅利。
>
> 公司負責人違反前項規定分派股息及紅利時，各處一年以下有期徒刑、拘役或科或併科新臺幣六萬元以下罰金。

## 白話解說

本條是盈餘分派的根本限制規定，體現了「資本維持原則」。公司在分派股息及紅利前，必須先完成兩項法定義務：

第一，彌補虧損：公司若有以往年度累積之虧損，必須先用盈餘彌補虧損。這確保公司不會在虧損狀態下仍分派盈餘，損及公司資本。

第二，提列法定盈餘公積：依第237條規定，公司應就稅後淨利提撥百分之十作為法定盈餘公積。

完成上述兩項義務後，剩餘的盈餘才能分派。若公司根本沒有盈餘，則絕對不得分派。不過，當法定盈餘公積已累積超過實收資本額百分之五十時，其超過部分得用以分派股息及紅利，以避免過度累積不必要之公積。

為確保此規定之落實，本條第二項設有嚴厲的刑事罰則：公司負責人違法分派者，處一年以下有期徒刑、拘役或科或併科罰金。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司 |
| 核心規定 | 非彌補虧損及提列法定盈餘公積後不得分派股息紅利 |
| 違反效果 | 負責人處一年以下有期徒刑、拘役或罰金（刑事責任） |
| 注意事項 | 法定公積超過資本額50%時，超過部分得分派 |

## 實務案例

**情境：** 乙公司帳上尚有以往年度累積虧損三千萬元，但本年度獲利五千萬元。董事會擬直接將五千萬元全數分派予股東，不先彌補虧損。

**分析：** 依本條規定，公司非彌補虧損後不得分派股息及紅利。乙公司應先以本年度獲利五千萬元彌補累積虧損三千萬元，再提列法定盈餘公積（剩餘二千萬元之10%，即二百萬元），剩餘一千八百萬元才能分派。董事會擬全數分派之計畫違反本條規定，負責人可能面臨刑事責任。

**結論：** 盈餘分派的順序為：先彌補虧損、再提法定盈餘公積，最後才是分派。違反此順序者，公司負責人將面臨刑事處罰。
