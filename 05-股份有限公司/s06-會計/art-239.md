# 第 239 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 法定盈餘公積，除填補公司虧損外，不得使用之。

## 白話解說

本條明確限定法定盈餘公積的使用目的：僅能用於填補公司虧損，不得作為其他用途。這是因為法定盈餘公積的設置目的就是作為公司的財務安全墊，在公司出現虧損時用來彌補，維護公司資本的完整性。

與資本公積（第237-1條）相比，法定盈餘公積的使用限制更為嚴格。資本公積尚有例外可以發放新股或現金（溢價及受贈部分），但法定盈餘公積則完全不得用於分配，僅能用於填補虧損。

不過，須注意第241條另有規定，在公司無虧損的情況下，法定盈餘公積超過實收資本額25%的部分，得撥充資本（發行新股）。此規定應與本條搭配理解：法定公積在「有虧損」時用來填補虧損（本條），在「無虧損」且超過25%時得撥充資本（第241條）。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司 |
| 核心規定 | 法定盈餘公積僅得用於填補虧損 |
| 違反效果 | 挪作他用之決議無效 |
| 注意事項 | 第241條有例外允許撥充資本 |

## 實務案例

**情境：** 壬公司法定盈餘公積累積達五千萬元，公司目前無虧損。董事會擬以法定盈餘公積二千萬元發放現金股利給股東。

**分析：** 依本條規定，法定盈餘公積除填補虧損外不得使用。壬公司擬直接以法定盈餘公積發放現金股利，已違反本條規定。法定盈餘公積不同於可分配之資本公積（第237-1條），不得直接分派。但若符合第241條條件（公司無虧損且超過25%），可將超過部分撥充資本，以發行新股方式間接回饋股東。

**結論：** 法定盈餘公積不得直接用於現金分派，公司如欲運用法定盈餘公積，應透過第241條之撥充資本方式辦理。
