# 第 229 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 董事會所造具之各項表冊與監察人之報告書，應於股東常會開會十日前，備置於本公司，股東得隨時查閱，並得偕同其所委託之律師或會計師查閱。

## 白話解說

本條規定了股東的資訊取得權。董事會編造的各項表冊（營業報告書、財務報表、盈餘分派或虧損撥補議案）以及監察人的查核報告書，必須在股東常會開會十日前備置於公司，讓股東有充分的時間在開會前閱讀相關資料，以便在股東會上做出知情的決定。

股東不僅有權「隨時」查閱這些文件，還可以攜帶其委託的律師或會計師一同前往查閱。此規定考量到財務報表等文件具有專業性，一般股東可能不具備完整的閱讀能力，因此允許其委託專業人士協助審閱，以確保股東權益的實質保障。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司 |
| 核心規定 | 表冊應於股東常會十日前備置於公司，供股東查閱 |
| 違反效果 | 未備置者可能影響股東會決議之效力 |
| 注意事項 | 股東得偕同律師或會計師查閱 |

## 實務案例

**情境：** 戌公司股東D於股東常會前七日前往公司要求查閱財務報表，但公司以「文件尚在整理中」為由拒絕。

**分析：** 依本條規定，各項表冊應於股東常會開會「十日前」備置於公司。股東D於七日前要求查閱時，公司已逾法定備置期限而仍未備妥，已違反本條規定。公司不得以任何理由拒絕股東之查閱要求。若持續拒絕，股東得主張股東會對於該等表冊之承認決議有瑕疵。

**結論：** 公司應嚴格遵守十日前備置之期限，確保股東之資訊取得權受到保障。
