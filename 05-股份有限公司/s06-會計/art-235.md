# 第 235 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 股息及紅利之分派，除章程另有規定外，以各股東持有股份之比例為準。

## 白話解說

本條規定了股息及紅利分派的基本原則：按照各股東所持有股份之比例分派。亦即，持有越多股份的股東，獲得的股息及紅利就越多。此為「股份平等原則」在盈餘分派上的體現，每一股享有相同的盈餘分配權。

不過，本條設有「除章程另有規定外」的但書，允許公司在章程中訂定不同的分派方式。例如，公司可以在章程中規定特別股享有較高的股息率，或優先分派等。這種彈性安排必須在章程中明文規定，不能僅由董事會或股東會臨時決議變更。

2018年修法時，原條文中有關員工紅利分派比率之規定已移至第235-1條獨立規範，使本條回歸純粹規範股東間之分派比例原則。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司之全體股東 |
| 核心規定 | 股息紅利按各股東持有股份比例分派 |
| 違反效果 | 不按比例分派且無章程依據者，決議有瑕疵 |
| 注意事項 | 章程得另行規定不同之分派方式 |

## 實務案例

**情境：** 戊公司有普通股及特別股。章程規定特別股享有年息8%之優先分派權，剩餘盈餘再由普通股股東按比例分派。某年度盈餘僅夠支付特別股股息，普通股股東未獲分配。

**分析：** 依本條規定，章程得另行規定分派方式。戊公司章程明定特別股之優先分派權，此為合法之安排。當盈餘僅夠支付特別股股息時，普通股股東未獲分配並不違法，因為已按章程規定之順序辦理。

**結論：** 公司可透過章程設計不同之盈餘分派機制，但必須在章程中明確規定，以保障各類股東之權益。
