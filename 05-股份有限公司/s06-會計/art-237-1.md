# 第 237-1 條

> [!info] 章節位置
> 第五章 股份有限公司 → 第六節 會計

## 條文原文

> 資本公積除填補公司虧損外，不得使用之。但下列資本公積，得按股東原有股份之比例發給新股或現金：
>
> 一、超過票面金額發行股票所得之溢額。
>
> 二、受領贈與之所得。

## 白話解說

本條規定了資本公積的使用限制及例外。資本公積是指公司因非營業活動所產生的權益增加，原則上僅能用於填補公司虧損，不得作為其他用途。此規定旨在維護資本公積的穩定性，保障公司及債權人之利益。

但本條設有例外規定，允許以下兩種資本公積得按股東原有股份比例發給新股或現金：

第一，股票溢價發行之溢額：公司以高於面額之價格發行股票時，超過面額之部分屬於資本公積。此溢額本質上是股東額外繳入的資金，允許將其發回股東合乎道理。

第二，受領贈與之所得：公司因受贈而取得之財產，在會計上列為資本公積，亦可用於發給新股或現金。

發放方式須按「股東原有股份之比例」為之，以維護股份平等原則。

## 重點摘要

| 項目 | 內容 |
|------|------|
| 適用對象 | 股份有限公司 |
| 核心規定 | 資本公積原則上僅得填補虧損；例外得發給新股或現金 |
| 違反效果 | 違法使用資本公積者，相關決議無效 |
| 注意事項 | 僅股票溢價及受贈所得之資本公積得分派 |

## 實務案例

**情境：** 辛公司曾以每股15元（面額10元）發行股票，累積資本公積（股票溢價）達三千萬元。董事會擬以此資本公積發放現金予股東。

**分析：** 依本條但書第一款，超過票面金額發行股票所得之溢額屬於可分配之資本公積，得按股東原有股份比例發給現金或新股。辛公司之股票溢價資本公積三千萬元符合此規定，得依法發放。但須注意依第241條規定，以資本公積撥充資本（發新股）時，須先無虧損。

**結論：** 並非所有資本公積均得用於分派，僅限於股票溢價及受贈所得。公司應注意區分資本公積之來源，以確保合法使用。
